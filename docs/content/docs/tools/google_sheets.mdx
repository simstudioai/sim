## Usage Instructions

Integrate Google Sheets functionality to manage spreadsheet data. Read data from specific ranges, write new data, and update existing cells using OAuth authentication. Supports various input and output formats for flexible data handling.



## Tools

### `google_sheets_read`

Read data from a Google Sheets spreadsheet

#### Input Parameters

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `accessToken` | string | Yes | No description |

#### Output

| Parameter | Type |
| --------- | ---- |
| `data` | json |

### `google_sheets_write`

Write data to a Google Sheets spreadsheet

#### Input Parameters

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `accessToken` | string | Yes | No description |

#### Output

| Parameter | Type |
| --------- | ---- |
| `updatedRange` | string |
| `updatedRows` | string |
| `updatedColumns` | string |
| `updatedCells` | string |
| `metadata` | string |
| `spreadsheetId` | string |
| `spreadsheetUrl` | string |

### `google_sheets_update`

Update data in a Google Sheets spreadsheet

#### Input Parameters

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `accessToken` | string | Yes | No description |

#### Output

| Parameter | Type |
| --------- | ---- |
| `updatedRange` | string |
| `updatedRows` | string |
| `updatedColumns` | string |
| `updatedCells` | string |
| `metadata` | string |
| `spreadsheetId` | string |
| `spreadsheetUrl` | string |



## Configuration

### Input Parameters

The `response` output contains the following fields:
