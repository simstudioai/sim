## Usage Instructions

Integrate Gmail functionality to send email messages within your workflow. Automate email communications and process email content using OAuth authentication.



## Tools

### `gmail_send`

Send emails using Gmail

#### Input Parameters

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `accessToken` | string | Yes | Access token for Gmail API |

#### Output

| Parameter | Type |
| --------- | ---- |
| `content` | string |
| `metadata` | string |
| `threadId` | string |
| `labelIds` | string |

### `gmail_read`

Read emails from Gmail

#### Input Parameters

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `accessToken` | string | Yes | Access token for Gmail API |

#### Output

| Parameter | Type |
| --------- | ---- |
| `content` | string |
| `metadata` | string |

### `gmail_search`

Search emails in Gmail

#### Input Parameters

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `accessToken` | string | Yes | Access token for Gmail API |

#### Output

| Parameter | Type |
| --------- | ---- |
| `content` | string |



## Configuration

No configuration parameters required.

The `response` output contains the following fields:

| `content` | `string` |
