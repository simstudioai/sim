## Usage Instructions

Integrate Google Docs functionality to manage documents. Read content from existing documents, write to documents, and create new documents using OAuth authentication. Supports text content manipulation for document creation and editing.



## Tools

### `google_docs_read`

Read content from a Google Docs document

#### Input Parameters

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `accessToken` | string | Yes | No description |

#### Output

| Parameter | Type |
| --------- | ---- |
| `content` | string |

### `google_docs_write`

Write or update content in a Google Docs document

#### Input Parameters

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `accessToken` | string | Yes | No description |

#### Output

| Parameter | Type |
| --------- | ---- |
| `updatedContent` | string |

### `google_docs_create`

Create a new Google Docs document

#### Input Parameters

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `accessToken` | string | Yes | No description |

#### Output

| Parameter | Type |
| --------- | ---- |
| `metadata` | string |



## Configuration

### Input Parameters

| Parameter | Type | Required | Description | 
| --------- | ---- | -------- | ----------- | 
| `operation` | string | Yes | Operation |



## Outputs

The `response` output contains the following fields:

| Field | Type |
| ----- | ---- |
| `content` | `string` |
| `metadata` | `json` |
| `updatedContent` | `boolean` |

**Note:** The actual fields returned will depend on the selected operation.
