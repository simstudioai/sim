## Usage Instructions

Integrate with Supabase to manage your database, authentication, storage, and more. Query data, manage users, and interact with Supabase services directly.



## Tools

### `supabase_query`

Query data from a Supabase table

#### Input Parameters

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Yes | Your Supabase client anon key |

#### Output

| Parameter | Type |
| --------- | ---- |
| `message` | string |
| `results` | string |

### `supabase_insert`

Insert data into a Supabase table

#### Input Parameters

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Yes | Your Supabase client anon key |

#### Output

| Parameter | Type |
| --------- | ---- |
| `message` | string |
| `results` | string |



## Configuration

### Input Parameters

| Parameter | Type | Required | Description | 
| --------- | ---- | -------- | ----------- | 
| `operation` | string | Yes | Operation |



## Outputs

The `response` output contains the following fields:

| Field | Type |
| ----- | ---- |
| `message` | `string` |
| `results` | `json` |

**Note:** The actual fields returned will depend on the selected operation.
