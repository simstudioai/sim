Google Forms provides 1 trigger for automating workflows based on events.

## Google Forms Webhook

Trigger workflow from Google Form submissions (via Apps Script forwarder)

### Configuration

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `token` | short-input | Yes | We validate requests using this secret. Send it as Authorization: Bearer &lt;token&gt; or a custom header. |
| `secretHeaderName` | short-input | No | If set, the webhook will validate this header equals your Shared Secret instead of Authorization. |
| `triggerFormId` | short-input | No | Optional, for clarity and matching in workflows. Not required for webhook to work. |
| `includeRawPayload` | switch | No | Include the original payload from Apps Script in the workflow input. |
| `setupScript` | code | No | Copy this code and paste it into your Google Forms Apps Script editor |

### Output

| Field | Type | Description |
| ----- | ---- | ----------- |
| `responseId` | string | Unique response identifier \(if available\) |
| `createTime` | string | Response creation timestamp |
| `lastSubmittedTime` | string | Last submitted timestamp |
| `formId` | string | Google Form ID |
| `answers` | object | Normalized map of question -&gt; answer |
| `raw` | object | Original payload \(when enabled\) |
