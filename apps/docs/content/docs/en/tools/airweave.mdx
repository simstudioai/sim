## Usage Instructions

Search across all your connected data sources using Airweave. Supports 50+ integrations including Stripe, GitHub, Notion, Slack, HubSpot, Zendesk, and more. Get raw search results or AI-generated summaries.



## Tools

### `airweave_search`

Search across all connected data sources in your Airweave collection. Supports 50+ integrations including Stripe, GitHub, Notion, Slack, HubSpot, Zendesk, and more. Returns relevant search results with metadata or AI-generated summaries.

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `collectionId` | string | Yes | The readable ID of the Airweave collection to search |
| `query` | string | Yes | The search query to find relevant information from connected data sources |
| `limit` | number | No | Maximum number of results to return \(1-100, default: 10\) |
| `offset` | number | No | Number of results to skip for pagination \(default: 0\) |
| `responseType` | string | No | Response format: 'raw' for search results or 'completion' for AI-generated answer \(default: 'raw'\) |
| `recencyBias` | number | No | Weight for recent results \(0.0=no bias, 1.0=only recency, default: 0.0\) |
| `apiKey` | string | Yes | Airweave API Key \(get from https://app.airweave.ai\) |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `status` | string | Status of the search operation \(success, no_results, no_relevant_results\) |
| `results` | array | Search results with content and metadata from connected data sources |
