[Kalshi](https://kalshi.com) is a federally regulated exchange that enables users to trade directly on the outcomes of future events, known as prediction markets. Kalshi offers an open API, giving developers access to real-time market and event data, as well as account and trading information for automated or data-driven workflows.

With Kalshi's API and Sim integration, you can enable agents to retrieve up-to-date prediction market information, explore open events, check account balances, manage orders, and track positionsâ€”all programmatically. This provides powerful capabilities for research, portfolio management, and creating intelligent workflows based on real-world event probabilities and outcomes.

Key features of the Kalshi integration include:

- **Market Listing & Filtering:** Retrieve lists of markets or individual market details, including filtering by status, series, or event.
- **Event Data:** Explore and filter real-world event listings and retrieve detailed event information.
- **Account and Trading:** Access user account balances, open positions, and manage order flow.
- **Automation Ready:** Build intelligent agents that leverage real-time prediction market data for automated actions and insights.

By using the documented API tools, you can seamlessly incorporate Kalshi's rich event and financial market data into your AI workflows, dashboards, and trading automations.

## Usage Instructions

Integrate Kalshi prediction markets into the workflow. Can get markets, get market, get events, get event, get balance, get positions, get orders.



## Tools

### `kalshi_get_markets`

Retrieve a list of prediction markets from Kalshi with optional filtering

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `status` | string | No | Filter by status \(unopened, open, closed, settled\) |
| `seriesTicker` | string | No | Filter by series ticker |
| `eventTicker` | string | No | Filter by event ticker |
| `limit` | string | No | Number of results \(1-1000, default: 100\) |
| `cursor` | string | No | Pagination cursor for next page |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Operation success status |
| `output` | object | Markets data and metadata |

### `kalshi_get_market`

Retrieve details of a specific prediction market by ticker

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `ticker` | string | Yes | The market ticker \(e.g., "KXBTC-24DEC31"\) |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Operation success status |
| `output` | object | Market data and metadata |

### `kalshi_get_events`

Retrieve a list of events from Kalshi with optional filtering

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `status` | string | No | Filter by status \(open, closed, settled\) |
| `seriesTicker` | string | No | Filter by series ticker |
| `withNestedMarkets` | string | No | Include nested markets in response \(true/false\) |
| `limit` | string | No | Number of results \(1-200, default: 200\) |
| `cursor` | string | No | Pagination cursor for next page |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Operation success status |
| `output` | object | Events data and metadata |

#### Output

| Parameter | Type | Description |

### `kalshi_get_balance`

Retrieve your account balance and portfolio value from Kalshi

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `keyId` | string | Yes | Your Kalshi API Key ID |
| `privateKey` | string | Yes | Your RSA Private Key \(PEM format\) |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Operation success status |
| `output` | object | Balance data and metadata |

### `kalshi_get_positions`

Retrieve your open positions from Kalshi

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `keyId` | string | Yes | Your Kalshi API Key ID |
| `privateKey` | string | Yes | Your RSA Private Key \(PEM format\) |
| `ticker` | string | No | Filter by market ticker |
| `eventTicker` | string | No | Filter by event ticker \(max 10 comma-separated\) |
| `settlementStatus` | string | No | Filter by settlement status \(all, unsettled, settled\). Default: unsettled |
| `limit` | string | No | Number of results \(1-1000, default: 100\) |
| `cursor` | string | No | Pagination cursor for next page |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Operation success status |
| `output` | object | Positions data and metadata |
