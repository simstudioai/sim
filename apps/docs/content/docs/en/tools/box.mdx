[Box](https://box.com/) is a leading cloud content management and collaboration platform that enables users and organizations to securely store, access, and collaborate on files and folders from anywhere.

With Box, you can:

- **Manage files and folders**: Upload, organize, move, copy, rename, or delete content in your Box cloud storage
- **Collaborate with others**: Share files and folders, set permissions, and invite collaborators for secure teamwork
- **Search content**: Find files or folders by name or content within your Box account
- **Create shared links**: Generate secure, shareable links for files and folders

In Sim, the Box integration allows your agents to interact programmatically with your Box account, automating the management of your files and folders. Using the tools below, you can:

- Retrieve information about specific files (`box_get_file`)
- List the contents of any Box folder (`box_list_folder`)
- Create new folders (`box_create_folder`)
- Search for files or folders
- And more â€” enabling workflow automation involving your Box data.

These capabilities let your agents manage content, streamline collaboration, and automate file operations as part of your workflow execution in Sim.

## Usage Instructions

Integrate Box into the workflow. Can manage files and folders, search content, create shared links, and collaborate with users. Supports file operations like copy, move, delete, and download.



## Tools

### `box_get_file`

Get information about a file in Box

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `fileId` | string | Yes | The ID of the file to retrieve |
| `fields` | string | No | Comma-separated list of fields to include \(e.g., "id,name,size,modified_at,shared_link"\) |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `file` | object | The file information |

### `box_list_folder`

List items in a Box folder

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `folderId` | string | Yes | The ID of the folder to list \(use "0" for root folder\) |
| `limit` | number | No | Maximum number of items to return \(default: 100, max: 1000\) |
| `offset` | number | No | Offset for pagination \(default: 0\) |
| `fields` | string | No | Comma-separated list of fields to include |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `items` | array | Array of files and folders in the folder |

### `box_create_folder`

Create a new folder in Box

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `parentFolderId` | string | Yes | The ID of the parent folder \(use "0" for root folder\) |
| `folderName` | string | Yes | The name of the new folder |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `folder` | object | The created folder |

### `box_delete_file`

Delete a file from Box. Standard delete moves to trash. Use permanent flag for files already in trash.

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `fileId` | string | Yes | The ID of the file to delete |
| `permanent` | boolean | No | If true, permanently delete a file that is already in trash |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Whether the file was successfully deleted |
| `fileId` | string | The ID of the deleted file |

### `box_delete_folder`

Delete a folder from Box (moves to trash by default)

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `folderId` | string | Yes | The ID of the folder to delete |
| `recursive` | boolean | No | Delete folder contents recursively \(default: true\) |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Whether the folder was successfully deleted |
| `folderId` | string | The ID of the deleted folder |

### `box_copy_file`

Create a copy of a file in Box

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `fileId` | string | Yes | The ID of the file to copy |
| `parentFolderId` | string | Yes | The ID of the destination folder |
| `newName` | string | No | Optional new name for the copy |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `file` | object | The copied file |

### `box_update_file`

Update file information in Box (rename, move, or update description)

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `fileId` | string | Yes | The ID of the file to update |
| `name` | string | No | New name for the file |
| `description` | string | No | New description for the file |
| `parentId` | string | No | ID of the new parent folder \(to move the file\) |
| `tags` | array | No | Array of tags to set on the file |
| `items` | string | No | No description |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `file` | object | The updated file |

### `box_search`

Search for files and folders in Box

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `query` | string | Yes | The search query string |
| `scope` | string | No | Search scope: "user_content" \(default\) or "enterprise_content" |
| `fileExtensions` | string | No | Comma-separated file extensions to filter by \(e.g., "pdf,docx"\) |
| `ancestorFolderIds` | string | No | Comma-separated folder IDs to search within |
| `contentTypes` | string | No | Comma-separated content types: "name", "description", "file_content", "comments", "tag" |
| `limit` | number | No | Maximum number of results \(default: 30, max: 200\) |
| `offset` | number | No | Offset for pagination \(default: 0\) |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `results` | array | Array of search results |

### `box_download_file`

Get the download URL for a file in Box

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `fileId` | string | Yes | The ID of the file to download |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `content` | string | File content \(for text files\) |
| `downloadUrl` | string | URL to download the file |
| `fileName` | string | Name of the file |
| `mimeType` | string | MIME type of the file |

### `box_get_current_user`

Get information about the authenticated Box user

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `user` | object | The authenticated user |

### `box_create_shared_link`

Create a shared link for a file or folder in Box

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `itemId` | string | Yes | The ID of the file or folder |
| `itemType` | string | Yes | The type of item: "file" or "folder" |
| `access` | string | No | Access level: "open" \(anyone\), "company", or "collaborators" |
| `password` | string | No | Password to protect the shared link |
| `unsharedAt` | string | No | Expiration date in ISO 8601 format |
| `canDownload` | boolean | No | Whether the link allows downloads |
| `canPreview` | boolean | No | Whether the link allows previews |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `sharedLink` | object | The shared link details |

### `box_create_collaboration`

Share a file or folder with a user or group in Box

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `itemId` | string | Yes | The ID of the file or folder to share |
| `itemType` | string | Yes | The type of item: "file" or "folder" |
| `accessibleByLogin` | string | No | Email address of the user to share with |
| `accessibleById` | string | No | ID of the user or group to share with |
| `accessibleByType` | string | Yes | Type of collaborator: "user" or "group" |
| `role` | string | Yes | Access role: "editor", "viewer", "previewer", "uploader", or "co-owner" |
| `canViewPath` | boolean | No | Whether the collaborator can see the full path to the folder \(folders only\) |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `collaboration` | object | The created collaboration |
