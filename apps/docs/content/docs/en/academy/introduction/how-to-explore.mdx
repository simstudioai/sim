**Purpose.** End with a simple orientation for self-guided exploration.

**Learning outcome.** The learner knows how to keep exploring: ask Mothership, open workflows, inspect resources, run things, check logs, and watch module videos when they want depth.

## What to say

- The fastest way to learn Sim is to build something small and inspect it.
- Start with Mothership. Open the workflow it creates. Run it. Look at the logs.
- When you want to go deeper, watch the module videos.

## The exploration loop

```
Ask → Inspect → Run → Look at logs → Adjust
```

## Suggested order

1. Recap: *You've seen the workspace, Mothership, workflows, resources, and logs.*
2. Give the exploration loop above.
3. Point to the module videos for depth.
4. Close: *You don't need to understand everything before starting. Build something small, inspect what happens, and follow the questions that come up.*
