**Learning outcome.** The learner understands skills as reusable instructions / playbooks for agents.

## Key beats

### 1. What skills are

Reusable instruction packages, attachable to agents and workflows, useful for consistent behavior across systems.

### 2. Why not just prompt?

Prompts are local to one agent or block. Skills are reusable and named, reduce repeated prompt-stuffing, and make behavior easier to share and maintain.

### 3. Use cases

A support-triage policy, a writing style guide, a research methodology, a compliance checklist, a partner implementation pattern.
