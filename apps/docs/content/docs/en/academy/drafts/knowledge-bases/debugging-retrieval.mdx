**Learning outcome.** The learner can diagnose weak knowledge-base answers.

## Key beats

- Check the source documents exist.
- Check the processing status.
- Inspect the retrieved chunks, if possible.
- Improve the query.
- Use tags, filters, and connectors.
- Separate knowledge bases by domain if needed.
- Know when to use a table or file instead.
