**Learning outcome.** The learner understands how workflows can produce artifacts.

## Key beats

- A workflow takes structured data.
- An agent or function creates a report.
- The file is saved to the workspace.
- The file is reused, shared, or downloaded.
- Logs confirm the output.

## Good demo material

A research report, a customer summary, a generated proposal, or a PDF / markdown export.
