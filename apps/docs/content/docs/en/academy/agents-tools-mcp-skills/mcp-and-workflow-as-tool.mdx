## Key beats

### 1. MCP

A standard way for AI systems to call external tools. Sim can connect MCP tools into agents and workflows.

### 2. Workflow as tool

A workflow can be used as a callable capability — useful for modular systems.

### 3. Deploy a workflow as MCP

Expose a Sim workflow to Claude, Cursor, Codex, and other clients.

> Bringing Sim into your other AI tools.

### 4. Agent-facing metadata

The tool name and description matter — they help external agents know when to use the tool.
