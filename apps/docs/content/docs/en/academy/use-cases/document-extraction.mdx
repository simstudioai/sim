## What you'll build

A file comes in, an Agent block extracts the fields into a structured shape, and a Table block writes the row. Run it across a batch and a folder of documents becomes a queryable table.

This composes [files](/academy/files), [agents](/academy/agents), and [tables](/academy/tables).
