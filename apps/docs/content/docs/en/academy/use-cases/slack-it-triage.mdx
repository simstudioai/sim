## What you'll build

A Slack trigger starts the workflow on each new message. An Agent block classifies the request and decides the path: a knowledge-base search drafts an answer for common questions, while anything it can't resolve is routed to a human or filed as a ticket. Every run is logged.

This pulls together [workflows](/academy/workflows), [agents](/academy/agents), and [knowledge bases](/academy/knowledge-bases) into one system.
