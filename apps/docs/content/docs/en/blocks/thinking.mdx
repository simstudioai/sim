The Thinking block forces an AI model to explicitly outline its thought process before proceeding. By encouraging step-by-step analysis, this block can improve reasoning quality and output accuracy on complex tasks.

</Callout>

## Configuration Options

### Thought

The thought process or reasoning step that the model should document. This field is hidden from the UI but is required for internal processing.

- **Type**: String (Markdown supported)
- **Purpose**: Captures the model's explicit reasoning before generating the final output

## Outputs

- **`<thinking.acknowledgedThought>`**: The model's acknowledgment and processing of the outlined thought process

## How It Works

The Thinking block works by:

1. Accepting a thought or reasoning prompt
2. Instructing the AI model to explicitly work through the thought process
3. Returning the acknowledged thought as output that can feed into subsequent blocks

This encourages the model to break down complex reasoning into explicit steps, which can lead to more accurate and reliable outputs, especially for:

- **Multi-step problem solving**: Tasks requiring sequential reasoning
- **Complex analysis**: Situations where intermediate steps matter
- **Chain-of-thought reasoning**: Workflows that benefit from explicit step-by-step logic
- **Verification steps**: Adding explicit reasoning checks before final outputs

## Example Use Cases

**Complex Problem Solving with Reasoning Steps**
```
Input → Thinking (outline approach) → Agent (execute with reasoning) → Response
```

**Quality Assurance with Explicit Analysis**
```
Content → Agent (analyze) → Thinking (verify reasoning) → Function (validate) → Response
```

**Chain-of-Thought Verification**
```
Question → Thinking (break down problem) → Agent (solve step-by-step) → Response
```

## Best Practices

- **Use for complex tasks**: Apply Thinking blocks to workflows involving complex reasoning, multi-step logic, or high-accuracy requirements
- **Combine with Agent blocks**: Use Thinking in conjunction with Agent blocks to enhance their reasoning capabilities
- **Keep thoughts focused**: Write clear, specific thoughts that guide the model toward better reasoning
- **Monitor performance**: Test workflows with and without Thinking blocks to measure the impact on output quality

<Callout type="tip">
  For standard AI interactions, the Agent block alone is usually sufficient. The Thinking block is most valuable when dealing with complex reasoning tasks where explicit step-by-step analysis measurably improves results.
</Callout>
