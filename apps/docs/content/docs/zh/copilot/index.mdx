Copilot 是您编辑器中的助手，可帮助您构建和编辑工作流。它可以：

- **解释**：回答有关 Sim 和您当前工作流的问题
- **指导**：建议编辑和最佳实践
- **构建**：添加模块、连接变量和配置设置
- **调试**：分析执行问题并优化性能

<Callout type="info">
  Copilot 是一个由 Sim 管理的服务。对于自托管部署：
  1. 前往 [sim.ai](https://sim.ai) → 设置 → Copilot 并生成一个 Copilot API 密钥
  2. 在您的自托管环境中将 `COPILOT_API_KEY` 设置为该值
</Callout>

## 模型

使用输入区域右下方的模型选择器选择您首选的 AI 模型。

**可用模型：**
- **Anthropic**: Claude Opus 4.6（默认）、Claude Opus 4.5、Claude Sonnet 4.5、Claude Haiku 4.5
- **OpenAI**: GPT-5.4、GPT-5.2、GPT-5.1、GPT-5、GPT-4o、GPT-4.1、o3、o4-mini
- **Google**: Gemini 2.5 Pro、Gemini 2.5 Flash、Gemini 2.0 Flash
- **其他提供商**: Groq、Cerebras、DeepSeek、Mistral、xAI、Azure OpenAI、OpenRouter
- **本地模型**: Ollama 或 VLLM 兼容模型

根据您的需求选择：更快的模型用于简单任务，更强大的模型用于复杂工作流。

## 上下文菜单 (@)

使用 `@` 符号引用资源并为 Copilot 提供更多上下文：

| 引用 | 描述 |
|-----------|-------------|
| **聊天** | 之前与 Copilot 的对话 |
| **工作流** | 工作空间中的任何工作流 |
| **工作流模块** | 当前工作流中的模块 |
| **模块** | 模块类型和模板 |
| **知识** | 上传的文档和知识库 |
| **文档** | Sim 文档 |
| **模板** | 工作流模板 |
| **日志** | 执行日志和结果 |

在输入字段中输入 `@` 以打开上下文菜单，然后搜索或浏览以找到所需内容。

## 斜杠命令 (/)

使用斜杠命令进行快速操作：

| 命令 | 描述 |
|---------|-------------|
| `/fast` | 快速模式执行 |
| `/research` | 研究和探索模式 |
| `/actions` | 执行代理操作 |

**Web 命令：**

| 命令 | 描述 |
|---------|-------------|
| `/search` | 在网络上搜索 |
| `/read` | 读取特定 URL |
| `/scrape` | 抓取网页内容 |
| `/crawl` | 抓取多个页面 |

在输入字段中输入 `/` 以查看可用的命令。

## 聊天管理

### 开始新聊天

单击 Copilot 标题中的 **+** 按钮以开始新对话。

### 聊天历史

单击**历史记录**以查看按日期分组的以前的对话。您可以：
- 单击聊天以恢复它
- 删除您不再需要的聊天

### 编辑消息

将鼠标悬停在您的任何消息上，然后单击**编辑**以修改并重新发送。这对于完善您的提示很有用。

### 消息队列

如果您在 Copilot 仍在响应时发送消息，该消息会被加入队列。您可以：
- 在可展开的队列面板中查看排队的消息
- 立即发送排队消息（会中止当前响应）
- 从队列中移除消息

## 文件附件

点击附件图标即可随消息上传文件。支持的文件类型包括：
- 图片（显示预览缩略图）
- PDF
- 文本文件、 JSON 、 XML
- 其他文档格式

文件会以可点击的缩略图形式展示，点击后将在新标签页中打开。

## 检查点与更改

当 Copilot 对你的工作流进行更改时，会保存检查点，方便你在需要时回退。

### 查看检查点

将鼠标悬停在 Copilot 消息上，点击检查点图标即可查看该消息保存的工作流状态。

### 撤销更改

点击任意检查点的 **撤销**，即可将工作流恢复到该状态。确认对话框会提示此操作无法撤销。

### 接受更改

当 Copilot 提出更改时，你可以：
- **接受**：应用建议的更改（`Mod+Shift+Enter`）
- **拒绝**：忽略更改，保留当前工作流

## 思考块

对于复杂请求， Copilot 可能会在可展开的思考块中展示其推理过程：

- Copilot 思考时，思考块会自动展开
- 点击可手动展开/收起
- 显示思考过程的持续时间
- 帮助你理解 Copilot 如何得出解决方案

## 选项选择

当 Copilot 提供多个选项时，你可以通过以下方式选择：

| 控制键 | 操作 |
|---------|--------|
| **1-9** | 通过数字选择选项 |
| **向上/向下箭头** | 在选项间导航 |
| **Enter** | 选择高亮选项 |

已选中的选项会高亮显示，未选中的选项会显示删除线。

## 键盘快捷键

| 快捷键 | 操作 |
|----------|--------|
| `@` | 打开上下文菜单 |
| `/` | 打开斜杠命令 |
| `Arrow Up/Down` | 菜单项导航 |
| `Enter` | 选择菜单项 |
| `Esc` | 关闭菜单 |
| `Mod+Shift+Enter` | 接受 Copilot 更改 |

## 使用限制

Copilot 的使用按底层 LLM 的 token 计费，并计入您的计划的信用额度使用中。如果达到使用上限，请从"设置">"订阅"启用按需计费，以继续在计划包含的信用额度之外使用 Copilot。

  计费和计划详情请参见 [成本计算页面](/execution/costs)。

## Copilot MCP

你可以在常用编辑器或 AI 客户端中将 Copilot 作为 MCP 服务器使用。这样可以直接在 Cursor、Claude Code、Claude Desktop 和 VS Code 等工具中构建、测试、部署和管理 Sim 工作流。

### 生成 Copilot API 密钥

要连接 Copilot MCP 服务器，你需要一个 **Copilot API 密钥**：

1. 访问 [sim.ai](https://sim.ai) 并登录
2. 进入 **设置** → **Copilot**
3. 点击 **生成 API 密钥**
4. 复制密钥 — 该密钥只显示一次

密钥类似于 `sk-sim-copilot-...`。你将在下方配置中用到它。

### Cursor

将以下内容添加到你的 `.cursor/mcp.json`（项目级）或全局 Cursor MCP 设置中：

```json
{
  "mcpServers": {
    "sim-copilot": {
      "url": "https://www.sim.ai/api/mcp/copilot",
      "headers": {
        "X-API-Key": "YOUR_COPILOT_API_KEY"
      }
    }
  }
}
```

将 `YOUR_COPILOT_API_KEY` 替换为你上面生成的密钥。

### Claude Code

运行以下命令以添加 Copilot MCP 服务器：

```bash
claude mcp add sim-copilot \
  --transport http \
  https://www.sim.ai/api/mcp/copilot \
  --header "X-API-Key: YOUR_COPILOT_API_KEY"
```

将 `YOUR_COPILOT_API_KEY` 替换为你的密钥。

### Claude Desktop

Claude Desktop 需要 [`mcp-remote`](https://www.npmjs.com/package/mcp-remote) 以连接基于 HTTP 的 MCP 服务器。请将以下内容添加到你的 Claude Desktop 配置文件（macOS 上为 `~/Library/Application Support/Claude/claude_desktop_config.json`）：

```json
{
  "mcpServers": {
    "sim-copilot": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://www.sim.ai/api/mcp/copilot",
        "--header",
        "X-API-Key: YOUR_COPILOT_API_KEY"
      ]
    }
  }
}
```

将 `YOUR_COPILOT_API_KEY` 替换为你的密钥。

### VS Code

请将以下内容添加到你的 VS Code `settings.json` 或工作区 `.vscode/settings.json`：

```json
{
  "mcp": {
    "servers": {
      "sim-copilot": {
        "type": "http",
        "url": "https://www.sim.ai/api/mcp/copilot",
        "headers": {
          "X-API-Key": "YOUR_COPILOT_API_KEY"
        }
      }
    }
  }
}
```

将 `YOUR_COPILOT_API_KEY` 替换为你的密钥。
