## 您可以构建的内容

**AI 助手与聊天机器人**
创建能够搜索网络、访问日历、发送电子邮件以及与您的业务工具交互的智能代理。

**业务流程自动化**
自动化重复性任务，例如数据录入、报告生成、客户支持回复和内容创建。

**数据处理与分析**
从文档中提取洞察，分析数据集，生成报告，并在系统之间同步数据。

**API 集成工作流**
将多个服务连接到统一的端点，编排复杂的业务逻辑，并处理事件驱动的自动化。

## 工作原理

**可视化画布**
通过拖放模块来构建工作流。使用简单的点选连接将 AI 模型、数据库、API 和业务工具连接起来。

**智能模块**
从处理模块（AI 代理、API、函数）、逻辑模块（条件、循环、路由器）和输出模块（响应、评估器）中进行选择。

**多种触发方式**
通过聊天界面、REST API、webhooks、计划任务或来自 Slack 和 GitHub 等服务的外部事件启动工作流。

**团队协作**
与团队成员同时在同一工作流上工作，支持实时编辑和权限管理。

## 内置集成

Sim 开箱即用地连接到 80 多种服务：

- **AI 模型**：OpenAI、Anthropic、Google、Groq、Cerebras、本地 Ollama 模型
- **通信**：Gmail、Slack、Teams、Telegram、WhatsApp  
- **生产力**：Notion、Google Sheets、Airtable、Monday.com
- **开发**：GitHub、Jira、Linear、浏览器自动化
- **搜索与网络**：Google Search、Perplexity、Firecrawl、Exa AI
- **数据库**：PostgreSQL、MySQL、Supabase、Pinecone、Qdrant

需要定制功能？使用我们的 [MCP 集成](/mcp) 连接任何外部服务。

## 部署选项

**云托管**：在 [sim.ai](https://sim.ai) 上立即开始，享受托管基础设施、自动扩展和内置监控。

**自托管**：使用 Docker 在您自己的基础设施上部署，通过 Ollama 支持本地 AI 模型，确保数据完全隐私。
