# Документация для Sim
Добро пожаловать в Sim – платформу для разработки и управления ИИ, основанную на открытом исходном коде, где команды создают, развертывают и управляют агентами ИИ. Создавайте агентов визуально с помощью конструктора рабочих процессов, посредством интерфейса Mothership или программно через API — который интегрирован со более чем 1000 сервисами и всеми основными LLM.
