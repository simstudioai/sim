[ZoomInfo](https://www.zoominfo.com/) is a B2B go-to-market platform with one of the largest verified databases of company and contact data. It pairs deep firmographic and contact records with buyer intent signals and company news, so your sales, marketing, and operations teams can find, prioritize, and reach the right accounts.

With ZoomInfo, you can:

- **Search companies**: Find organizations by name, industry, location, revenue, and employee size
- **Search contacts**: Discover people by name, job title, department, and management level
- **Enrich companies and contacts**: Resolve detailed firmographics, verified emails, and phone numbers in bulk
- **Track buyer intent**: Surface companies showing intent signals on the topics you care about
- **Monitor company news**: Pull recent articles by category, source, or date range

In Sim, the ZoomInfo integration lets your agents power prospecting, enrichment, and account research workflows end-to-end:

- **Search Companies**: Use `zoominfo_search_companies` to find accounts matching firmographic filters.
- **Search Contacts**: Use `zoominfo_search_contacts` to find people by role and company. Search results do not include emails or phone numbers — use Enrich Contacts for engagement data.
- **Enrich Companies**: Use `zoominfo_enrich_companies` to resolve up to 25 companies per request from a name, domain, ticker, or ZoomInfo ID.
- **Enrich Contacts**: Use `zoominfo_enrich_contacts` to resolve up to 25 contacts per request, returning verified emails, phones, and job details.
- **Search Intent**: Use `zoominfo_search_intent` to find companies researching specific topics, with signal scores and recommended contacts.
- **Search News**: Use `zoominfo_search_news` to monitor company news by category, source URL, or publish date.

ZoomInfo authenticates with the OAuth 2.0 client credentials flow: Sim exchanges your client ID and secret for a short-lived bearer token automatically on each call, so you only ever provide your credentials. A common pattern is to search first to identify the records you want, then pass their IDs into the matching enrich tool to pull full detail efficiently.

## Инструкции по использованию

Integrates ZoomInfo в рабочий процесс. Search companies and contacts, enrich firmographic and contact data, find intent signals, and pull news — all using the ZoomInfo GTM API.



## Действия

### `zoominfo_search_companies`

Search the ZoomInfo company database by name, industry, location, and size.

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `clientId` | string | Да | ZoomInfo OAuth client ID |
| `clientSecret` | string | Да | ZoomInfo OAuth client secret |
| `companyName` | string | Нет | Company name to search for |
| `companyWebsite` | string | Нет | Company website \(comma-separated for multiple\) |
| `companyTicker` | string | Нет | Stock ticker symbols — JSON array, comma-separated list, or single ticker. Sent to the API as an array. |
| `industryCodes` | string | Нет | Industry codes — JSON array or comma-separated list. Sent to the API as a comma-separated string. |
| `country` | string | Нет | Country name |
| `state` | string | Нет | State or province |
| `metroRegion` | string | Нет | US/Canada metro region |
| `revenueMin` | number | Нет | Minimum annual revenue in thousands USD |
| `revenueMax` | number | Нет | Maximum annual revenue in thousands USD |
| `employeeRangeMin` | number | Нет | Minimum employee count |
| `employeeRangeMax` | number | Нет | Maximum employee count |
| `excludeDefunctCompanies` | boolean | Нет | Exclude inactive companies |
| `page` | number | Нет | Page number \(1-based\) |
| `rpp` | number | Нет | Results per page \(1-100, по умолчанию 25\) |
| `sortBy` | string | Нет | Field to sort by |
| `sortOrder` | string | Нет | Sort order \(asc or desc\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `companies` | array | Matching companies |

### `zoominfo_search_contacts`

Search ZoomInfo for contacts (people) by name, job title, company, and other filters. Does not return emails or phone numbers — use Enrich Contacts for engagement data.

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `clientId` | string | Да | ZoomInfo OAuth client ID |
| `clientSecret` | string | Да | ZoomInfo OAuth client secret |
| `firstName` | string | Нет | First name |
| `lastName` | string | Нет | Last name |
| `fullName` | string | Нет | Full name |
| `emailAddress` | string | Нет | Email address |
| `jobTitle` | string | Нет | Job title |
| `managementLevel` | string | Нет | Management level — JSON array or comma-separated list. Sent to the API as a comma-separated string. |
| `department` | string | Нет | Department — JSON array or comma-separated list. Sent to the API as a comma-separated string. |
| `companyId` | string | Нет | ZoomInfo company ID |
| `companyName` | string | Нет | Company name |
| `contactAccuracyScoreMin` | number | Нет | Minimum accuracy score \(70-99\) |
| `requiredFields` | string | Нет | Fields that must exist in results — JSON array or comma-separated list. Sent to the API as a comma-separated string. |
| `excludePartialProfiles` | boolean | Нет | Exclude partial profiles |
| `page` | number | Нет | Page number \(1-based\) |
| `rpp` | number | Нет | Results per page \(1-100, по умолчанию 25\) |
| `sortBy` | string | Нет | Field to sort by |
| `sortOrder` | string | Нет | Sort order \(asc or desc\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `contacts` | array | Matching contacts \(without emails or phone numbers\) |

### `zoominfo_enrich_companies`

Enrich up to 25 companies in one request with detailed firmographics, industry, financials, and more.

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `clientId` | string | Да | ZoomInfo OAuth client ID |
| `clientSecret` | string | Да | ZoomInfo OAuth client secret |
| `matchCompanyInput` | string | Да | JSON array \(1-25 items\) of company matching criteria, e.g. \[\{"companyName":"Acme","companyWebsite":"acme.com"\}\] |
| `outputFields` | string | Нет | JSON array or comma-separated list of fields to return (например, \["id","name","website","revenue","employeeCount"\]\). Defaults to a standard firmographic set if omitted. |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `results` | array | Enrichment results, one per input with match status and attributes |

### `zoominfo_enrich_contacts`

Enrich up to 25 contacts in one request with verified emails, phone numbers, job details, and more.

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `clientId` | string | Да | ZoomInfo OAuth client ID |
| `clientSecret` | string | Да | ZoomInfo OAuth client secret |
| `matchPersonInput` | string | Да | JSON array \(1-25 items\) of contact matching criteria, e.g. \[\{"firstName":"Jane","lastName":"Doe","companyName":"Acme"\}\] |
| `outputFields` | string | Нет | JSON array or comma-separated list of fields to return (например, \["id","firstName","email","phone","jobTitle"\]\). Defaults to a standard contact set if omitted. |
| `requiredFields` | string | Нет | JSON array or comma-separated list of fields that must exist in results (например, \["email"\]\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `results` | array | Enrichment results, one per input with match status and attributes |

### `zoominfo_search_intent`

Search for companies showing intent signals on specific topics.

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `clientId` | string | Да | ZoomInfo OAuth client ID |
| `clientSecret` | string | Да | ZoomInfo OAuth client secret |
| `topics` | string | Да | Up to 50 intent topics as JSON array or comma-separated list (например, \["CRM Software","Marketing Automation"\]\) |
| `signalStartDate` | string | Нет | Earliest signal date \(YYYY-MM-DD\) |
| `signalEndDate` | string | Нет | Latest signal date \(YYYY-MM-DD\) |
| `signalScoreMin` | number | Нет | Minimum signal score \(60-100\) |
| `signalScoreMax` | number | Нет | Maximum signal score \(60-100\) |
| `audienceStrengthMin` | string | Нет | Minimum audience strength \(A-E, A is largest\) |
| `audienceStrengthMax` | string | Нет | Maximum audience strength \(A-E, A is largest\) |
| `findRecommendedContacts` | boolean | Нет | Include recommended contacts \(по умолчанию true\) |
| `country` | string | Нет | Country filter |
| `state` | string | Нет | State filter |
| `industryCodes` | string | Нет | Industry codes — JSON array or comma-separated list. Sent to the API as a comma-separated string. |
| `page` | number | Нет | Page number \(1-based\) |
| `rpp` | number | Нет | Results per page \(1-100, по умолчанию 25\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `signals` | array | Intent signals with topic, score, audience strength, and company |

### `zoominfo_search_news`

Search ZoomInfo news articles by category, URL, or date range.

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `clientId` | string | Да | ZoomInfo OAuth client ID |
| `clientSecret` | string | Да | ZoomInfo OAuth client secret |
| `categories` | string | Нет | News categories as JSON array or comma-separated list |
| `url` | string | Нет | News source URLs as JSON array or comma-separated list |
| `pageDateMin` | string | Нет | Earliest publish date \(YYYY-MM-DD\) |
| `pageDateMax` | string | Нет | Latest publish date \(YYYY-MM-DD\) |
| `page` | number | Нет | Page number \(1-based\) |
| `rpp` | number | Нет | Results per page \(1-100, по умолчанию 25\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `articles` | array | News articles matching the filters |
