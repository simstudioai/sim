[Serper](https://www.serper.com/) — это API Google Поиска, который предоставляет разработчикам программный доступ к результатам поиска Google. Он предлагает надёжный и высокопроизводительный способ интеграции поисковых возможностей Google в приложения без сложностей веб-скрапинга или ограничений других поисковых API.

С помощью Serper вы можете:

- **Получать результаты поиска Google**: Получайте структурированные данные из результатов поиска Google программным способом
- **Выполнять различные типы поиска**: Выполняйте веб-поиск, поиск изображений, поиск новостей и многое другое
- **Извлекать расширенные метаданные**: Получайте заголовки, сниппеты, URL-адреса и другую релевантную информацию из результатов поиска
- **Масштабировать приложения**: Создавайте функции на основе поиска с помощью надёжного и быстрого API
- **Избегать ограничений частоты запросов**: Получайте стабильный доступ к результатам поиска, не беспокоясь о блокировке IP-адресов

В Sim интеграция с Serper позволяет вашим агентам использовать возможности веб-поиска в рамках своих рабочих процессов. Это даёт возможность создавать сложные сценарии автоматизации, требующие актуальной информации из интернета. Ваши агенты могут формулировать поисковые запросы, извлекать релевантные результаты и использовать эту информацию для принятия решений или предоставления ответов. Эта интеграция устраняет разрыв между автоматизацией рабочих процессов и обширными знаниями, доступными в интернете, позволяя вашим агентам получать доступ к актуальной информации без ручного вмешательства. Подключив Sim к Serper, вы можете создавать агентов, которые остаются в курсе последней информации, предоставляют более точные ответы и приносят больше пользы пользователям.

## Инструкция по использованию

Интегрируйте Serper в рабочий процесс. Позволяет выполнять поиск в интернете.



## Действия

### `serper_search`

Мощный инструмент веб-поиска, который предоставляет доступ к результатам поиска Google через API Serper.dev. Поддерживает различные типы поиска, включая обычный веб-поиск, новости, места, изображения, видео и покупки. Возвращает комплексные результаты, включая органические результаты, граф знаний, блок ответов, блок «Люди также спрашивают», связанные запросы и главные новости.

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `query` | string | Да | Поисковый запрос \(например, "последние новости ИИ", "лучшие рестораны в Москве"\) |
| `num` | number | Нет | Количество возвращаемых результатов \(например, 10, 20, 50\) |
| `gl` | string | Нет | Код страны для результатов поиска \(например, "us", "uk", "de", "fr"\) |
| `hl` | string | Нет | Код языка для результатов поиска \(например, "en", "es", "de", "fr"\) |
| `type` | string | Нет | Тип поиска \(например, "search", "news", "images", "videos", "places", "shopping"\) |
| `apiKey` | string | Да | Ключ API Serper |
| `pricing` | custom | Нет | Нет описания |
| `rateLimit` | string | Нет | Нет описания |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `searchResults` | array | Результаты поиска с заголовками, ссылками, сниппетами и метаданными по типу \(дата для новостей, рейтинг для мест, imageUrl для изображений\) |
|   ↳ `title` | string | Заголовок результата |
|   ↳ `link` | string | URL результата |
|   ↳ `snippet` | string | Описание/сниппет результата |
|   ↳ `position` | number | Позиция в результатах поиска |
|   ↳ `date` | string | Дата публикации \(новости/видео\) |
|   ↳ `imageUrl` | string | URL изображения \(изображения/новости/покупки\) |
|   ↳ `source` | string | Название источника \(новости/видео/покупки\) |
|   ↳ `rating` | number | Рейтинг \(места\) |
|   ↳ `ratingCount` | number | Количество отзывов \(места\) |
|   ↳ `address` | string | Адрес \(места\) |
|   ↳ `price` | string | Цена \(покупки\) |
|   ↳ `duration` | string | Продолжительность \(видео\) |
