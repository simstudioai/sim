[Polymarket](https://polymarket.com) is a decentralized prediction markets platform where users can trade on the outcome of future events using blockchain technology. Polymarket provides a comprehensive API, enabling developers and agents to access live market data, event listings, price information, and orderbook statistics to power data-driven workflows and AI automations.

With Polymarket’s API and Sim integration, you can enable agents to programmatically retrieve prediction market information, explore open markets and associated events, analyze historical price data, and access orderbooks and market midpoints. This creates new possibilities for research, automated analysis, and developing intelligent agents that react to real-time event probabilities derived from market prices.

Key features of the Polymarket integration include:

- **Market Listing & Filtering:** Получение списка current or historical prediction markets, filter by tag, sort, and paginate through results.
- **Market Detail:** Получение details for a single market by market ID or slug, including its outcomes and status.
- **Event Listings:** Access lists of Polymarket events and detailed event information.
- **Orderbook & Price Data:** Analyze the orderbook, get the latest market prices, view the midpoint, or obtain historical price information for any market.
- **Automation Ready:** Build agents or tools that react programmatically to market developments, changing odds, or specific event outcomes.

By using these documented API endpoints, you can seamlessly integrate Polymarket’s rich on-chain prediction market data into your own AI workflows, dashboards, research tools, and trading automations.

## Инструкции по использованию

Интегрируйте Polymarket prediction markets в рабочий процесс. Can get markets, market, events, event, tags, series, orderbook, price, midpoint, price history, last trade price, spread, tick size, positions, trades, activity, leaderboard, holders, and search.



## Действия

### `polymarket_get_markets`

Получение a list of prediction markets from Polymarket with optional filtering

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `closed` | string | Нет | Filter by closed status \(true/false\). Use false for open markets only. |
| `order` | string | Нет | Sort field (например, volumeNum, liquidityNum, startDate, endDate, createdAt\) |
| `ascending` | string | Нет | Sort direction \(true for ascending, false for descending\) |
| `tagId` | string | Нет | Filter by tag ID |
| `limit` | string | Нет | Number of results per page (например, "25"\). Max: 50. |
| `offset` | string | Нет | Number of results to skip for pagination (например, "50"\). |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `markets` | array | Array of market objects |
|   ↳ `id` | string | Market ID |
|   ↳ `question` | string | Market question |
|   ↳ `conditionId` | string | Condition ID |
|   ↳ `slug` | string | Market slug |
|   ↳ `endDate` | string | End date |
|   ↳ `image` | string | Market image URL |
|   ↳ `outcomes` | string | Outcomes JSON string |
|   ↳ `outcomePrices` | string | Outcome prices JSON string |
|   ↳ `volume` | string | Total volume |
|   ↳ `liquidity` | string | Total liquidity |
|   ↳ `active` | boolean | Whether market is active |
|   ↳ `closed` | boolean | Whether market is closed |
|   ↳ `volumeNum` | number | Volume as number |
|   ↳ `liquidityNum` | number | Liquidity as number |
|   ↳ `clobTokenIds` | array | CLOB token IDs |

### `polymarket_search`

Search for markets, events, and profiles on Polymarket

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `query` | string | Да | Search query term (например, "presidential election", "bitcoin price"\). |

| `page` | string | Нет | Page number for pagination (например, "2"\). 1-indexed. |
| `cache` | string | Нет | Enable caching \(true/false\) |
| `eventsStatus` | string | Нет | Filter events by status |
| `eventsTag` | string | Нет | Filter by event tags \(comma-separated\) |
| `sort` | string | Нет | Sort field |
| `ascending` | string | Нет | Sort direction \(true for ascending, false for descending\) |
| `searchTags` | string | Нет | Include tags in search results \(true/false\) |
| `searchProfiles` | string | Нет | Include profiles in search results \(true/false\) |
| `recurrence` | string | Нет | Filter by recurrence type |
| `excludeTagId` | string | Нет | Exclude events with these tag IDs \(comma-separated\) |
| `keepClosedMarkets` | string | Нет | Include closed markets in results \(0 or 1\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `results` | object | Search results containing events, tags, and profiles arrays |
|   ↳ `events` | array | Array of matching event objects \(markets nested\) |
|   ↳ `tags` | array | Array of matching tag objects |
|   ↳ `profiles` | array | Array of matching profile objects |
