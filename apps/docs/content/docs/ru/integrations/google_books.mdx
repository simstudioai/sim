[Google Books](https://books.google.com) — это комплексный сервис Google для поиска и получения метаданных книг, предоставляющий доступ к миллионам книг от издателей, библиотек и оцифрованных коллекций по всему миру.

С интеграцией Google Books в Sim вы можете:

- **Искать книги**: находить издания по названию, автору, ISBN или ключевым словам во всём каталоге Google Books
- **Получать сведения об издании**: получать подробные метаданные для конкретной книги, включая название, авторов, описание, рейтинги и информацию о публикации

В Sim интеграция Google Books позволяет вашим агентам искать книги и получать сведения об изданиях в рамках автоматизированных рабочих процессов. Это открывает такие сценарии использования, как исследование контента, составление списков для чтения, обогащение библиографических данных и сбор знаний из опубликованных работ.

## Инструкции по использованию

Поиск книг с помощью Google Books API. Находите издания по названию, автору, ISBN или ключевым словам, а также получайте подробную информацию о конкретных книгах, включая описания, рейтинги и сведения о публикации.



## Действия

### `google_books_volume_search`

Поиск книг с помощью Google Books API

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Да | Ключ API Google Books |
| `query` | string | Да | Поисковый запрос. Поддерживаются специальные ключевые слова: intitle:, inauthor:, inpublisher:, subject:, isbn: |
| `filter` | string | Нет | Фильтрация результатов по доступности \(partial, full, free-ebooks, paid-ebooks, ebooks\) |
| `printType` | string | Нет | Ограничение по типу издания \(all, books, magazines\) |
| `orderBy` | string | Нет | Порядок сортировки \(relevance, newest\) |
| `startIndex` | number | Нет | Индекс первого возвращаемого результата \(для пагинации\) |
| `maxResults` | number | Нет | Максимальное количество возвращаемых результатов \(1-40\) |
| `langRestrict` | string | Нет | Ограничение результатов определённым языком \(код ISO 639-1\) |
| `pricing` | per_request | Нет | Нет описания |
| `rateLimit` | string | Нет | Нет описания |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `totalItems` | number | Общее количество совпадающих результатов |
| `volumes` | array | Список найденных изданий |
|   ↳ `id` | string | Идентификатор издания |
|   ↳ `title` | string | Название книги |
|   ↳ `subtitle` | string | Подзаголовок книги |
|   ↳ `authors` | array | Список авторов |
|   ↳ `publisher` | string | Название издателя |
|   ↳ `publishedDate` | string | Дата публикации |
|   ↳ `description` | string | Описание книги |
|   ↳ `pageCount` | number | Количество страниц |
|   ↳ `categories` | array | Категории книги |
|   ↳ `averageRating` | number | Средний рейтинг \(1-5\) |
|   ↳ `ratingsCount` | number | Количество оценок |
|   ↳ `language` | string | Код языка |
|   ↳ `previewLink` | string | Ссылка на предпросмотр в Google Books |
|   ↳ `infoLink` | string | Ссылка на страницу с информацией |
|   ↳ `thumbnailUrl` | string | URL миниатюры обложки книги |
|   ↳ `isbn10` | string | Идентификатор ISBN-10 |
|   ↳ `isbn13` | string | Идентификатор ISBN-13 |

### `google_books_volume_details`

Получение подробной информации о конкретном издании книги

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Да | Ключ API Google Books |
| `volumeId` | string | Да | Идентификатор издания для получения |
| `projection` | string | Нет | Уровень проекции \(full, lite\) |
| `pricing` | per_request | Нет | Нет описания |
| `rateLimit` | string | Нет | Нет описания |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `id` | string | Идентификатор издания |
| `title` | string | Название книги |
| `subtitle` | string | Подзаголовок книги |
| `authors` | array | Список авторов |
| `publisher` | string | Название издателя |
| `publishedDate` | string | Дата публикации |
| `description` | string | Описание книги |
| `pageCount` | number | Количество страниц |
| `categories` | array | Категории книги |
| `averageRating` | number | Средний рейтинг \(1-5\) |
| `ratingsCount` | number | Количество оценок |
| `language` | string | Код языка |
| `previewLink` | string | Ссылка на предпросмотр в Google Books |
| `infoLink` | string | Ссылка на страницу с информацией |
| `thumbnailUrl` | string | URL миниатюры обложки книги |
| `isbn10` | string | Идентификатор ISBN-10 |
| `isbn13` | string | Идентификатор ISBN-13 |
