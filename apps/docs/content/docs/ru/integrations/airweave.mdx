[Airweave](https://airweave.ai/) — это платформа семантического поиска на базе ИИ, которая помогает находить и извлекать знания из всех синхронизированных источников данных. Созданная для современных команд, Airweave обеспечивает быстрые и релевантные результаты поиска с использованием нейронных, гибридных или ключевых стратегий, адаптированных под ваши потребности.

С Airweave вы можете:

- **Искать эффективнее**: Используйте запросы на естественном языке для поиска информации, хранящейся в подключённых инструментах и базах данных
- **Объединять данные**: Бесшовный доступ к контенту из таких источников, как код, документация, чаты, электронная почта, облачные файлы и многое другое
- **Настраивать поиск**: Выбирайте между гибридной (семантическая + ключевая), нейронной или ключевой стратегией поиска для оптимальных результатов
- **Повышать полноту выдачи**: Расширяйте поисковые запросы с помощью ИИ для получения более полных ответов
- **Переранжировать результаты с помощью ИИ**: Приоритизируйте наиболее релевантные ответы с помощью мощных языковых моделей
- **Получать мгновенные ответы**: Генерируйте чёткие ответы на базе ИИ, синтезированные из ваших данных

В Sim интеграция с Airweave позволяет вашим агентам искать, обобщать и извлекать инсайты из всех данных организации с помощью одного инструмента. Используйте Airweave для насыщенного контекстного поиска знаний в рамках ваших рабочих процессов — будь то ответы на вопросы, создание сводок или поддержка динамического принятия решений.

## Инструкция по использованию

Поиск по синхронизированным источникам данных с помощью Airweave. Поддерживает семантический поиск с гибридной, нейронной или ключевой стратегией извлечения. Возможна генерация ответов на базе ИИ по результатам поиска.



## Действия

### `airweave_search`

Поиск по синхронизированным коллекциям данных с помощью Airweave. Поддерживает семантический поиск с гибридной, нейронной или ключевой стратегией извлечения. Возможна генерация ответов на базе ИИ по результатам поиска.

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Yes | API-ключ Airweave для аутентификации |
| `collectionId` | string | Yes | Читаемый идентификатор коллекции для поиска |
| `query` | string | Yes | Текст поискового запроса |
| `limit` | number | No | Максимальное количество возвращаемых результатов \(по умолчанию: 100\) |
| `retrievalStrategy` | string | No | Стратегия извлечения: hybrid \(по умолчанию\), neural или keyword |
| `expandQuery` | boolean | No | Генерация вариантов запроса для улучшения полноты выдачи |
| `rerank` | boolean | No | Переупорядочивание результатов для повышения релевантности с помощью LLM |
| `generateAnswer` | boolean | No | Генерация ответа на естественном языке на запрос |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `results` | array | Результаты поиска с контентом, оценками и метаданными из синхронизированных данных |
|   ↳ `entity_id` | string | Уникальный идентификатор сущности результата поиска |
|   ↳ `source_name` | string | Название источника данных \(например, "GitHub", "Slack"\) |
|   ↳ `md_content` | string | Содержимое результата в формате Markdown |
|   ↳ `score` | number | Оценка релевантности поиска |
|   ↳ `metadata` | object | Дополнительные метаданные, связанные с результатом |
|   ↳ `breadcrumbs` | array | Путь навигации к результату внутри его источника |
|   ↳ `url` | string | URL-адрес исходного контента |
| `completion` | string | Сгенерированный ИИ ответ на запрос \(если включена опция generateAnswer\) |
