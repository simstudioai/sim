[Airweave](https://airweave.ai/)Airweavehttps://airweave.ai/является платформой для семантического поиска на основе искусственного интеллекта, которая помогает вам находить и извлекать информацию из всех ваших источников данных, синхронизированных с Airweave. Разработанная для современных команд, Airweave обеспечивает быстрые и релевантные результаты поиска, используя нейронные, гибридные или стратегии, основанные на ключевых словах, адаптированные к вашим потребностям.

С помощью Airweave вы можете:

- **Ищите более эффективно:** Используйте запросы на естественном языке для поиска информации, хранящейся в ваших подключенных инструментах и базах данных.- **Объедините свои данные:** Беспрепятственный доступ к контенту из различных источников, таких как код, документация, чаты, электронная почта, облачные файлы и другие.- **Настройте параметры поиска:** Выберите между гибридными (семантическими + ключевыми словами) или нейронными стратегиями поиска, а также поиском по ключевым словам, чтобы получить наилучшие результаты.- **Улучшите запоминание**: Расширьте поисковые запросы с помощью ИИ для получения более полных ответов.- **Переранжирование результатов с использованием ИИ:** Приоритизация наиболее релевантных ответов с помощью мощных языковых моделей.- **Получите мгновенные ответы:** Создавайте четкие ответы на основе искусственного интеллекта, синтезированные из ваших данных.
В Sim интеграция с Airweave позволяет вашим сотрудникам искать, обобщать и извлекать информацию из всех данных вашей организации, используя единый инструмент. Используйте Airweave для получения богатой, контекстно-зависимой информации в рамках ваших рабочих процессов – будь то ответы на вопросы, создание сводок или поддержка принятия решений в режиме реального времени.

## Инструкция по применению
Ищите информацию в ваших синхронизированных источниках данных с помощью Airweave. Поддерживает семантический поиск с использованием гибридных, нейронных или поисковых стратегий на основе ключевых слов. При необходимости можно генерировать ответы на основе результатов поиска, использующие искусственный интеллект.



## Действия
### `airweave_search`

Ищите данные в ваших синхронизированных коллекциях с помощью Airweave. Поддерживает семантический поиск с использованием гибридных, нейронных или поисковых стратегий на основе ключевых слов. При необходимости можно генерировать ответы на основе результатов поиска, использующие искусственный интеллект.

#### Ввод
|  | Параметр | Тип | Необходимо | Описание |  || --------- | ---- | -------- | ----------- |
|  | `apiKey` | строка | Да | Ключ API Airweave для аутентификации |  ||  | `collectionId` | строка | Да | Читаемый идентификатор коллекции для поиска |  ||  | `query` | строка | Да | Текст поискового запроса |  ||  | `limit` | число | Нет | Максимальное количество результатов, которые необходимо вернуть (по умолчанию: 100) |  ||  | `retrievalStrategy` | строка | Нет | Стратегия извлечения: гибридная (по умолчанию), нейронная или по ключевым словам. |  ||  | `expandQuery` | логическое значение | Нет | Создавайте различные варианты запросов для повышения точности поиска. |  ||  | `rerank` | логическое значение | Нет | Переупорядочьте результаты для повышения релевантности с использованием модели LLM. |  ||  | `generateAnswer` | логическое значение | Нет | Сгенерируйте ответ на запрос, используя естественный язык. |  |
#### Вывод
|  | Параметр | Тип | Описание |  || --------- | ---- | ----------- |
|  | `results` | массив | Результаты поиска с содержанием, оценками и метаданными из ваших синхронизированных данных. |  ||  | ↳ `entity_id` | строка | Уникальный идентификатор для сущности результата поиска |  ||  | ↳ `source_name` | строка | Название источника данных (например, «GitHub», «Slack») |  ||  | ↳ `md_content` | строка | Содержимое результата в формате Markdown |  ||  | ↳ `score` | число | Оценка релевантности из результатов поиска |  ||  | ↳ `metadata` | объект | Дополнительные метаданные, связанные с результатом. |  ||  | ↳ `breadcrumbs` | массив | Путь навигации к результату внутри его источника. |  ||  | ↳ `url` | строка | Ссылка на исходный контент |  ||  | `completion` | строка | Ответ, сгенерированный ИИ, в ответ на запрос (когда включена функция `generateAnswer`) |  |
