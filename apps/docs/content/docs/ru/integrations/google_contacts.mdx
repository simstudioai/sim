## Инструкции по использованию

Интеграция Google Contacts в рабочий процесс. Позволяет создавать, читать, обновлять, удалять, просматривать списки и искать контакты.



## Действия

### `google_contacts_create`

Создание нового контакта в Google Contacts

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `givenName` | string | Да | Имя контакта |
| `familyName` | string | Нет | Фамилия контакта |
| `email` | string | Нет | Адрес электронной почты контакта |
| `emailType` | string | Нет | Тип email: home, work или other |
| `phone` | string | Нет | Номер телефона контакта |
| `phoneType` | string | Нет | Тип телефона: mobile, home, work или other |
| `organization` | string | Нет | Название организации/компании |
| `jobTitle` | string | Нет | Должность в организации |
| `notes` | string | Нет | Заметки или биография контакта |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `content` | string | Сообщение с подтверждением создания контакта |
| `metadata` | json | Метаданные созданного контакта, включая имя ресурса и детали |

### `google_contacts_get`

Получение конкретного контакта из Google Contacts

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `resourceName` | string | Да | Имя ресурса контакта \(например, people/c1234567890\) |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `content` | string | Сообщение с подтверждением получения контакта |
| `metadata` | json | Детали контакта, включая имя, email, телефон и организацию |

### `google_contacts_list`

Получение списка контактов из Google Contacts

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `pageSize` | number | Нет | Количество возвращаемых контактов \(1-1000, по умолчанию 100\) |
| `pageToken` | string | Нет | Токен страницы из предыдущего запроса списка для пагинации |
| `sortOrder` | string | Нет | Порядок сортировки контактов |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `content` | string | Сводка с количеством найденных контактов |
| `metadata` | json | Список контактов с токенами пагинации |

### `google_contacts_search`

Поиск контактов в Google Contacts по имени, email, телефону или организации

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `query` | string | Да | Поисковый запрос для сопоставления с именами, адресами электронной почты, телефонами и организациями контактов |
| `pageSize` | number | Нет | Количество возвращаемых результатов \(по умолчанию 10, макс. 30\) |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `content` | string | Сводка с количеством результатов поиска |
| `metadata` | json | Результаты поиска с найденными контактами |

### `google_contacts_update`

Обновление существующего контакта в Google Contacts

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `resourceName` | string | Да | Имя ресурса контакта \(например, people/c1234567890\) |
| `etag` | string | Да | ETag из предыдущего запроса get \(требуется для контроля параллельного доступа\) |
| `givenName` | string | Нет | Обновлённое имя |
| `familyName` | string | Нет | Обновлённая фамилия |
| `email` | string | Нет | Обновлённый адрес электронной почты |
| `emailType` | string | Нет | Тип email: home, work или other |
| `phone` | string | Нет | Обновлённый номер телефона |
| `phoneType` | string | Нет | Тип телефона: mobile, home, work или other |
| `organization` | string | Нет | Обновлённое название организации/компании |
| `jobTitle` | string | Нет | Обновлённая должность |
| `notes` | string | Нет | Обновлённые заметки или биография |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `content` | string | Сообщение с подтверждением обновления контакта |
| `metadata` | json | Метаданные обновлённого контакта |

### `google_contacts_delete`

Удаление контакта из Google Contacts

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `resourceName` | string | Да | Имя ресурса удаляемого контакта \(например, people/c1234567890\) |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `content` | string | Сообщение с подтверждением удаления контакта |
| `metadata` | json | Детали удаления, включая имя ресурса |
