Инструмент [Pulse](https://www.runpulse.com) обеспечивает удобное извлечение текста и структурированного содержимого из широкого спектра документов — включая PDF-файлы, изображения и файлы Office — с использованием современного OCR (оптического распознавания символов) на базе Pulse. Разработанный для автоматизированных агентных рабочих процессов, Pulse Parser позволяет легко извлекать ценную информацию из неструктурированных документов и интегрировать извлечённое содержимое непосредственно в ваш рабочий процесс.

С Pulse вы можете:

- **Извлекать текст из документов**: Быстро преобразовывать отсканированные PDF-файлы, изображения и документы Office в пригодный для использования текст, markdown или JSON.
- **Обрабатывать документы по URL или через загрузку**: Просто укажите URL файла или используйте загрузку для извлечения текста из локальных документов или удалённых ресурсов.
- **Гибкие форматы вывода**: Выбирайте между представлениями извлечённого содержимого в формате markdown, обычного текста или JSON для последующей обработки.
- **Выборочная обработка страниц**: Укажите диапазон страниц для обработки, сокращая время и стоимость, когда вам нужна только часть документа.
- **Извлечение рисунков и таблиц**: При необходимости извлекайте рисунки и таблицы с автоматической генерацией подписей и описаний для наполненного контекста.
- **Получение сведений об обработке**: Получайте подробные метаданные о каждом задании, включая тип файла, количество страниц, время обработки и многое другое.
- **Ответы, готовые к интеграции**: Включайте извлечённое содержимое в исследовательские процессы, автоматизацию рабочих процессов или конвейеры анализа данных.

Идеально подходит для автоматизации утомительного просмотра документов, summarization содержимого, исследований и многого другого — Pulse Parser переносит реальные документы в эпоху цифровых рабочих процессов.

Если вам нужны точные, масштабируемые и удобные для разработчиков возможности разбора документов — независимо от форматов, языков и макетов — Pulse даёт вашим агентам возможность читать мир.

## Инструкции по использованию

Интегрируйте Pulse в рабочий процесс. Извлекайте текст из PDF-документов, изображений и файлов Office через загрузку или ссылки на файлы.



## Действия

### `pulse_parser`

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `file` | file | Да | Документ для обработки |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `markdown` | string | Извлечённое содержимое в формате markdown |
| `page_count` | number | Количество страниц в документе |
| `job_id` | string | Уникальный идентификатор задания |
| `bounding_boxes` | json | Информация о расположении ограничивающих рамок |
| `extraction_url` | string | URL для результатов извлечения \(для больших документов\) |
| `html` | string | HTML-содержимое, если было запрошено |
| `structured_output` | json | Структурированный вывод, если была предоставлена схема |
| `chunks` | json | Фрагментированное содержимое, если было включено разбиение на фрагменты |
| `figures` | json | Извлечённые рисунки, если было включено извлечение рисунков |
