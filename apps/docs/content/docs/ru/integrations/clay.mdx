[Clay](https://www.clay.com/) — это платформа обогащения данных и автоматизации рабочих процессов, разработанная для помощи командам в оптимизации лидогенерации, исследований и других операций с данными с использованием мощных интеграций и гибких вариантов ввода.

В Sim интеграция с Clay позволяет вашим агентам легко вставлять структурированные данные в рабочие книги Clay через вебхук-триггеры. Это упрощает сбор, обогащение и управление динамическими результатами — такими как лиды, сводки исследований или элементы действий — непосредственно в интерфейсе, похожем на электронную таблицу, с возможностями совместной работы.

С помощью Clay вы можете:

- **Обогащать выводы агентов**: автоматически передавать данные агентов Sim в таблицы Clay для структурированного отслеживания и анализа.
- **Запускать рабочие процессы через вебхуки**: использовать поддержку вебхуков Clay для инициации задач агентов Sim непосредственно из Clay или отправки данных агентов в Clay как часть вашего рабочего процесса.
- **Использовать циклы данных**: легко итерировать по обогащённым строкам данных с агентами, работающими с динамическими наборами данных.

Интеграция поддерживает рабочие процессы, в которых ваши агенты заполняют строки в реальном времени, обеспечивая асинхронную совместную работу. Независимо от того, автоматизируете ли вы исследования, обогащаете данные CRM или отслеживаете операционные результаты, Clay становится живым слоем данных, который интеллектуально взаимодействует с вашими агентами. Подключая Sim к Clay, вы можете операционализировать результаты, созданные агентами, автоматизировать обработку наборов данных и поддерживать проверяемую, актуальную запись работы, управляемой ИИ.

## Инструкции по использованию

Интегрируйте Clay в рабочий процесс. Может заполнять таблицу данными.



## Действия

### `clay_populate`

Заполняет Clay данными из JSON-файла. Обеспечивает прямую связь и уведомления с отслеживанием временных меток и подтверждением канала.

#### Входные параметры

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `webhookURL` | string | Yes | URL вебхука для заполнения |
| `data` | json | Yes | Данные для заполнения |
| `authToken` | string | No | Опциональный токен аутентификации для вебхука Clay \(большинство вебхуков не требуют этого\) |

#### Выходные параметры

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `data` | json | Данные ответа от вебхука Clay |
| `metadata` | object | Метаданные ответа вебхука |
|   ↳ `status` | number | HTTP-код статуса |
|   ↳ `statusText` | string | Текст HTTP-статуса |
|   ↳ `headers` | object | Заголовки ответа от Clay |
|   ↳ `timestamp` | string | ISO-временная метка получения вебхука |
|   ↳ `contentType` | string | Тип содержимого ответа |
