## Инструкции по использованию

Upload, download, list, and manage files on remote servers via SFTP. Supports both password and private key authentication for secure file transfers.



## Действия

### `sftp_upload`

Upload files to a remote SFTP server

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `host` | string | Да | SFTP server hostname or IP address |
| `port` | number | Да | SFTP server port \(по умолчанию: 22\) |
| `username` | string | Да | SFTP username |
| `password` | string | Нет | Password for authentication \(if not using private key\) |
| `privateKey` | string | Нет | Private key for authentication \(OpenSSH format\) |
| `passphrase` | string | Нет | Passphrase for encrypted private key |
| `remotePath` | string | Да | Destination directory on the remote server |
| `files` | file[] | Нет | Files to upload |
| `fileContent` | string | Нет | Direct file content to upload \(for text files\) |
| `fileName` | string | Нет | File name when using direct content |
| `overwrite` | boolean | Нет | Whether to overwrite existing files \(по умолчанию: true\) |
| `permissions` | string | Нет | File permissions (например, 0644\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Указывает,  upload was successful |
| `uploadedFiles` | json | Array of uploaded file details \(name, remotePath, size\) |
| `message` | string | Статус операции message |

### `sftp_download`

Download a file from a remote SFTP server

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `host` | string | Да | SFTP server hostname or IP address |
| `port` | number | Да | SFTP server port \(по умолчанию: 22\) |
| `username` | string | Да | SFTP username |
| `password` | string | Нет | Password for authentication \(if not using private key\) |
| `privateKey` | string | Нет | Private key for authentication \(OpenSSH format\) |
| `passphrase` | string | Нет | Passphrase for encrypted private key |
| `remotePath` | string | Да | Path to the file on the remote server |
| `encoding` | string | Нет | Output encoding: utf-8 for text, base64 for binary \(по умолчанию: utf-8\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Указывает,  download was successful |
| `file` | file | Downloaded file stored in execution files |
| `fileName` | string | Name of the downloaded file |
| `content` | string | File content \(text or base64 encoded\) |
| `size` | number | File size in bytes |
| `encoding` | string | Content encoding \(utf-8 or base64\) |
| `message` | string | Статус операции message |

### `sftp_list`

Получение списка files and directories on a remote SFTP server

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `host` | string | Да | SFTP server hostname or IP address |
| `port` | number | Да | SFTP server port \(по умолчанию: 22\) |
| `username` | string | Да | SFTP username |
| `password` | string | Нет | Password for authentication \(if not using private key\) |
| `privateKey` | string | Нет | Private key for authentication \(OpenSSH format\) |
| `passphrase` | string | Нет | Passphrase for encrypted private key |
| `remotePath` | string | Да | Directory path on the remote server |
| `detailed` | boolean | Нет | Include detailed file information \(size, permissions, modified date\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Указывает,  operation was successful |
| `path` | string | Directory path that was listed |
| `entries` | json | Array of directory entries with name, type, size, permissions, modifiedAt |
| `count` | number | Number of entries in the directory |
| `message` | string | Статус операции message |
