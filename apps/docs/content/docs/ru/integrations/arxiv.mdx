[ArXiv](https://arxiv.org/) — это бесплатный открытый репозиторий научных исследовательских работ в таких областях, как физика, математика, информатика, количественная биология, количественные финансы, статистика, электротехника, системные науки и экономика. ArXiv предоставляет обширную коллекцию препринтов и опубликованных статей, что делает его основным ресурсом для исследователей и практиков по всему миру.

С помощью ArXiv вы можете:

- **Искать научные публикации**: находите исследования по ключевым словам, именам авторов, названиям, категориям и другим параметрам
- **Получать метаданные публикаций**: доступ к аннотациям, спискам авторов, датам публикации и другой библиографической информации
- **Скачивать полные тексты в PDF**: получайте полный текст большинства статей для углублённого изучения
- **Изучать вклад авторов**: просматривайте все публикации конкретного автора
- **Быть в курсе новинок**: открывайте последние поступления и актуальные темы в вашей области

В Sim интеграция с ArXiv позволяет вашим агентам программно искать, получать и анализировать научные публикации из ArXiv. Это даёт возможность автоматизировать обзоры литературы, создавать исследовательских ассистентов или включать актуальные научные знания в ваши агентные рабочие процессы. Используйте ArXiv как динамический источник данных для исследований, открытий и извлечения знаний в рамках ваших проектов Sim.

## Инструкция по использованию

Интегрирует ArXiv в рабочий процесс. Позволяет искать публикации, получать детали публикаций и работы авторов. Не требует OAuth или API-ключа.



## Действия

### `arxiv_search`

Поиск научных публикаций в ArXiv по ключевым словам, авторам, названиям и другим полям.

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `searchQuery` | string | Yes | Поисковый запрос для выполнения |
| `searchField` | string | No | Поле для поиска: all (все), ti (название), au (автор), abs (аннотация), co (комментарий), jr (журнал), cat (категория), rn (номер отчёта) |
| `maxResults` | number | No | Максимальное количество результатов (по умолчанию: 10, макс.: 2000) |
| `sortBy` | string | No | Сортировка: relevance (по релевантности), lastUpdatedDate (по дате обновления), submittedDate (по дате подачи) (по умолчанию: relevance) |
| `sortOrder` | string | No | Порядок сортировки: ascending (по возрастанию), descending (по убыванию) (по умолчанию: descending) |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `papers` | json | Массив публикаций, соответствующих поисковому запросу |
| `totalResults` | number | Общее количество найденных результатов по запросу |

### `arxiv_get_paper`

Получение подробной информации о конкретной публикации ArXiv по её идентификатору.

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `paperId` | string | Yes | Идентификатор публикации ArXiv (например, "1706.03762") |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `paper` | json | Подробная информация о запрашиваемой публикации ArXiv |

### `arxiv_get_author_papers`

Поиск публикаций конкретного автора в ArXiv.

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `authorName` | string | Yes | Имя автора для поиска |
| `maxResults` | number | No | Максимальное количество результатов (по умолчанию: 10, макс.: 2000) |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `authorPapers` | json | Массив публикаций указанного автора |
| `totalResults` | number | Общее количество найденных публикаций автора |
