[Circleback](https://circleback.ai/) — это платформа на базе ИИ, которая автоматизирует заметки о встречах, элементы действий, транскрипты и записи для вашей команды. Когда встреча завершена, Circleback обрабатывает разговор и предоставляет подробные заметки и элементы действий, а также транскрипт и запись (при наличии). Это помогает командам эффективно фиксировать инсайты, распределять элементы действий и гарантировать, что ничего не упущено — всё легко интегрируется в ваши рабочие процессы.

С помощью интеграции Sim с Circleback вы можете:

- **Получать подробные заметки о встречах и элементы действий**: автоматически собирать хорошо отформатированные сводки встреч и отслеживать actionable-задачи, обсуждавшиеся во время звонков.
- **Получать полные записи и транскрипты встреч**: получать полный разговор и связанную запись, что упрощает просмотр ключевых моментов или обмен с коллегами.
- **Собирать информацию об участниках и контекст встречи**: списки участников, метаданные встречи и теги помогают держать данные организованными и пригодными к действию.
- **Доставлять инсайты непосредственно в ваши рабочие процессы**: запускать автоматизации или отправлять данные Circleback в другие системы в момент завершения встречи, используя мощные вебхук-триггеры Sim.

**Как это работает в Sim:**  
Circleback использует вебхук-триггеры: когда встреча обработана, данные автоматически отправляются вашему агенту или автоматизации. Вы можете строить дальнейшие автоматизации на основе:

- Завершение встречи (все обработанные данные доступны)
- Новые заметки (заметки готовы ещё до полной обработки встречи)
- Интеграция с сырыми вебхуками для продвинутых сценариев использования

**Следующая информация доступна в полезной нагрузке вебхука встречи Circleback:**

| Field          | Type    | Description                                        |
|----------------|---------|----------------------------------------------------|
| `id`           | number  | ID встречи Circleback                              |
| `name`         | string  | Название встречи                                   |
| `url`          | string  | URL виртуальной встречи (Zoom, Meet, Teams и т.д.) |
| `createdAt`    | string  | Временная метка создания встречи                   |
| `duration`     | number  | Длительность в секундах                            |
| `recordingUrl` | string  | URL записи (действителен 24 часа)                  |
| `tags`         | json    | Массив тегов                                       |
| `icalUid`      | string  | ID события календаря                               |
| `attendees`    | json    | Массив объектов участников                         |
| `notes`        | string  | Заметки встречи в формате Markdown                 |
| `actionItems`  | json    | Массив элементов действий                          |
| `transcript`   | json    | Массив сегментов транскрипта                       |
| `insights`     | json    | Инсайты, созданные пользователем                   |
| `meeting`      | json    | Полная полезная нагрузка встречи                   |

Независимо от того, хотите ли вы распространять мгновенные сводки, регистрировать элементы действий или строить пользовательские рабочие процессы, запускаемые новыми данными встреч, Circleback и Sim позволяют легко обрабатывать всё, что связано с вашими встречами — автоматически.

## Триггеры

**Триггер** — это блок, который запускает рабочий процесс при наступлении события в этом сервисе.

### Circleback Meeting Completed

Запускает рабочий процесс, когда встреча обработана и готова в Circleback

#### Конфигурация

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `webhookSecret` | string | No | Проверяет, что доставки вебхуков originating от Circleback с использованием HMAC-SHA256. |

#### Выходные параметры

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `id` | number | ID встречи Circleback |
| `name` | string | Название/имя встречи |
| `url` | string | URL виртуальной встречи \(Zoom, Google Meet, Teams и т.д.\) |
| `createdAt` | string | Временная метка ISO8601 создания встречи |
| `duration` | number | Длительность встречи в секундах |
| `recordingUrl` | string | URL записи \(действителен 24 часа, если включено\) |
| `tags` | array | Массив строк тегов |
| `icalUid` | string | Идентификатор события календаря |
| `attendees` | array | Массив объектов участников с именем и email |
|   ↳ `name` | string | Имя участника |
|   ↳ `email` | string | Email участника |
| `notes` | string | Заметки встречи в формате Markdown |
| `actionItems` | array | Массив объектов элементов действий |
|   ↳ `id` | number | ID элемента действия |
|   ↳ `title` | string | Название элемента действия |
|   ↳ `description` | string | Описание элемента действия |
|   ↳ `assignee` | object | Человек, назначенный на элемент действия \(или null\) |
|     ↳ `name` | string | Имя назначенного |
|     ↳ `email` | string | Email назначенного |
|   ↳ `status` | string | Статус: PENDING или DONE |
| `transcript` | array | Массив сегментов транскрипта |
|   ↳ `speaker` | string | Имя говорящего |
|   ↳ `text` | string | Текст транскрипта |
|   ↳ `timestamp` | number | Временная метка в секундах |
| `insights` | object | Инсайты, созданные пользователем, с ключом по имени инсайта |
| `meeting` | object | Полный объект полезной нагрузки встречи |
|   ↳ `id` | number | ID встречи |
|   ↳ `name` | string | Название встречи |
|   ↳ `url` | string | URL встречи |
|   ↳ `duration` | number | Длительность в секундах |
|   ↳ `createdAt` | string | Временная метка создания |
|   ↳ `recordingUrl` | string | URL записи |
