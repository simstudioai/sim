[Amazon S3](https://aws.amazon.com/s3/) is a highly scalable, secure, and durable cloud storage service provided by Amazon Web Services. It's designed to store and retrieve any amount of data from anywhere on the web, making it one of the most widely used cloud storage solutions for businesses of all sizes.

With Amazon S3, you can:

- **Store unlimited data**: Upload files of any size and type with virtually unlimited storage capacity
- **Access from anywhere**: Получение your files from anywhere in the world with low-latency access
- **Ensure data durability**: Benefit from 99.999999999% (11 9's) durability with automatic data replication
- **Control access**: Manage permissions and access controls with fine-grained security policies
- **Scale automatically**: Handle varying workloads without manual intervention or capacity planning
- **Интегрируйте seamlessly**: Connect with other AWS services and third-party applications easily
- **Optimize costs**: Choose from multiple storage classes to optimize costs based on access patterns

## Инструкции по использованию

Интегрируйте S3 в рабочий процесс. Upload files, download objects, list bucket contents, delete objects, and copy objects between buckets. Requires AWS access key and secret access key.



## Действия

### `s3_put_object`

Upload a file to an AWS S3 bucket

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `accessKeyId` | string | Да | Your AWS Access Key ID |
| `secretAccessKey` | string | Да | Your AWS Secret Access Key |
| `region` | string | Да | Регион AWS (например, us-east-1\) |
| `bucketName` | string | Да | S3 bucket name (например, my-bucket\) |
| `objectKey` | string | Да | Object key/path in S3 (например, folder/filename.ext\) |
| `file` | file | Нет | File to upload |
| `content` | string | Нет | Text content to upload \(alternative to file\) |
| `contentType` | string | Нет | Content-Type header \(auto-detected from file if not provided\) |
| `acl` | string | Нет | Access control list (например, private, public-read\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `url` | string | URL of the uploaded S3 object |
| `uri` | string | S3 URI of the uploaded object \(s3://bucket/key\) |
| `metadata` | object | Upload metadata including ETag and location |

### `s3_get_object`

Получение an object from an AWS S3 bucket

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `accessKeyId` | string | Да | Your AWS Access Key ID |
| `secretAccessKey` | string | Да | Your AWS Secret Access Key |
| `region` | string | Нет | Optional region override when URL does not include region (например, us-east-1, eu-west-1\) |
| `s3Uri` | string | Да | S3 Object URL (например, https://bucket.s3.region.amazonaws.com/path/to/file\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `url` | string | Pre-signed URL for downloading the S3 object |
| `file` | file | Downloaded file stored in execution files |
| `metadata` | object | File metadata including type, size, name, and last modified date |

### `s3_list_objects`

Получение списка objects in an AWS S3 bucket

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `accessKeyId` | string | Да | Your AWS Access Key ID |
| `secretAccessKey` | string | Да | Your AWS Secret Access Key |
| `region` | string | Да | Регион AWS (например, us-east-1\) |
| `bucketName` | string | Да | S3 bucket name (например, my-bucket\) |
| `prefix` | string | Нет | Prefix to filter objects (например, folder/, images/2024/\) |
| `maxKeys` | number | Нет | Maximum number of objects to return \(по умолчанию: 1000\) |
| `continuationToken` | string | Нет | Token for pagination from previous list response |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `objects` | array | Получение списка of S3 objects |
|   ↳ `key` | string | Object key |
|   ↳ `size` | number | Object size in bytes |
|   ↳ `lastModified` | string | Last modified timestamp |
|   ↳ `etag` | string | Entity tag |
| `metadata` | object | Listing metadata including pagination info |

### `s3_delete_object`

Удалениеn object from an AWS S3 bucket

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `accessKeyId` | string | Да | Your AWS Access Key ID |
| `secretAccessKey` | string | Да | Your AWS Secret Access Key |
| `region` | string | Да | Регион AWS (например, us-east-1\) |
| `bucketName` | string | Да | S3 bucket name (например, my-bucket\) |
| `objectKey` | string | Да | Object key/path to delete (например, folder/file.txt\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `deleted` | boolean | Указывает,  object was successfully deleted |
| `metadata` | object | Deletion metadata |

### `s3_copy_object`

Copy an object within or between AWS S3 buckets

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `accessKeyId` | string | Да | Your AWS Access Key ID |
| `secretAccessKey` | string | Да | Your AWS Secret Access Key |
| `region` | string | Да | Регион AWS (например, us-east-1\) |
| `sourceBucket` | string | Да | Source bucket name (например, my-bucket\) |
| `sourceKey` | string | Да | Source object key/path (например, folder/file.txt\) |
| `destinationBucket` | string | Да | Destination bucket name (например, my-other-bucket\) |
| `destinationKey` | string | Да | Destination object key/path (например, backup/file.txt\) |
| `acl` | string | Нет | Access control list for the copied object (например, private, public-read\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `url` | string | URL of the copied S3 object |
| `uri` | string | S3 URI of the copied object \(s3://bucket/key\) |
| `metadata` | object | Copy operation metadata |
