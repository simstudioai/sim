## Инструкция по использованию

Интегрируйте Brandfetch в ваш рабочий процесс. Получайте логотипы, цвета, шрифты и данные компаний по домену, тикеру или поиску по названию.



## Действия

### `brandfetch_get_brand`

Получение активов бренда, включая логотипы, цвета, шрифты и информацию о компании по домену, тикеру, ISIN или криптосимволу

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Yes | API-ключ Brandfetch |
| `identifier` | string | Yes | Идентификатор бренда: домен (nike.com), биржевой тикер (NKE), ISIN (US6541061031) или криптосимвол (BTC) |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `id` | string | Уникальный идентификатор бренда |
| `name` | string | Название бренда |
| `domain` | string | Домен бренда |
| `claimed` | boolean | Подтверждён ли профиль бренда |
| `description` | string | Краткое описание бренда |
| `longDescription` | string | Подробное описание бренда |
| `links` | array | Ссылки на социальные сети и веб-сайты |
|   ↳ `name` | string | Название ссылки (например, twitter, linkedin) |
|   ↳ `url` | string | URL ссылки |
| `logos` | array | Логотипы бренда с форматами и темами |
|   ↳ `type` | string | Тип логотипа (logo, icon, symbol, other) |
|   ↳ `theme` | string | Тема логотипа (light, dark) |
|   ↳ `formats` | array | Доступные форматы с URL, форматом, шириной и высотой |
| `colors` | array | Цвета бренда с hex-значениями и типами |
|   ↳ `hex` | string | Hex-код цвета |
|   ↳ `type` | string | Тип цвета (accent, dark, light, brand) |
|   ↳ `brightness` | number | Значение яркости |
| `fonts` | array | Шрифты бренда с названиями и типами |
|   ↳ `name` | string | Название шрифта |
|   ↳ `type` | string | Тип шрифта (title, body) |
|   ↳ `origin` | string | Источник шрифта (google, custom, system) |
| `company` | json | Фирмографические данные компании, включая количество сотрудников, местоположение и отрасли |
| `qualityScore` | number | Оценка качества данных от 0 до 1 |
| `isNsfw` | boolean | Содержит ли бренд контент для взрослых |

### `brandfetch_search`

Поиск брендов по названию с получением их доменов и логотипов

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Yes | API-ключ Brandfetch |
| `name` | string | Yes | Название компании или бренда для поиска |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `results` | array | Список подходящих брендов |
|   ↳ `brandId` | string | Уникальный идентификатор бренда |
|   ↳ `name` | string | Название бренда |
|   ↳ `domain` | string | Домен бренда |
|   ↳ `claimed` | boolean | Подтверждён ли профиль бренда |
|   ↳ `icon` | string | URL иконки бренда |
