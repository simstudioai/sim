Инструмент [Reducto](https://reducto.ai/) обеспечивает быстрое и точное извлечение текста и данных из PDF-документов с помощью OCR (оптического распознавания символов). Reducto разработан для агентных рабочих процессов, что упрощает обработку загруженных или связанных PDF-файлов и преобразование их содержимого в готовую к использованию информацию.

С помощью инструмента Reducto вы можете:

- **Извлекать текст и таблицы из PDF-файлов**: Быстро преобразовывать отсканированные или цифровые PDF-файлы в текст, markdown или структурированный JSON.
- **Разбирать PDF-файлы из загрузок или URL**: Обрабатывать документы, загружая PDF-файл или указывая прямой URL.
- **Настраивать форматирование вывода**: Выбирайте предпочитаемый формат вывода — markdown, обычный текст или JSON — и указывайте формат таблиц как markdown или HTML.
- **Выбирать определённые страницы**: При необходимости извлекайте содержимое с конкретных страниц для оптимизации обработки и фокусировки на важном.
- **Получать подробные метаданные обработки**: Наряду с извлечённым содержимым получайте детали задания, время обработки, информацию об исходном файле, количество страниц и статистику использования OCR для аудита и автоматизации.

Независимо от того, автоматизируете ли вы этапы рабочих процессов, извлекаете критически важную для бизнеса информацию или открываете архивные документы для поиска и анализа, OCR-парсер Reducto предоставляет вам структурированные, пригодные для использования данные даже из самых сложных PDF-файлов.

Ищете надёжный и масштабируемый разбор PDF-файлов? Reducto оптимизирован для использования разработчиками и агентами — обеспечивая точность, скорость и гибкость для современного понимания документов.

## Инструкции по использованию

Интегрируйте Reducto Parse в рабочий процесс. Можно извлекать текст из загруженных PDF-документов или ссылок на файлы.



## Действия

### `reducto_parser`

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `file` | file | Да | PDF-документ для обработки |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `job_id` | string | Уникальный идентификатор задания обработки |
| `duration` | number | Время обработки в секундах |
| `usage` | json | Данные о потреблении ресурсов |
| `result` | json | Разобранное содержимое документа с фрагментами и блоками |
| `pdf_url` | string | URL хранения преобразованного PDF |
| `studio_link` | string | Ссылка на интерфейс Reducto studio |
