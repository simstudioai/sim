ZeroBounce — это сервис проверки и доставляемости электронной почты в реальном времени. Используйте эту интеграцию для проверки отдельных адресов электронной почты перед рассылкой — она помечает недействительные, универсальные (catch-all), спам-ловушки (spamtrap), жалобные (abuse) и запрещённые к отправке (do-not-mail) адреса, позволяя исключить рискованные контакты и защитить репутацию отправителя, — а также для проверки оставшихся кредитов проверки на вашем аккаунте. ZeroBounce является верификатором по умолчанию во многих каскадных системах поиска email.
{/* MANUAL-CONTENT-END */}


## Инструкции по использованию

Интегрируйте ZeroBounce для проверки доставляемости email в реальном времени — выявляйте недействительные, универсальные (catch-all), спам-ловушки (spamtrap), жалобные (abuse) и запрещённые к отправке (do-not-mail) адреса — и проверяйте оставшиеся кредиты проверки.



## Действия

### `zerobounce_verify_email`

Проверка доставляемости адреса электронной почты в реальном времени. Использует один кредит проверки.

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `email` | string | Да | Адрес электронной почты для проверки \(например, john@example.com\) |
| `apiKey` | string | Да | Ключ API ZeroBounce |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `email` | string | Проверенный адрес электронной почты |
| `status` | string | Статус проверки \(valid, invalid, catch_all, unknown, spamtrap, abuse, do_not_mail\) |
| `deliverable` | boolean | Является ли email действительным и безопасным для отправки |
| `subStatus` | string | Подробный подстатус от ZeroBounce |
| `freeEmail` | boolean | Использует ли адрес бесплатного почтового провайдера |
| `didYouMean` | string | Предлагаемое исправление для вероятной опечатки |

### `zerobounce_get_credits`

Получение оставшихся кредитов проверки для аутентифицированного аккаунта.

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Да | Ключ API ZeroBounce |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `credits` | number | Оставшиеся кредиты проверки \(-1, если недоступно\) |
