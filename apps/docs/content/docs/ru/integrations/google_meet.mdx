[Google Meet](https://meet.google.com) — это платформа Google для видеоконференций и онлайн-встреч, обеспечивающая безопасные, высококачественные видеозвонки для отдельных пользователей и команд. Являясь ключевым компонентом Google Workspace, Google Meet обеспечивает совместную работу в реальном времени через видеовстречи, демонстрацию экрана и встроенный чат.

REST API Google Meet (v2) позволяет программно управлять пространствами для встреч и записями конференций, обеспечивая автоматизированным рабочим процессам возможность создавать встречи, отслеживать участие и управлять активными конференциями без ручного вмешательства.

Ключевые возможности Google Meet API:

- **Управление пространствами для встреч**: создание, получение и настройка пространств для встреч с настраиваемыми элементами управления доступом.
- **Записи конференций**: доступ к историческим данным конференций, включая время начала/окончания и связанные пространства.
- **Отслеживание участников**: просмотр сведений об участниках любой конференции, включая время подключения/отключения и типы пользователей.
- **Управление доступом**: настройка того, кто может присоединяться к встречам (открытый, доверенный или ограниченный доступ) и какие точки входа разрешены.
- **Управление активными конференциями**: программное завершение активных конференций в пространствах для встреч.

В Sim интеграция Google Meet позволяет вашим агентам создавать пространства для встреч по требованию, отслеживать активность конференций, отслеживать участие во встречах и управлять активными конференциями в рамках автоматизированных рабочих процессов. Это открывает такие сценарии, как автоматическое предоставление комнат для встреч для запланированных событий, создание отчётов о посещаемости, завершение устаревших конференций и построение аналитических панелей встреч.

## Инструкции по использованию

Интеграция Google Meet в ваш рабочий процесс. Создавайте пространства для встреч, получайте сведения о пространствах, завершайте конференции, просматривайте записи конференций и участников.



## Действия

### `google_meet_create_space`

Создание нового пространства для встреч Google Meet

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `accessType` | string | Нет | Кто может присоединиться к встрече без стука: OPEN \(любой по ссылке\), TRUSTED \(участники организации\), RESTRICTED \(только приглашённые\) |
| `entryPointAccess` | string | Нет | Разрешённые точки входа: ALL \(все точки входа\) или CREATOR_APP_ONLY \(только через приложение\) |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `name` | string | Имя ресурса пространства \(например, spaces/abc123\) |
| `meetingUri` | string | URL встречи \(например, https://meet.google.com/abc-defg-hij\) |
| `meetingCode` | string | Код встречи \(например, abc-defg-hij\) |
| `accessType` | string | Конфигурация типа доступа |
| `entryPointAccess` | string | Конфигурация доступа к точкам входа |

### `google_meet_get_space`

Получение сведений о пространстве встречи Google Meet по имени или коду встречи

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `spaceName` | string | Да | Имя ресурса пространства \(spaces/abc123\) или код встречи \(abc-defg-hij\) |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `name` | string | Имя ресурса пространства |
| `meetingUri` | string | URL встречи |
| `meetingCode` | string | Код встречи |
| `accessType` | string | Конфигурация типа доступа |
| `entryPointAccess` | string | Конфигурация доступа к точкам входа |
| `activeConference` | string | Имя записи активной конференции |

### `google_meet_end_conference`

Завершение активной конференции в пространстве Google Meet

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `spaceName` | string | Да | Имя ресурса пространства \(например, spaces/abc123\) |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `ended` | boolean | Была ли конференция успешно завершена |

### `google_meet_list_conference_records`

Список записей конференций для встреч, которые вы организовали

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `filter` | string | Нет | Фильтр по имени пространства \(например, space.name = "spaces/abc123"\) или диапазону времени \(например, start_time &gt; "2024-01-01T00:00:00Z"\) |
| `pageSize` | number | Нет | Максимальное количество возвращаемых записей конференций \(макс. 100\) |
| `pageToken` | string | Нет | Токен страницы из предыдущего запроса списка |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `conferenceRecords` | json | Список записей конференций с именем, временем начала/окончания и пространством |
| `nextPageToken` | string | Токен для следующей страницы результатов |

### `google_meet_get_conference_record`

Получение сведений о конкретной записи конференции

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `conferenceName` | string | Да | Имя ресурса записи конференции \(например, conferenceRecords/abc123\) |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `name` | string | Имя ресурса записи конференции |
| `startTime` | string | Время начала конференции |
| `endTime` | string | Время окончания конференции |
| `expireTime` | string | Время истечения срока действия записи конференции |
| `space` | string | Имя связанного ресурса пространства |

### `google_meet_list_participants`

Список участников записи конференции

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `conferenceName` | string | Да | Имя ресурса записи конференции \(например, conferenceRecords/abc123\) |
| `filter` | string | Нет | Фильтр участников \(например, earliest_start_time &gt; "2024-01-01T00:00:00Z"\) |
| `pageSize` | number | Нет | Максимальное количество возвращаемых участников \(по умолчанию 100, макс. 250\) |
| `pageToken` | string | Нет | Токен страницы из предыдущего запроса списка |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `participants` | json | Список участников с именем, временем, отображаемым именем и типом пользователя |
| `nextPageToken` | string | Токен для следующей страницы результатов |
| `totalSize` | number | Общее количество участников |
