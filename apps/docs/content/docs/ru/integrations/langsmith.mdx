With the LangSmith integration, you can:

## Инструкции по использованию

Send run data to LangSmith to trace executions, attach metadata, and monitor workflow performance.



## Действия

### `langsmith_create_run`

Forward a single run to LangSmith for ingestion.

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Да | LangSmith API key |
| `id` | string | Нет | Unique run identifier |
| `name` | string | Да | Run name |
| `run_type` | string | Да | Run type \(tool, chain, llm, retriever, embedding, prompt, parser\) |
| `start_time` | string | Нет | Run start time in ISO-8601 format |
| `end_time` | string | Нет | Run end time in ISO-8601 format |
| `inputs` | json | Нет | Inputs payload |
| `run_outputs` | json | Нет | Outputs payload |
| `extra` | json | Нет | Additional metadata \(extra\) |
| `tags` | json | Нет | Array of tag strings |
| `parent_run_id` | string | Нет | Parent run ID |
| `trace_id` | string | Нет | Trace ID |
| `session_id` | string | Нет | Session ID |
| `session_name` | string | Нет | Session name |
| `status` | string | Нет | Run status |
| `error` | string | Нет | Error details |
| `dotted_order` | string | Нет | Dotted order string |
| `events` | json | Нет | Structured events array |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `accepted` | boolean | Whether the run was accepted for ingestion |
| `runId` | string | Run identifier provided in the request |
| `message` | string | Response message from LangSmith |

### `langsmith_create_runs_batch`

Forward multiple runs to LangSmith in a single batch.

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Да | LangSmith API key |
| `post` | json | Нет | Array of new runs to ingest |
| `patch` | json | Нет | Array of runs to update/patch |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `accepted` | boolean | Whether the batch was accepted for ingestion |
| `runIds` | array | Run identifiers provided in the request |
| `message` | string | Response message from LangSmith |
| `messages` | array | Per-run response messages, when provided |
