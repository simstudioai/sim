[Quartr](https://quartr.com/) is an investor relations data platform that provides structured, AI-ready access to live and historical earnings data from more than 15,000 public companies across 65+ markets. It covers earnings calls, transcripts, filings, reports, slide presentations, and AI-generated event summaries.

With the Quartr integration in Sim, you can:

- **Look up companies**: Find any covered public company by ticker, ISIN, CIK, OpenFIGI, country, or exchange, and resolve it to a Quartr company ID.
- **Track corporate events**: Получение списка earnings calls and other corporate events by company, event type, and date range, or retrieve a single event with its fiscal period details.
- **Summarize earnings calls**: Fetch AI-generated event summaries in one-line, short, or long form — with source references back to the underlying documents.
- **Download filings and reports**: Получение списка 10-Ks, 10-Qs, earnings releases, and other filings, and download report PDFs directly в ваш рабочий процесс as files.
- **Work with slide decks and transcripts**: Получение investor presentation PDFs and structured transcript JSON files (with timestamps and speaker identification) for downstream analysis.
- **Access event audio**: Получение download and streaming URLs for archived earnings call audio, including where the Q&A section starts.
- **Monitor live events**: Получение списка live and upcoming events with their live audio and transcript stream URLs to react the moment a company goes live.

Downloaded reports, slide decks, and transcripts are stored as execution files, so they can be passed straight into agents, knowledge bases, or other blocks in your workflow.

To use the integration, generate an Ключ API from the Quartr API portal and paste it into the block. Your key inherits the datasets enabled on your Quartr subscription.

## Инструкции по использованию

Интегрируйте Quartr в рабочий процесс. Look up public companies, corporate events, and event types; fetch AI-generated event summaries; list and download filings, reports, slide decks, and transcripts; and access archived audio and live event streams. Requires Ключ API.



## Действия

### `quartr_list_companies`

Получение списка companies covered by Quartr, filterable by ticker, ISIN, CIK, OpenFIGI, country, and exchange.

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Да | Quartr Ключ API |
| `tickers` | string | Нет | Comma-separated list of company tickers (например, "AAPL,MSFT"\) |
| `isins` | string | Нет | Comma-separated list of ISINs (например, "US0378331005"\) |
| `ciks` | string | Нет | Comma-separated list of SEC CIKs (например, "0000320193"\) |
| `countries` | string | Нет | Comma-separated list of ISO 3166-1 alpha-2 country codes (например, "US,SE"\) |
| `exchanges` | string | Нет | Comma-separated list of exchange symbols, without whitespace (например, "NasdaqGS"\) |
| `companyIds` | string | Нет | Comma-separated list of Quartr company IDs (например, "4742,128"\) |
| `openfigis` | string | Нет | Comma-separated list of OpenFIGI codes \(figi, compositeFigi, or shareClassFigi\) |
| `updatedAfter` | string | Нет | Only return data updated after this ISO 8601 date (например, "2024-01-01"\) |
| `updatedBefore` | string | Нет | Only return data updated before this ISO 8601 date (например, "2024-12-31"\) |
| `limit` | number | Нет | Maximum number of items to return in a single request \(по умолчанию: 10, max: 500\) |
| `cursor` | number | Нет | Pagination cursor from the previous response \(nextCursor\) for the next page |
| `direction` | string | Нет | Sort direction by id: "asc" or "desc" \(по умолчанию: asc\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `companies` | array | Companies matching the filters |
|   ↳ `id` | number | Quartr company ID |
|   ↳ `name` | string | Legal company name |
|   ↳ `displayName` | string | Display name |
|   ↳ `country` | string | ISO 3166-1 alpha-2 country code |
|   ↳ `tickers` | array | Ticker listings for the company |
|     ↳ `ticker` | string | Ticker symbol |
|     ↳ `exchange` | string | Exchange symbol |
|   ↳ `isins` | array | ISINs for the company |
|   ↳ `cik` | string | SEC Central Index Key |
|   ↳ `openfigi` | array | OpenFIGI share class identifiers |
|   ↳ `backlinkUrl` | string | Quartr backlink URL for the company |
|   ↳ `createdAt` | string | Creation timestamp \(ISO 8601\) |
|   ↳ `updatedAt` | string | Last update timestamp \(ISO 8601\) |
| `nextCursor` | number | Cursor for fetching the next page of results \(null when no more pages\) |

### `quartr_get_event_summary`

Получение the AI-generated summary of a corporate event from Quartr, with selectable length and optional plain-text formatting.

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Да | Quartr Ключ API |
| `eventId` | number | Да | Quartr event ID (например, 128301\) |
| `summaryLength` | string | Нет | Length preset for the summary: "line", "short", or "long" \(по умолчанию: short\) |
| `plainSummary` | boolean | Нет | Return a plain-text summary without embedded document source tags |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `summary` | string | AI-generated event summary in Markdown \(includes embedded document source tags unless a plain-text summary is requested\) |
| `sources` | array | Source documents referenced by the summary |
|   ↳ `sourceId` | string | ID linking the source document to tags embedded in the summary |
|   ↳ `documentId` | number | Quartr document ID of the source |
|   ↳ `page` | number | Page number or timestamp in seconds depending on the document type |
|   ↳ `timestamp` | number | Timestamp in seconds |
|   ↳ `typeId` | number | Document type ID of the source |
| `summaryId` | number | Quartr summary ID |
| `summaryCreatedAt` | string | Summary creation timestamp \(ISO 8601\) |
| `summaryUpdatedAt` | string | Summary last update timestamp \(ISO 8601\) |
