[Perplexity AI](https://www.perplexity.ai) — это поисковая система и система ответов на базе искусственного интеллекта, которая объединяет возможности больших языковых моделей с поиском в интернете в реальном времени для предоставления точной, актуальной информации и исчерпывающих ответов на сложные вопросы.

С помощью Perplexity AI вы можете:

- **Получать точные ответы**: Получайте исчерпывающие ответы на вопросы с цитированием из надёжных источников
- **Получать доступ к актуальной информации**: Получайте самую свежую информацию благодаря возможностям веб-поиска Perplexity
- **Углублённо изучать темы**: Погружайтесь глубже в темы с помощью уточняющих вопросов и связанной информации
- **Проверять информацию**: Проверяйте достоверность ответов через предоставленные источники и ссылки
- **Генерировать контент**: Создавайте резюме, аналитику и творческий контент на основе актуальной информации
- **Эффективно исследовать**: Оптимизируйте исследовательские процессы с помощью исчерпывающих ответов на сложные запросы
- **Взаимодействовать в диалоге**: Участвуйте в естественном диалоге для уточнения вопросов и изучения тем

В Sim интеграция с Perplexity позволяет вашим агентам использовать эти мощные возможности AI программно в рамках своих рабочих процессов. Это позволяет создавать сложные сценарии автоматизации, объединяющие понимание естественного языка, получение информации в реальном времени и генерацию контента. Ваши агенты могут формулировать запросы, получать исчерпывающие ответы с цитированием и включать эту информацию в свои процессы принятия решений или выходные данные. Эта интеграция устраняет разрыв между автоматизацией рабочих процессов и доступом к актуальной, надёжной информации, позволяя вашим агентам принимать более обоснованные решения и предоставлять более точные ответы. Подключив Sim к Perplexity, вы можете создавать агентов, которые остаются в курсе последней информации, предоставляют хорошо исследованные ответы и выдают более ценные инсайты пользователям — всё это без необходимости ручного исследования или сбора информации.

## Инструкции по использованию

Интегрируйте Perplexity в рабочий процесс. Можно генерировать завершения с использованием чат-моделей Perplexity AI или выполнять веб-поиск с расширенной фильтрацией.



## Действия

### `perplexity_chat`

Генерация завершений с использованием чат-моделей Perplexity AI

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `systemPrompt` | string | Нет | Системный промпт для управления поведением модели |
| `content` | string | Да | Текст сообщения пользователя для отправки модели |
| `model` | string | Да | Модель для чат-завершений \(например, "sonar", "sonar-pro", "sonar-reasoning"\) |
| `max_tokens` | number | Нет | Максимальное количество токенов для генерации \(например, 1024, 2048, 4096\) |
| `temperature` | number | Нет | Температура выборки от 0 до 1 \(например, 0.0 для детерминированного, 0.7 для творческого\) |
| `apiKey` | string | Да | Ключ API Perplexity |
| `pricing` | custom | Нет | Нет описания |
| `rateLimit` | string | Нет | Нет описания |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `content` | string | Сгенерированный текстовый контент |
| `model` | string | Модель, использованная для генерации |
| `usage` | object | Информация об использовании токенов |
|   ↳ `prompt_tokens` | number | Количество токенов в промпте |
|   ↳ `completion_tokens` | number | Количество токенов в завершении |
|   ↳ `total_tokens` | number | Общее количество использованных токенов |

### `perplexity_search`

Получение ранжированных результатов поиска из постоянно обновляемого индекса Perplexity с расширенными возможностями фильтрации и настройки

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `query` | string | Да | Поисковый запрос или массив запросов \(макс. 5 для мультизапросного поиска\) |
| `max_results` | number | Нет | Максимальное количество результатов поиска \(1-20, по умолчанию: 10\) |
| `search_domain_filter` | array | Нет | Список доменов/URL для ограничения результатов поиска \(например, \["github.com", "stackoverflow.com"\], макс. 20\) |
| `max_tokens_per_page` | number | Нет | Максимальное количество токенов, извлекаемых с каждой веб-страницы \(по умолчанию: 1024\) |
| `country` | string | Нет | Код страны для фильтрации результатов поиска \(например, US, GB, DE\) |
| `search_recency_filter` | string | Нет | Фильтрация результатов по давности \(например, "hour", "day", "week", "month", "year"\) |
| `search_after_date` | string | Нет | Включать только контент, опубликованный после этой даты \(формат: MM/DD/YYYY\) |
| `search_before_date` | string | Нет | Включать только контент, опубликованный до этой даты \(формат: MM/DD/YYYY\) |
| `apiKey` | string | Да | Ключ API Perplexity |
| `pricing` | per_request | Нет | Нет описания |
| `rateLimit` | string | Нет | Нет описания |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `results` | array | Массив результатов поиска |
|   ↳ `title` | string | Заголовок результата поиска |
|   ↳ `url` | string | URL результата поиска |
|   ↳ `snippet` | string | Краткий отрывок или резюме содержимого |
|   ↳ `date` | string | Дата, когда страница была просканирована и добавлена в индекс Perplexity |
|   ↳ `last_updated` | string | Дата последнего обновления страницы в индексе Perplexity |
