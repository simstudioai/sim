[Google BigQuery](https://cloud.google.com/bigquery) — это полностью управляемое, бессерверное хранилище данных Google Cloud, разработанное для масштабной аналитики данных. BigQuery позволяет выполнять быстрые SQL-запросы на массивных наборах данных, что делает его идеальным для бизнес-аналитики, исследования данных и конвейеров машинного обучения.

С помощью интеграции Google BigQuery в Sim вы можете:

- **Выполнять SQL-запросы**: Выполнять запросы к вашим наборам данных BigQuery и получать результаты для анализа или дальнейшей обработки
- **Просматривать наборы данных**: Просматривать доступные наборы данных в проекте Google Cloud
- **Просматривать и изучать таблицы**: Перечислять таблицы в наборе данных и получать подробную информацию о схеме
- **Вставлять строки**: Потоково вставлять новые строки в таблицы BigQuery для приёма данных в реальном времени

В Sim интеграция с Google BigQuery позволяет вашим агентам запрашивать наборы данных, проверять схемы и вставлять строки в рамках автоматизированных рабочих процессов. Это идеально подходит для автоматизированной отчётности, оркестрации конвейеров данных, приёма данных в реальном времени и принятия решений на основе аналитики.

## Инструкции по использованию

Подключитесь к Google BigQuery для выполнения SQL-запросов, просмотра наборов данных и таблиц, получения метаданных таблиц и вставки строк.



## Действия

### `google_bigquery_query`

Выполнение SQL-запроса к Google BigQuery и возврат результатов

#### Входные параметры

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `projectId` | string | Yes | Идентификатор проекта Google Cloud |
| `query` | string | Yes | SQL-запрос для выполнения |
| `useLegacySql` | boolean | No | Использовать ли устаревший синтаксис SQL \(по умолчанию: false\) |
| `maxResults` | number | No | Максимальное количество возвращаемых строк |
| `defaultDatasetId` | string | No | Набор данных по умолчанию для неквалифицированных имён таблиц |
| `location` | string | No | Местоположение обработки \(например, "US", "EU"\) |

#### Выходные параметры

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `columns` | array | Массив имён столбцов из результата запроса |
| `rows` | array | Массив объектов строк с ключами по именам столбцов |
| `totalRows` | string | Общее количество строк в полном наборе результатов |
| `jobComplete` | boolean | Завершился ли запрос в пределах таймаута |
| `totalBytesProcessed` | string | Общее количество байт, обработанных запросом |
| `cacheHit` | boolean | Был ли результат запроса получен из кэша |
| `jobReference` | object | Ссылка на задание \(полезна, когда jobComplete = false\) |
|   ↳ `projectId` | string | Идентификатор проекта, содержащего задание |
|   ↳ `jobId` | string | Уникальный идентификатор задания |
|   ↳ `location` | string | Географическое местоположение задания |
| `pageToken` | string | Токен для получения дополнительных страниц результатов |

### `google_bigquery_list_datasets`

Вывод списка всех наборов данных в проекте Google BigQuery

#### Входные параметры

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `projectId` | string | Yes | Идентификатор проекта Google Cloud |
| `maxResults` | number | No | Максимальное количество возвращаемых наборов данных |
| `pageToken` | string | No | Токен для пагинации |

#### Выходные параметры

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `datasets` | array | Массив объектов наборов данных |
|   ↳ `datasetId` | string | Уникальный идентификатор набора данных |
|   ↳ `projectId` | string | Идентификатор проекта, содержащего этот набор данных |
|   ↳ `friendlyName` | string | Описательное имя набора данных |
|   ↳ `location` | string | Географическое местоположение, где находятся данные |
| `nextPageToken` | string | Токен для получения следующей страницы результатов |

### `google_bigquery_list_tables`

Вывод списка всех таблиц в наборе данных Google BigQuery

#### Входные параметры

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `projectId` | string | Yes | Идентификатор проекта Google Cloud |
| `datasetId` | string | Yes | Идентификатор набора данных BigQuery |
| `maxResults` | number | No | Максимальное количество возвращаемых таблиц |
| `pageToken` | string | No | Токен для пагинации |

#### Выходные параметры

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `tables` | array | Массив объектов таблиц |
|   ↳ `tableId` | string | Идентификатор таблицы |
|   ↳ `datasetId` | string | Идентификатор набора данных, содержащего эту таблицу |
|   ↳ `projectId` | string | Идентификатор проекта, содержащего эту таблицу |
|   ↳ `type` | string | Тип таблицы \(TABLE, VIEW, EXTERNAL и т.д.\) |
|   ↳ `friendlyName` | string | Пользовательское имя таблицы |
|   ↳ `creationTime` | string | Время создания, в миллисекундах с начала эпохи |
| `totalItems` | number | Общее количество таблиц в наборе данных |
| `nextPageToken` | string | Токен для получения следующей страницы результатов |

### `google_bigquery_get_table`

Получение метаданных и схемы таблицы Google BigQuery

#### Входные параметры

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `projectId` | string | Yes | Идентификатор проекта Google Cloud |
| `datasetId` | string | Yes | Идентификатор набора данных BigQuery |
| `tableId` | string | Yes | Идентификатор таблицы BigQuery |

#### Выходные параметры

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `tableId` | string | Идентификатор таблицы |
| `datasetId` | string | Идентификатор набора данных |
| `projectId` | string | Идентификатор проекта |
| `type` | string | Тип таблицы \(TABLE, VIEW, SNAPSHOT, MATERIALIZED_VIEW, EXTERNAL\) |
| `description` | string | Описание таблицы |
| `numRows` | string | Общее количество строк |
| `numBytes` | string | Общий размер в байтах, исключая данные в потоковом буфере |
| `schema` | array | Массив определений столбцов |
|   ↳ `name` | string | Имя столбца |
|   ↳ `type` | string | Тип данных \(STRING, INTEGER, FLOAT, BOOLEAN, TIMESTAMP, RECORD и т.д.\) |
|   ↳ `mode` | string | Режим столбца \(NULLABLE, REQUIRED или REPEATED\) |
|   ↳ `description` | string | Описание столбца |
| `creationTime` | string | Время создания таблицы \(миллисекунды с начала эпохи\) |
| `lastModifiedTime` | string | Время последнего изменения \(миллисекунды с начала эпохи\) |
| `location` | string | Географическое местоположение, где находится таблица |

### `google_bigquery_insert_rows`

Вставка строк в таблицу Google BigQuery с использованием потоковой вставки

#### Входные параметры

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `projectId` | string | Yes | Идентификатор проекта Google Cloud |
| `datasetId` | string | Yes | Идентификатор набора данных BigQuery |
| `tableId` | string | Yes | Идентификатор таблицы BigQuery |
| `rows` | string | Yes | JSON-массив объектов строк для вставки |
| `skipInvalidRows` | boolean | No | Вставлять ли допустимые строки, даже если некоторые недопустимы |
| `ignoreUnknownValues` | boolean | No | Игнорировать ли столбцы, отсутствующие в схеме таблицы |

#### Выходные параметры

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `insertedRows` | number | Количество успешно вставленных строк |
| `errors` | array | Массив ошибок вставки по строкам \(пуст, если все успешно\) |
|   ↳ `index` | number | Индекс (с нуля) строки, которая не удалась |
|   ↳ `errors` | array | Детали ошибки для этой строки |
|     ↳ `reason` | string | Краткий код ошибки, суммирующий проблему |
|     ↳ `location` | string | Где произошла ошибка |
|     ↳ `message` | string | Человекочитаемое описание ошибки |
