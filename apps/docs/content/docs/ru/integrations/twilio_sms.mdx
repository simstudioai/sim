[Twilio SMS](https://www.twilio.com/en-us/sms) — это мощная облачная коммуникационная платформа, которая позволяет компаниям интегрировать возможности обмена сообщениями в свои приложения и сервисы.

Twilio SMS предоставляет надёжный API для программной отправки и получения текстовых сообщений по всему миру. С покрытием в более чем 180 странах и соглашением об уровне обслуживания с доступностью 99,999%, Twilio зарекомендовал себя как лидер отрасли в области коммуникационных технологий.

Ключевые возможности Twilio SMS:

- **Глобальный охват**: Отправляйте сообщения получателям по всему миру с использованием местных телефонных номеров в различных странах
- **Программируемый обмен сообщениями**: Настраивайте доставку сообщений с помощью веб-хуков, отчётов о доставке и опций планирования
- **Расширенная аналитика**: Отслеживайте показатели доставки, метрики вовлечённости и оптимизируйте свои кампании по обмену сообщениями

В Sim интеграция с Twilio SMS позволяет вашим агентам использовать эти мощные возможности обмена сообщениями в рамках своих рабочих процессов. Это создаёт возможности для сложных сценариев взаимодействия с клиентами, таких как напоминания о встречах, коды подтверждения, оповещения и интерактивные беседы. Интеграция устраняет разрыв между вашими AI-рабочими процессами и каналами связи с клиентами, позволяя вашим агентам доставлять своевременную, актуальную информацию непосредственно на мобильные устройства пользователей. Подключив Sim к Twilio SMS, вы можете создавать интеллектуальных агентов, которые взаимодействуют с клиентами через предпочитаемый ими канал связи, улучшая пользовательский опыт и автоматизируя рутинные задачи по обмену сообщениями.

## Инструкции по использованию

Интегрируйте Twilio в рабочий процесс. Можно отправлять SMS-сообщения.



## Действия

### `twilio_send_sms`

Отправка текстовых сообщений одному или нескольким получателям с помощью API Twilio.

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `phoneNumbers` | string | Да | Номера телефонов для отправки сообщения в формате E.164 \(например, +15551234567\), разделённые переводами строк |
| `message` | string | Да | Сообщение для отправки |
| `accountSid` | string | Да | Идентификатор аккаунта Twilio (Account SID) |
| `authToken` | string | Да | Токен аутентификации Twilio (Auth Token) |
| `fromNumber` | string | Да | Номер телефона Twilio для отправки сообщения в формате E.164 \(например, +15551234567\) |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `success` | boolean | Статус успешности отправки SMS |
| `messageId` | string | Уникальный идентификатор сообщения Twilio \(SID\) |
| `status` | string | Статус доставки сообщения от Twilio |
| `fromNumber` | string | Номер телефона, с которого отправлено сообщение |
| `toNumber` | string | Номер телефона, на который отправлено сообщение |
