[AWS Security Token Service (STS)](https://docs.aws.amazon.com/STS/latest/APIReference/welcome.html) — это веб-сервис, который позволяет запрашивать временные учётные данные с ограниченными привилегиями для пользователей AWS Identity and Access Management (IAM) или для пользователей, прошедших аутентификацию (федеративных пользователей).

С помощью AWS STS вы можете:

- **Принимать роли IAM (Assume Role)**: Запрашивайте временные учётные данные для доступа к ресурсам AWS между аккаунтами или с повышенными разрешениями
- **Проверять личность (Verify Identity)**: Определяйте аккаунт AWS, ARN и идентификатор пользователя, связанные с вызывающими учётными данными
- **Генерировать токены сессии**: Получайте временные учётные данные с опциональной защитой MFA для повышенной безопасности
- **Аудировать ключи доступа**: Определяйте аккаунт AWS, которому принадлежит данный ключ доступа, для расследований безопасности

В Sim интеграция с AWS STS позволяет вашим агентам управлять временными учётными данными в рамках автоматизированных рабочих процессов. Это полезно для сценариев кросс-аккаунтного доступа, ротации учётных данных, проверки личности перед выполнением чувствительных операций и аудита безопасности. Агенты могут принимать роли для взаимодействия с другими сервисами AWS, проверять свою собственную личность или определять владельца ключа доступа без раскрытия долгосрочных учётных данных.

## Инструкции по использованию

Интегрируйте AWS STS в рабочий процесс. Принимайте роли, получайте временные учётные данные, проверяйте личность вызывающего и получайте информацию о ключах доступа.



## Действия

### `sts_assume_role`

Принятие роли IAM и получение временных учётных данных безопасности

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `region` | string | Да | Регион AWS \(например, us-east-1\) |
| `accessKeyId` | string | Да | Идентификатор ключа доступа AWS |
| `secretAccessKey` | string | Да | Секретный ключ доступа AWS |
| `roleArn` | string | Да | ARN роли IAM для принятия |
| `roleSessionName` | string | Да | Идентификатор для сессии принятой роли |
| `durationSeconds` | number | Нет | Длительность сессии в секундах \(900-43200, по умолчанию 3600\) |
| `policy` | string | Нет | JSON политики IAM для дополнительного ограничения разрешений сессии \(макс. 2048 символов\) |
| `externalId` | string | Нет | Внешний идентификатор для кросс-аккаунтного доступа |
| `serialNumber` | string | Нет | Серийный номер или ARN устройства MFA |
| `tokenCode` | string | Нет | Код токена MFA \(6 цифр\) |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `accessKeyId` | string | Временный идентификатор ключа доступа |
| `secretAccessKey` | string | Временный секретный ключ доступа |
| `sessionToken` | string | Временный токен сессии |
| `expiration` | string | Временная метка истечения срока действия учётных данных |
| `assumedRoleArn` | string | ARN принятой роли |
| `assumedRoleId` | string | Идентификатор принятой роли с именем сессии |
| `packedPolicySize` | number | Процент использования допустимого размера политики |
| `sourceIdentity` | string | Исходная личность, установленная на сессии роли, если есть |

### `sts_get_caller_identity`

Получение сведений о пользователе IAM или роли, чьи учётные данные используются для вызова API

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `region` | string | Да | Регион AWS \(например, us-east-1\) |
| `accessKeyId` | string | Да | Идентификатор ключа доступа AWS |
| `secretAccessKey` | string | Да | Секретный ключ доступа AWS |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `account` | string | Идентификатор аккаунта AWS |
| `arn` | string | ARN вызывающей сущности |
| `userId` | string | Уникальный идентификатор вызывающей сущности |

### `sts_get_session_token`

Получение временных учётных данных безопасности для пользователя IAM, опционально с MFA

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `region` | string | Да | Регион AWS \(например, us-east-1\) |
| `accessKeyId` | string | Да | Идентификатор ключа доступа AWS |
| `secretAccessKey` | string | Да | Секретный ключ доступа AWS |
| `durationSeconds` | number | Нет | Длительность сессии в секундах \(900-129600, по умолчанию 43200\) |
| `serialNumber` | string | Нет | Серийный номер или ARN устройства MFA |
| `tokenCode` | string | Нет | Код токена MFA \(6 цифр\) |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `accessKeyId` | string | Временный идентификатор ключа доступа |
| `secretAccessKey` | string | Временный секретный ключ доступа |
| `sessionToken` | string | Временный токен сессии |
| `expiration` | string | Временная метка истечения срока действия учётных данных |

### `sts_get_access_key_info`

Получение идентификатора аккаунта AWS, связанного с ключом доступа

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `region` | string | Да | Регион AWS \(например, us-east-1\) |
| `accessKeyId` | string | Да | Идентификатор ключа доступа AWS |
| `secretAccessKey` | string | Да | Секретный ключ доступа AWS |
| `targetAccessKeyId` | string | Да | Идентификатор ключа доступа для поиска |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `account` | string | Идентификатор аккаунта AWS, которому принадлежит ключ доступа |
