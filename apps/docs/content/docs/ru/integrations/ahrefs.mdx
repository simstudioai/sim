[Ahrefs](https://ahrefs.com/) — это ведущий набор SEO-инструментов для анализа веб-сайтов, отслеживания позиций, мониторинга обратных ссылок и исследования ключевых слов. Он предоставляет подробную информацию как о вашем собственном сайте, так и о сайтах конкурентов, помогая принимать обоснованные решения для улучшения видимости в поисковых системах.

С интеграцией Ahrefs в Sim вы можете:

- **Анализировать Domain Rating и авторитетность**: Мгновенно проверять Domain Rating (DR) и Ahrefs Rank любого сайта для оценки его авторитетности.
- **Получать обратные ссылки**: Извлекать список обратных ссылок, указывающих на сайт или конкретный URL, с такими деталями, как анкорный текст, DR ссылающейся страницы и многое другое.
- **Получать статистику обратных ссылок**: Получать метрики по типам обратных ссылок (dofollow, nofollow, текстовые, изображения, редиректы и т.д.) для домена или URL.
- **Исследовать органические ключевые слова** *(планируется)*: Просматривать ключевые слова, по которым домен ранжируется, и их позиции в результатах поиска Google.
- **Находить топ-страницы** *(планируется)*: Определять наиболее эффективные страницы по органическому трафику и ссылкам.

Эти инструменты позволяют вашим агентам автоматизировать SEO-исследования, отслеживать конкурентов и генерировать отчёты — всё в рамках автоматизации рабочих процессов. Для использования интеграции Ahrefs вам потребуется подписка Ahrefs Enterprise с доступом к API.

## Инструкция по использованию

Интегрируйте SEO-инструменты Ahrefs в ваш рабочий процесс. Анализируйте рейтинги доменов, обратные ссылки, органические ключевые слова, топ-страницы и многое другое. Требуется тарифный план Ahrefs Enterprise с доступом к API.



## Действия

### `ahrefs_domain_rating`

Получение Domain Rating (DR) и Ahrefs Rank для целевого домена. Domain Rating показывает силу профиля обратных ссылок сайта по шкале от 0 до 100.

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `target` | string | Yes | Целевой домен для анализа \(например, example.com\) |
| `date` | string | No | Дата для исторических данных в формате YYYY-MM-DD \(по умолчанию — сегодня\) |
| `apiKey` | string | Yes | API-ключ Ahrefs |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `domainRating` | number | Оценка Domain Rating \(0-100\) |
| `ahrefsRank` | number | Ahrefs Rank — глобальный рейтинг на основе силы профиля обратных ссылок |

### `ahrefs_backlinks`

Получение списка обратных ссылок, указывающих на целевой домен или URL. Возвращает детали о каждой обратной ссылке, включая исходный URL, анкорный текст и рейтинг домена.

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `target` | string | Yes | Целевой домен или URL для анализа. Пример: "example.com" или "https://example.com/page" |
| `mode` | string | No | Режим анализа: domain \(весь домен\), prefix \(префикс URL\), subdomains \(включая все поддомены\), exact \(точное совпадение URL\). Пример: "domain" |
| `date` | string | No | Дата для исторических данных в формате YYYY-MM-DD \(по умолчанию — сегодня\) |
| `limit` | number | No | Максимальное количество возвращаемых результатов. Пример: 50 \(по умолчанию: 100\) |
| `offset` | number | No | Количество пропускаемых результатов для пагинации. Пример: 100 |
| `apiKey` | string | Yes | API-ключ Ahrefs |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `backlinks` | array | Список обратных ссылок |
|   ↳ `urlFrom` | string | URL страницы, содержащей ссылку |
|   ↳ `urlTo` | string | Целевой URL ссылки |
|   ↳ `anchor` | string | Анкорный текст ссылки |
|   ↳ `domainRatingSource` | number | Domain Rating исходного домена |

### `ahrefs_backlinks_stats`

Получение агрегированной статистики обратных ссылок для домена или URL, включая распределение по типам ссылок.

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `target` | string | Yes | Целевой домен или URL для анализа. Пример: "example.com" или "https://example.com/page" |
| `mode` | string | No | Режим анализа: domain \(весь домен\), prefix \(префикс URL\), subdomains \(включая все поддомены\), exact \(точное совпадение URL\). Пример: "domain" |
| `date` | string | No | Дата для исторических данных в формате YYYY-MM-DD \(по умолчанию — сегодня\) |
| `apiKey` | string | Yes | API-ключ Ahrefs |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `stats` | object | Статистика обратных ссылок |
|   ↳ `total` | number | Общее количество обратных ссылок |
|   ↳ `dofollow` | number | Количество dofollow-ссылок |
|   ↳ `nofollow` | number | Количество nofollow-ссылок |
|   ↳ `text` | number | Количество текстовых ссылок |
|   ↳ `image` | number | Количество ссылок-изображений |
|   ↳ `redirect` | number | Количество ссылок-редиректов |
|   ↳ `gov` | number | Количество ссылок с доменов .gov |
|   ↳ `edu` | number | Количество ссылок с доменов .edu |

### `ahrefs_organic_keywords`

Получение списка органических ключевых слов, по которым целевой URL или домен ранжируется в результатах поиска. Возвращает ключевые слова с объёмом поиска, позицией в рейтинге и оценкой трафика.

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `target` | string | Yes | Целевой домен или URL для анализа. Пример: "example.com" или "https://example.com/page" |
| `country` | string | No | Код страны для поисковых результатов. Пример: "us", "gb", "de" \(по умолчанию: "us"\) |
| `mode` | string | No | Режим анализа: domain \(весь домен\), prefix \(префикс URL\), subdomains \(включая все поддомены\), exact \(точное совпадение URL\). Пример: "domain" |
| `date` | string | No | Дата для исторических данных в формате YYYY-MM-DD \(по умолчанию — сегодня\) |
| `limit` | number | No | Максимальное количество возвращаемых результатов. Пример: 50 \(по умолчанию: 100\) |
| `offset` | number | No | Количество пропускаемых результатов для пагинации. Пример: 100 |
| `apiKey` | string | Yes | API-ключ Ahrefs |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `keywords` | array | Список органических ключевых слов, по которым цель ранжируется |
|   ↳ `keyword` | string | Ключевое слово |
|   ↳ `volume` | number | Месячный объём поиска |
|   ↳ `position` | number | Текущая позиция в рейтинге |
|   ↳ `url` | string | URL, который ранжируется по этому ключевому слову |
|   ↳ `traffic` | number | Оценочный месячный органический трафик |
|   ↳ `keywordDifficulty` | number | Оценка сложности ключевого слова \(0-100\) |

### `ahrefs_top_pages`

Получение топ-страниц целевого домена, отсортированных по органическому трафику. Возвращает URL страниц с их трафиком, количеством ключевых слов и оценочной стоимостью трафика.

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `target` | string | Yes | Целевой домен для анализа. Пример: "example.com" |
| `country` | string | No | Код страны для данных трафика. Пример: "us", "gb", "de" \(по умолчанию: "us"\) |
| `mode` | string | No | Режим анализа: domain \(весь домен\), prefix \(префикс URL\), subdomains \(включая все поддомены\). Пример: "domain" |
| `date` | string | No | Дата для исторических данных в формате YYYY-MM-DD \(по умолчанию — сегодня\) |
| `limit` | number | No | Максимальное количество возвращаемых результатов. Пример: 50 \(по умолчанию: 100\) |
| `offset` | number | No | Количество пропускаемых результатов для пагинации. Пример: 100 |
| `select` | string | No | Список полей для возврата через запятую \(например, url,traffic,keywords,top_keyword,value\). По умолчанию: url,traffic,keywords,top_keyword,value |
| `apiKey` | string | Yes | API-ключ Ahrefs |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `pages` | array | Список топ-страниц по органическому трафику |
|   ↳ `url` | string | URL страницы |
|   ↳ `traffic` | number | Оценочный месячный органический трафик |
|   ↳ `keywords` | number | Количество ключевых слов, по которым страница ранжируется |
|   ↳ `topKeyword` | string | Основное ключевое слово, приносящее трафик на эту страницу |
|   ↳ `value` | number | Оценочная стоимость трафика в USD |

### `ahrefs_keyword_overview`

Получение подробных метрик для ключевого слова, включая объём поиска, сложность ключевого слова, CPC, клики и потенциал трафика.

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `keyword` | string | Yes | Ключевое слово для анализа |
| `country` | string | No | Код страны для данных ключевого слова. Пример: "us", "gb", "de" \(по умолчанию: "us"\) |
| `apiKey` | string | Yes | API-ключ Ahrefs |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `overview` | object | Обзор метрик ключевого слова |
|   ↳ `keyword` | string | Проанализированное ключевое слово |
|   ↳ `searchVolume` | number | Месячный объём поиска |
|   ↳ `keywordDifficulty` | number | Оценка сложности ключевого слова \(0-100\) |
|   ↳ `cpc` | number | Цена за клик в USD |
|   ↳ `clicks` | number | Оценочное количество кликов в месяц |
|   ↳ `clicksPercentage` | number | Процент поисковых запросов, приводящих к кликам |
|   ↳ `parentTopic` | string | Родительская тема для этого ключевого слова |
|   ↳ `trafficPotential` | number | Оценочный потенциал трафика при ранжировании на позиции #1 |

### `ahrefs_broken_backlinks`

Получение списка битых обратных ссылок, указывающих на целевой домен или URL. Полезно для выявления возможностей восстановления ссылок.

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `target` | string | Yes | Целевой домен или URL для анализа. Пример: "example.com" или "https://example.com/page" |
| `mode` | string | No | Режим анализа: domain \(весь домен\), prefix \(префикс URL\), subdomains \(включая все поддомены\), exact \(точное совпадение URL\). Пример: "domain" |
| `date` | string | No | Дата для исторических данных в формате YYYY-MM-DD \(по умолчанию — сегодня\) |
| `limit` | number | No | Максимальное количество возвращаемых результатов. Пример: 50 \(по умолчанию: 100\) |
| `offset` | number | No | Количество пропускаемых результатов для пагинации. Пример: 100 |
| `apiKey` | string | Yes | API-ключ Ahrefs |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `brokenBacklinks` | array | Список битых обратных ссылок |
|   ↳ `urlFrom` | string | URL страницы, содержащей битую ссылку |
|   ↳ `urlTo` | string | Битый URL, на который указывает ссылка |
|   ↳ `httpCode` | number | Код статуса HTTP \(например, 404, 410\) |
|   ↳ `anchor` | string | Анкорный текст ссылки |
|   ↳ `domainRatingSource` | number | Domain Rating ссылающегося домена |
