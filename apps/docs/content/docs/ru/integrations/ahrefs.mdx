[Ahrefs](https://ahrefs.com/)Ahrefshttps://ahrefs.com/является ведущим набором инструментов для анализа веб-сайтов, отслеживания позиций в поисковых системах, мониторинга обратных ссылок и исследования ключевых слов. Он предоставляет подробную информацию о вашем собственном сайте, а также о сайтах ваших конкурентов, помогая вам принимать решения, основанные на данных, для улучшения видимости вашего сайта в поисковых системах.

Благодаря интеграции с Ahrefs в Sim, вы можете:

- **Оцените рейтинг и авторитет домена:** Мгновенно проверьте рейтинг домена (DR) и позицию в рейтинге Ahrefs для любого веб-сайта, чтобы оценить его авторитетность.- **Получение обратных ссылок:** Получите список обратных ссылок, указывающих на сайт или конкретный URL, с подробной информацией, такой как текст анкора, рейтинг домена страницы-ссылки и т.д.- **Получите статистику обратных ссылок:** Получите данные о типах обратных ссылок (с атрибутом dofollow, без атрибута dofollow, текстовые, графические, перенаправление и т.д.) для домена или URL-адреса.- **Изучите органические ключевые слова** (*планируется*): Просмотрите список ключевых слов, по которым домен занимает определенные позиции в результатах поиска Google.- **Определите самые эффективные страницы** (*планируется*): Выявите страницы с наибольшим органическим трафиком и ссылками.
Эти инструменты позволяют вашим сотрудникам автоматизировать исследования в области SEO, отслеживать конкурентов и создавать отчеты – все это как часть ваших автоматизированных рабочих процессов. Для использования интеграции Ahrefs вам потребуется подписка на Ahrefs Enterprise с доступом к API.

## Инструкция по использованию
Интегрируйте инструменты SEO от Ahrefs в свою работу. Анализируйте рейтинги доменов, обратные ссылки, ключевые слова для органической выдачи, наиболее популярные страницы и многое другое. Требуется план Ahrefs Enterprise с доступом к API.



## Действия
### `ahrefs_domain_rating`

Получите рейтинг домена (Domain Rating, DR) и рейтинг Ahrefs для целевого домена. Рейтинг домена показывает силу профиля обратных ссылок веб-сайта в диапазоне от 0 до 100.

#### Ввод
|  | Параметр | Тип | Необходимо | Описание |  || --------- | ---- | -------- | ----------- |
|  | `target` | строка | Да | Целевой домен для анализа (например, example.com) |  ||  | `date` | строка | Нет | Дата для исторических данных в формате YYYY-MM-DD (по умолчанию – текущая дата) |  ||  | `apiKey` | строка | Да | Ключ API Ahrefs |  |
#### Вывод
|  | Параметр | Тип | Описание |  || --------- | ---- | ----------- |
|  | `domainRating` | число | Оценка рейтинга домена (от 0 до 100) |  ||  | `ahrefsRank` | число | Рейтинг Ahrefs – глобальный рейтинг, основанный на силе профиля обратных ссылок. |  |
### `ahrefs_backlinks`

Получите список обратных ссылок, указывающих на целевой домен или URL. Возвращает информацию о каждой обратной ссылке, включая исходный URL, текст ссылки и рейтинг домена.

#### Ввод
|  | Параметр | Тип | Необходимо | Описание |  || --------- | ---- | -------- | ----------- |
|  | `target` | строка | Да | Целевой домен или URL для анализа. Пример: «example.com» или «https://example.com/page». |  ||  | `mode` | строка | Нет | Режим анализа: домен (весь домен), префикс (префикс URL), поддомены (включая все поддомены), точное соответствие (точное совпадение с URL). Пример: "домен" |  ||  | `date` | строка | Нет | Дата для исторических данных в формате YYYY-MM-DD (по умолчанию – текущая дата) |  ||  | `limit` | число | Нет | Максимальное количество результатов, которые необходимо вернуть. Пример: 50 (по умолчанию: 100) |  ||  | `offset` | число | Нет | Количество результатов, которые нужно пропустить для постраничной навигации. Пример: 100 |  ||  | `apiKey` | строка | Да | Ключ API Ahrefs |  |
#### Вывод
|  | Параметр | Тип | Описание |  || --------- | ---- | ----------- |
|  | `backlinks` | массив | Список обратных ссылок, указывающих на целевой сайт. |  ||  | ↳ `urlFrom` | строка | URL страницы, содержащей обратную ссылку. |  ||  | ↳ `urlTo` | строка | URL, на который осуществляется переход. |  ||  | ↳ `anchor` | строка | Текст ссылки |  ||  | ↳ `domainRatingSource` | число | Оценка домена, на который ведет ссылка |  ||  | ↳ `isDofollow` | логическое значение | Будет ли ссылка вести на целевой сайт? |  ||  | ↳ `firstSeen` | строка | Когда обратная ссылка была впервые обнаружена |  ||  | ↳ `lastVisited` | строка | Когда последний раз проверялась обратная ссылка |  |
