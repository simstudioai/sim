[Ahrefs](https://ahrefs.com/)Ahrefshttps://ahrefs.com/является ведущим набором инструментов для анализа веб-сайтов, отслеживания позиций в поисковых системах, мониторинга обратных ссылок и исследования ключевых слов. Он предоставляет подробную информацию как о вашем собственном сайте, так и о сайтах ваших конкурентов, помогая вам принимать решения, основанные на данных, для улучшения видимости вашего сайта в поисковых системах.

Благодаря интеграции с Ahrefs в Sim, вы можете:

- **Оцените рейтинг и авторитет домена:** Мгновенно проверьте рейтинг домена (DR) и позицию в рейтинге Ahrefs для любого веб-сайта, чтобы оценить его авторитетность.- **Получение обратных ссылок:** Получите список обратных ссылок, указывающих на сайт или конкретный URL, с подробной информацией, такой как текст анкора, рейтинг домена страницы-ссылки и т.д.- **Получите статистику обратных ссылок:** Получите данные о типах обратных ссылок (с атрибутом dofollow, без атрибута dofollow, текстовые, графические, перенаправление и т.д.) для домена или URL-адреса.- **Изучите органические ключевые слова** (*планируется*): Просмотрите список ключевых слов, по которым домен занимает определенные позиции в результатах поиска Google.- **Определите самые эффективные страницы** *(планируется)*: Выявите страницы с наибольшим органическим трафиком и ссылками.
Эти инструменты позволяют вашим сотрудникам автоматизировать исследования в области SEO, отслеживать конкурентов и создавать отчеты – все это как часть ваших автоматизированных рабочих процессов. Для использования интеграции Ahrefs вам потребуется подписка на Ahrefs Enterprise с доступом к API.

## Инструкция по использованию
Интегрируйте инструменты SEO от Ahrefs в свою работу. Анализируйте рейтинги доменов, обратные ссылки, ключевые слова для органической выдачи, наиболее популярные страницы и многое другое. Требуется план Ahrefs Enterprise с доступом к API.



## Действия
### `ahrefs_domain_rating`

Получите рейтинг домена (Domain Rating, DR) и рейтинг Ahrefs для целевого домена. Рейтинг домена показывает силу профиля обратных ссылок веб-сайта в диапазоне от 0 до 100.

#### Ввод
|  | Параметр | Тип | Необходимо | Описание |  || --------- | ---- | -------- | ----------- |
|  | `target` | строка | Да | Целевой домен для анализа (например, example.com) |  ||  | `date` | строка | Нет | Дата для исторических данных в формате YYYY-MM-DD (по умолчанию – текущая дата) |  ||  | `apiKey` | строка | Да | Ключ API Ahrefs |  |
#### Вывод
|  | Параметр | Тип | Описание |  || --------- | ---- | ----------- |
|  | `domainRating` | число | Оценка рейтинга домена (от 0 до 100) |  ||  | `ahrefsRank` | число | Рейтинг Ahrefs – глобальный рейтинг, основанный на силе профиля обратных ссылок. |  |
### `ahrefs_backlinks`

Получите список обратных ссылок, указывающих на целевой домен или URL. Возвращает информацию о каждой обратной ссылке, включая исходный URL, текст ссылки и рейтинг домена.

#### Ввод
|  | Параметр | Тип | Необходимо | Описание |  || --------- | ---- | -------- | ----------- |
|  | `target` | строка | Да | Целевой домен или URL для анализа. Пример: «example.com» или «https://example.com/page». |  ||  | `mode` | строка | Нет | Режим анализа: домен (весь домен), префикс (префикс URL), поддомены (включая все поддомены), точное соответствие (точное совпадение с URL). Пример: "домен" |  ||  | `date` | string | No | Date for historical data in YYYY-MM-DD format \(defaults to today\) |  ||  | `limit` | number | No | Maximum number of results to return. Example: 50 \(default: 100\) |  ||  | `offset` | number | No | Number of results to skip for pagination. Example: 100 |  ||  | `apiKey` | string | Yes | Ahrefs API Key |  |
#### Output
|  | Parameter | Type | Description |  || --------- | ---- | ----------- |
|  | `backlinks` | array | List of backlinks pointing to the target |  ||  | ↳ `urlFrom` | string | The URL of the page containing the backlink |  ||  | ↳ `urlTo` | string | The URL being linked to |  ||  | ↳ `anchor` | string | The anchor text of the link |  ||  | ↳ `domainRatingSource` | number | Domain Rating of the linking domain |  ||  | ↳ `isDofollow` | boolean | Whether the link is dofollow |  ||  | ↳ `firstSeen` | string | When the backlink was first discovered |  ||  | ↳ `lastVisited` | string | When the backlink was last checked |  |
### `ahrefs_backlinks_stats`

Get backlink statistics for a target domain or URL. Returns totals for different backlink types including dofollow, nofollow, text, image, and redirect links.

#### Input
|  | Parameter | Type | Required | Description |  || --------- | ---- | -------- | ----------- |
|  | `target` | string | Yes | The target domain or URL to analyze. Example: "example.com" or "https://example.com/page" |  ||  | `mode` | string | No | Analysis mode: domain \(entire domain\), prefix \(URL prefix\), subdomains \(include all subdomains\), exact \(exact URL match\). Example: "domain" |  ||  | `date` | string | No | Date for historical data in YYYY-MM-DD format \(defaults to today\) |  ||  | `apiKey` | string | Yes | Ahrefs API Key |  |
#### Output
|  | Parameter | Type | Description |  || --------- | ---- | ----------- |
|  | `stats` | object | Backlink statistics summary |  ||  | ↳ `total` | number | Total number of live backlinks |  ||  | ↳ `dofollow` | number | Number of dofollow backlinks |  ||  | ↳ `nofollow` | number | Number of nofollow backlinks |  ||  | ↳ `text` | number | Number of text backlinks |  ||  | ↳ `image` | number | Number of image backlinks |  ||  | ↳ `redirect` | number | Number of redirect backlinks |  |
### `ahrefs_referring_domains`

Get a list of domains that link to a target domain or URL. Returns unique referring domains with their domain rating, backlink counts, and discovery dates.

#### Input
|  | Parameter | Type | Required | Description |  || --------- | ---- | -------- | ----------- |
|  | `target` | string | Yes | The target domain or URL to analyze. Example: "example.com" or "https://example.com/page" |  ||  | `mode` | string | No | Analysis mode: domain \(entire domain\), prefix \(URL prefix\), subdomains \(include all subdomains\), exact \(exact URL match\). Example: "domain" |  ||  | `date` | string | No | Date for historical data in YYYY-MM-DD format \(defaults to today\) |  ||  | `limit` | number | No | Maximum number of results to return. Example: 50 \(default: 100\) |  ||  | `offset` | number | No | Number of results to skip for pagination. Example: 100 |  ||  | `apiKey` | string | Yes | Ahrefs API Key |  |
#### Output
|  | Parameter | Type | Description |  || --------- | ---- | ----------- |
|  | `referringDomains` | array | List of domains linking to the target |  ||  | ↳ `domain` | string | The referring domain |  ||  | ↳ `domainRating` | number | Domain Rating of the referring domain |  ||  | ↳ `backlinks` | number | Total number of backlinks from this domain |  ||  | ↳ `dofollowBacklinks` | number | Number of dofollow backlinks from this domain |  ||  | ↳ `firstSeen` | string | When the domain was first seen linking |  ||  | ↳ `lastVisited` | string | When the domain was last checked |  |
### `ahrefs_organic_keywords`

Get organic keywords that a target domain or URL ranks for in Google search results. Returns keyword details including search volume, ranking position, and estimated traffic.

#### Input
|  | Parameter | Type | Required | Description |  || --------- | ---- | -------- | ----------- |
|  | `target` | string | Yes | The target domain or URL to analyze. Example: "example.com" or "https://example.com/page" |  ||  | `country` | string | No | Country code for search results. Example: "us", "gb", "de" \(default: "us"\) |  ||  | `mode` | string | No | Analysis mode: domain \(entire domain\), prefix \(URL prefix\), subdomains \(include all subdomains\), exact \(exact URL match\). Example: "domain" |  ||  | `date` | string | No | Date for historical data in YYYY-MM-DD format \(defaults to today\) |  ||  | `limit` | number | No | Maximum number of results to return. Example: 50 \(default: 100\) |  ||  | `offset` | number | No | Number of results to skip for pagination. Example: 100 |  ||  | `apiKey` | string | Yes | Ahrefs API Key |  |
#### Output
|  | Parameter | Type | Description |  || --------- | ---- | ----------- |
|  | `keywords` | array | List of organic keywords the target ranks for |  ||  | ↳ `keyword` | string | The keyword |  ||  | ↳ `volume` | number | Monthly search volume |  ||  | ↳ `position` | number | Current ranking position |  ||  | ↳ `url` | string | The URL that ranks for this keyword |  ||  | ↳ `traffic` | number | Estimated monthly organic traffic |  ||  | ↳ `keywordDifficulty` | number | Keyword difficulty score \(0-100\) |  |
### `ahrefs_top_pages`

Get the top pages of a target domain sorted by organic traffic. Returns page URLs with their traffic, keyword counts, and estimated traffic value.

#### Input
|  | Parameter | Type | Required | Description |  || --------- | ---- | -------- | ----------- |
|  | `target` | string | Yes | The target domain to analyze. Example: "example.com" |  ||  | `country` | string | No | Country code for traffic data. Example: "us", "gb", "de" \(default: "us"\) |  ||  | `mode` | string | No | Analysis mode: domain \(entire domain\), prefix \(URL prefix\), subdomains \(include all subdomains\). Example: "domain" |  ||  | `date` | string | No | Date for historical data in YYYY-MM-DD format \(defaults to today\) |  ||  | `limit` | number | No | Maximum number of results to return. Example: 50 \(default: 100\) |  ||  | `offset` | number | No | Number of results to skip for pagination. Example: 100 |  ||  | `select` | string | No | Comma-separated list of fields to return \(e.g., url,traffic,keywords,top_keyword,value\). Default: url,traffic,keywords,top_keyword,value |  ||  | `apiKey` | string | Yes | Ahrefs API Key |  |
#### Output
|  | Parameter | Type | Description |  || --------- | ---- | ----------- |
|  | `pages` | array | List of top pages by organic traffic |  ||  | ↳ `url` | string | The page URL |  ||  | ↳ `traffic` | number | Estimated monthly organic traffic |  ||  | ↳ `keywords` | number | Number of keywords the page ranks for |  ||  | ↳ `topKeyword` | string | The top keyword driving traffic to this page |  ||  | ↳ `value` | number | Estimated traffic value in USD |  |

Get detailed metrics for a keyword including search volume, keyword difficulty, CPC, clicks, and traffic potential.

#### Input
|  | Parameter | Type | Required | Description |  || --------- | ---- | -------- | ----------- |
|  | `keyword` | string | Yes | The keyword to analyze |  ||  | `country` | string | No | Country code for keyword data. Example: "us", "gb", "de" \(default: "us"\) |  ||  | `apiKey` | string | Yes | Ahrefs API Key |  |
#### Output
|  | Parameter | Type | Description |  || --------- | ---- | ----------- |
|  | `overview` | object | Keyword metrics overview |  ||  | ↳ `keyword` | string | The analyzed keyword |  ||  | ↳ `searchVolume` | number | Monthly search volume |  ||  | ↳ `keywordDifficulty` | number | Keyword difficulty score \(0-100\) |  ||  | ↳ `cpc` | number | Cost per click in USD |  ||  | ↳ `clicks` | number | Estimated clicks per month |  ||  | ↳ `clicksPercentage` | number | Percentage of searches that result in clicks |  ||  | ↳ `parentTopic` | string | The parent topic for this keyword |  ||  | ↳ `trafficPotential` | number | Estimated traffic potential if ranking #1 |  |
