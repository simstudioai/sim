[Google Docs](https://docs.google.com) — это облачный сервис Google для совместной работы с документами, позволяющий пользователям создавать, редактировать и делиться документами в реальном времени. Являясь неотъемлемой частью Google Workspace, Docs предлагает богатые инструменты форматирования, комментирование, историю версий и бесшовную интеграцию с другими инструментами Google для продуктивной работы.

Google Docs позволяет отдельным пользователям и командам:

- **Создавать и форматировать документы:** разрабатывать документы с расширенным форматированием, изображениями и таблицами.
- **Совместно работать и комментировать:** несколько пользователей могут мгновенно редактировать и комментировать с предложениями.
- **Отслеживать изменения и историю версий:** просматривать, откатывать и управлять правками с течением времени.
- **Доступ с любого устройства:** работать с документами через веб, мобильные устройства или компьютер с полной облачной синхронизацией.
- **Интеграция с сервисами Google:** связывать Docs с Drive, Sheets, Slides и внешними платформами для мощных рабочих процессов.

В Sim интеграция Google Docs позволяет вашим агентам читать содержимое документов, записывать новый контент и создавать документы программно в рамках автоматизированных рабочих процессов. Эта интеграция открывает возможности автоматизации, такие как генерация документов, написание отчётов, извлечение контента и совместное редактирование — сокращая разрыв между рабочими процессами на основе ИИ и управлением документами в вашей организации.

## Инструкции по использованию

Интеграция Google Docs в рабочий процесс. Позволяет читать, записывать и создавать документы.



## Действия

### `google_docs_read`

Чтение содержимого документа Google Docs

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `documentId` | string | Да | Идентификатор документа Google Docs |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `content` | string | Извлечённое текстовое содержимое документа |
| `metadata` | json | Метаданные документа, включая ID, название и URL |
|   ↳ `documentId` | string | Идентификатор документа Google Docs |
|   ↳ `title` | string | Название документа |
|   ↳ `mimeType` | string | MIME-тип документа |
|   ↳ `url` | string | URL документа |

### `google_docs_write`

Добавление содержимого в документ Google Docs. Содержимое вставляется буквально; Markdown не интерпретируется. Для форматированного вывода из Markdown используйте операцию Create с включённой опцией markdown.

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `documentId` | string | Да | Идентификатор документа для записи |
| `content` | string | Да | Содержимое для записи в документ |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `updatedContent` | boolean | Указывает, было ли содержимое документа успешно обновлено |
| `metadata` | json | Обновлённые метаданные документа, включая ID, название и URL |
|   ↳ `documentId` | string | Идентификатор документа Google Docs |
|   ↳ `title` | string | Название документа |
|   ↳ `mimeType` | string | MIME-тип документа |
|   ↳ `url` | string | URL документа |

### `google_docs_create`

Создание нового документа Google Docs

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `title` | string | Да | Название создаваемого документа |
| `content` | string | Нет | Содержимое создаваемого документа |
| `folderSelector` | string | Нет | Идентификатор папки Google Drive для создания документа \(например, 1ABCxyz...\) |
| `folderId` | string | Нет | Идентификатор папки для создания документа \(внутреннее использование\) |
| `markdown` | boolean | Нет | Если true, содержимое интерпретируется как Markdown и преобразуется в форматированное содержимое Google Docs \(заголовки, жирный/курсив, списки, таблицы, ссылки, блоки кода, цитаты\). По умолчанию: false \(содержимое вставляется как обычный текст\). |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `metadata` | json | Метаданные созданного документа, включая ID, название и URL |
|   ↳ `documentId` | string | Идентификатор документа Google Docs |
|   ↳ `title` | string | Название документа |
|   ↳ `mimeType` | string | MIME-тип документа |
|   ↳ `url` | string | URL документа |
