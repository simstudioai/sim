## Инструкции по использованию

Integrate MySQL into the workflow. Can query, insert, update, delete, and execute raw SQL.



## Действия

### `mysql_query`

Execute SELECT query on MySQL database

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `host` | string | Да | MySQL server hostname or IP address |
| `port` | number | Да | MySQL server port \(default: 3306\) |
| `database` | string | Да | Database name to connect to \(e.g., my_database\) |
| `username` | string | Да | Database username |
| `password` | string | Да | Database password |
| `ssl` | string | Нет | SSL connection mode \(disabled, required, preferred\) |
| `query` | string | Да | SQL SELECT query to execute \(e.g., SELECT * FROM users WHERE active = 1\) |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `message` | string | Operation status message |
| `rows` | array | Array of rows returned from the query |
| `rowCount` | number | Number of rows returned |
