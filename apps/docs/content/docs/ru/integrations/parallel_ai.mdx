[Parallel AI](https://parallel.ai/) — это продвинутая платформа для веб-поиска и извлечения контента, разработанная для предоставления комплексных, высококачественных результатов по любому запросу. Используя интеллектуальную обработку и масштабное извлечение данных, Parallel AI позволяет пользователям и агентам получать доступ, анализировать и синтезировать информацию из интернета с высокой скоростью и точностью.

С помощью Parallel AI вы можете:

- **Интеллектуально искать в интернете**: получать актуальную информацию из широкого спектра источников  
- **Извлекать и обобщать контент**: получать краткие, содержательные выдержки с веб-страниц и из документов  
- **Настраивать цели поиска**: адаптировать запросы под конкретные потребности или вопросы для целевых результатов  
- **Обрабатывать результаты в масштабе**: работать с большими объёмами результатов поиска с помощью продвинутых опций обработки  
- **Интегрировать с рабочими процессами**: использовать Parallel AI в Sim для автоматизации исследований, сбора контента и извлечения знаний  
- **Контролировать детализацию вывода**: указывать количество результатов и объём контента на каждый результат  
- **Обеспечить безопасный доступ к API**: защищать ваши поиски и данные с помощью аутентификации по API-ключу

В Sim интеграция с Parallel AI даёт вашим агентам возможность выполнять веб-поиск и извлекать контент программно. Это открывает возможности для мощных сценариев автоматизации, таких как исследование в реальном времени, конкурентный анализ, мониторинг контента и создание баз знаний. Подключив Sim к Parallel AI, вы открываете возможность для агентов собирать, обрабатывать и использовать веб-данные в рамках ваших автоматизированных рабочих процессов.

## Инструкции по использованию

Интегрируйте Parallel AI в рабочий процесс. Можно выполнять поиск в интернете, извлекать информацию из URL-адресов и проводить глубокие исследования.



## Действия

### `parallel_search`

Поиск в интернете с помощью Parallel AI. Предоставляет комплексные результаты поиска с интеллектуальной обработкой и извлечением контента.

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `objective` | string | Да | Цель поиска или вопрос, на который нужно ответить |
| `search_queries` | string | Нет | Список поисковых запросов через запятую |
| `mode` | string | Нет | Режим поиска: one-shot, agentic или fast \(по умолчанию: one-shot\) |
| `max_results` | number | Нет | Максимальное количество результатов \(по умолчанию: 10\) |
| `max_chars_per_result` | number | Нет | Максимальное количество символов на выдержку результата \(минимум: 1000\) |
| `include_domains` | string | Нет | Список доменов через запятую для ограничения результатов поиска |
| `exclude_domains` | string | Нет | Список доменов через запятую для исключения из результатов поиска |
| `apiKey` | string | Да | API-ключ Parallel AI |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `search_id` | string | Уникальный идентификатор этого поискового запроса |
| `results` | array | Результаты поиска с выдержками с релевантных страниц |
|   ↳ `url` | string | URL результата поиска |
|   ↳ `title` | string | Заголовок результата поиска |
|   ↳ `publish_date` | string | Дата публикации страницы \(ГГГГ-ММ-ДД\) |
|   ↳ `excerpts` | array | Оптимизированные для LLM выдержки со страницы |

### `parallel_extract`

Извлечение целевой информации из определённых URL-адресов с помощью Parallel AI. Обрабатывает предоставленные URL для получения релевантного контента на основе вашей цели.

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `urls` | string | Да | Список URL-адресов через запятую для извлечения информации |
| `objective` | string | Нет | Какую информацию извлечь из предоставленных URL-адресов |
| `excerpts` | boolean | Нет | Включить релевантные выдержки из контента \(по умолчанию: true\) |
| `full_content` | boolean | Нет | Включить полное содержимое страницы в формате markdown \(по умолчанию: false\) |
| `apiKey` | string | Да | API-ключ Parallel AI |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `extract_id` | string | Уникальный идентификатор этого запроса на извлечение |
| `results` | array | Извлечённая информация с предоставленных URL-адресов |
|   ↳ `url` | string | Исходный URL |
|   ↳ `title` | string | Заголовок страницы |
|   ↳ `publish_date` | string | Дата публикации \(ГГГГ-ММ-ДД\) |
|   ↳ `excerpts` | array | Релевантные текстовые выдержки в формате markdown |
|   ↳ `full_content` | string | Полное содержимое страницы в формате markdown |

### `parallel_deep_research`

Выполнение комплексного глубокого исследования по интернету с помощью Parallel AI. Синтезирует информацию из множества источников с цитированием. Выполнение может занимать до 45 минут.

#### Входные параметры

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `input` | string | Да | Исследовательский запрос или вопрос \(до 15 000 символов\) |
| `processor` | string | Нет | Уровень обработки: pro, ultra, pro-fast, ultra-fast \(по умолчанию: pro\) |
| `include_domains` | string | Нет | Список доменов через запятую для ограничения исследования \(политика источников\) |
| `exclude_domains` | string | Нет | Список доменов через запятую для исключения из исследования \(политика источников\) |
| `apiKey` | string | Да | API-ключ Parallel AI |

#### Выходные параметры

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `status` | string | Статус задачи \(completed, failed, running\) |
| `run_id` | string | Уникальный ID этой исследовательской задачи |
| `message` | string | Сообщение о статусе |
| `content` | object | Результаты исследования \(структурированы на основе output_schema\) |
| `basis` | array | Цитаты и источники с обоснованием и уровнями уверенности |
|   ↳ `field` | string | Путь к полю вывода в точечной нотации |
|   ↳ `reasoning` | string | Объяснение результата |
|   ↳ `citations` | array | Массив источников |
|     ↳ `url` | string | URL источника |
|     ↳ `title` | string | Заголовок источника |
|     ↳ `excerpts` | array | Релевантные выдержки из источника |
|   ↳ `confidence` | string | Уровень уверенности \(high, medium\) |
