[BrowserUse](https://browser-use.com/) — это мощная платформа автоматизации браузера, которая позволяет создавать и запускать браузерные задачи программно. Она предоставляет способ автоматизировать веб-взаимодействия с помощью инструкций на естественном языке, позволяя вам переходить на веб-сайты, заполнять формы, извлекать данные и выполнять сложные последовательности действий без написания кода.

С помощью BrowserUse вы можете:

- **Автоматизировать веб-взаимодействия**: переходить на веб-сайты, нажимать кнопки, заполнять формы и выполнять другие действия в браузере
- **Извлекать данные**: собирать контент с веб-сайтов, включая текст, изображения и структурированные данные
- **Выполнять сложные рабочие процессы**: объединять несколько действий для выполнения комплексных веб-задач
- **Отслеживать выполнение задач**: наблюдать за выполнением браузерных задач в реальном времени с визуальной обратной связью
- **Программно обрабатывать результаты**: получать структурированный вывод от задач веб-автоматизации

В Sim интеграция с BrowserUse позволяет вашим агентам взаимодействовать с вебом так, как если бы они были реальными пользователями. Это открывает такие сценарии, как исследование, сбор данных, отправка форм и веб-тестирование — всё через простые инструкции на естественном языке. Ваши агенты могут собирать информацию с веб-сайтов, взаимодействовать с веб-приложениями и выполнять действия, которые обычно требуют ручного просмотра, расширяя свои возможности, включая весь веб как ресурс.

## Инструкции по использованию

Интегрируйте Browser Use в рабочий процесс. Может перемещаться по вебу и выполнять действия так, как если бы реальный пользователь взаимодействовал с браузером.



## Действия

### `browser_use_run_task`

Запускает задачу автоматизации браузера с помощью BrowserUse

#### Входные параметры

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `task` | string | Yes | Что должен сделать браузерный агент |
| `startUrl` | string | No | Начальный URL страницы, с которой агент начинает работу \(сокращает шаги навигации\) |
| `variables` | json | No | Опциональные секреты, внедряемые в задачу \(формат: \{key: value\}\) |
| `allowedDomains` | string | No | Список доменов через запятую, которые агенту разрешено посещать |
| `maxSteps` | number | No | Максимальное количество шагов, которое может выполнить агент \(по умолчанию 100, макс. 10000\) |
| `flashMode` | boolean | No | Включить flash-режим \(быстрее, менее осторожная навигация\) |
| `thinking` | boolean | No | Включить расширенный режим рассуждения |
| `vision` | string | No | Возможность визуального анализа: "true", "false" или "auto" |
| `systemPromptExtension` | string | No | Опциональный текст, добавляемый к системному промпту агента \(макс. 2000 символов\) |
| `structuredOutput` | string | No | Строковое представление JSON-схемы для структурированного вывода |
| `highlightElements` | boolean | No | Подсвечивать интерактивные элементы на странице \(по умолчанию true\) |
| `metadata` | json | No | Пользовательские метаданные ключ-значение \(до 10 пар\) для отслеживания |
| `model` | string | No | Идентификатор LLM-модели \(например, browser-use-2.0\) |
| `apiKey` | string | Yes | API-ключ для BrowserUse API |
| `profile_id` | string | No | ID профиля браузера для постоянных сессий \(куки, состояние входа\) |

#### Выходные параметры

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `id` | string | Идентификатор выполнения задачи |
| `success` | boolean | Статус завершения задачи |
| `output` | json | Итоговый вывод задачи \(строка или структурированный\) |
| `steps` | array | Шаги, выполненные агентом \(number, memory, nextGoal, url, actions, duration\) |
|   ↳ `number` | number | Порядковый номер шага |
|   ↳ `memory` | string | Память агента на этом шаге |
|   ↳ `evaluationPreviousGoal` | string | Оценка выполнения предыдущей цели |
|   ↳ `nextGoal` | string | Цель для следующего шага |
|   ↳ `url` | string | Текущий URL браузера |
|   ↳ `screenshotUrl` | string | Опциональный URL скриншота |
|   ↳ `actions` | array | Строковое представление выполненных JSON-действий |
|   ↳ `duration` | number | Длительность шага в секундах |
| `liveUrl` | string | Встраиваемый URL живой сессии браузера \(активен во время выполнения\) |
| `shareUrl` | string | Публичный URL для доступа к записанной сессии \(после выполнения\) |
| `sessionId` | string | Идентификатор сессии Browser Use |
