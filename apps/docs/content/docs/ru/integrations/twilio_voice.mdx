[Twilio Voice](https://www.twilio.com/en-us/voice) — это мощная облачная коммуникационная платформа, которая позволяет компаниям совершать, принимать и управлять телефонными звонками программно через простой API.

Twilio Voice предоставляет надёжный API для создания сложных голосовых приложений с глобальным охватом. С покрытием в более чем 100 странах, надёжностью операторского уровня и соглашением об уровне обслуживания с доступностью 99,95%, Twilio зарекомендовал себя как лидер отрасли в области программируемых голосовых коммуникаций.

Ключевые возможности Twilio Voice:

- **Глобальная голосовая сеть**: Совершайте и принимайте звонки по всему миру с использованием местных телефонных номеров в различных странах
- **Программируемое управление звонками**: Используйте TwiML для управления потоком вызовов, записи разговоров, сбора DTMF-ввода и реализации IVR-систем
- **Расширенные возможности**: Распознавание речи, преобразование текста в речь, переадресация вызовов, конференц-связь и определение автоответчика
- **Аналитика в реальном времени**: Отслеживайте качество звонков, длительность, стоимость и оптимизируйте ваши голосовые приложения

В Sim интеграция с Twilio Voice позволяет вашим агентам использовать эти мощные голосовые возможности в рамках своих рабочих процессов. Это создаёт возможности для сложных сценариев взаимодействия с клиентами, таких как напоминания о встречах, проверочные звонки, автоматизированные линии поддержки и системы интерактивного голосового ответа. Интеграция устраняет разрыв между вашими AI-рабочими процессами и каналами голосовой связи, позволяя вашим агентам доставлять своевременную, актуальную информацию непосредственно через телефонные звонки. Подключив Sim к Twilio Voice, вы можете создавать интеллектуальных агентов, которые взаимодействуют с клиентами через предпочитаемый ими канал связи, улучшая пользовательский опыт и автоматизируя рутинные задачи по звонкам.

## Инструкции по использованию

Интегрируйте Twilio Voice в рабочий процесс. Совершайте исходящие звонки и получайте записи разговоров.



## Действия

### `twilio_voice_make_call`

Совершение исходящего телефонного звонка с помощью Twilio Voice API.

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `to` | string | Да | Номер телефона для звонка в формате E.164 \(например, +14155551234\) |
| `from` | string | Да | Ваш номер телефона Twilio для звонка в формате E.164 \(например, +14155559876\) |
| `url` | string | Нет | URL вебхука, возвращающий инструкции TwiML для звонка \(например, https://example.com/twiml\) |
| `twiml` | string | Нет | Инструкции TwiML для выполнения. Используйте квадратные скобки вместо угловых \(например, \[Response\]\[Say\]Hello\[/Say\]\[/Response\]\) |
| `statusCallback` | string | Нет | URL вебхука для обновлений статуса звонка |
| `statusCallbackMethod` | string | Нет | HTTP-метод для обратного вызова статуса \(GET или POST\) |
| `accountSid` | string | Да | Идентификатор аккаунта Twilio (Account SID) |
| `authToken` | string | Да | Токен аутентификации Twilio (Auth Token) |
| `record` | boolean | Нет | Записывать ли звонок |
| `recordingStatusCallback` | string | Нет | URL вебхука для обновлений статуса записи |
| `timeout` | number | Нет | Время ожидания ответа до завершения \(секунды, по умолчанию: 60\) |
| `machineDetection` | string | Нет | Определение автоответчика: Enable или DetectMessageEnd |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `success` | boolean | Успешно ли инициирован звонок |
| `callSid` | string | Уникальный идентификатор звонка |
| `status` | string | Статус звонка \(queued, ringing, in-progress, completed и т.д.\) |
| `direction` | string | Направление звонка \(outbound-api\) |
| `from` | string | Номер телефона, с которого совершается звонок |
| `to` | string | Номер телефона, на который совершается звонок |
| `duration` | number | Длительность звонка в секундах |
| `price` | string | Стоимость звонка |
| `priceUnit` | string | Валюта стоимости |
| `error` | string | Сообщение об ошибке, если звонок не удался |

### `twilio_voice_list_calls`

Получение списка звонков, совершённых на аккаунт и с аккаунта.

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `accountSid` | string | Да | Идентификатор аккаунта Twilio (Account SID) |
| `authToken` | string | Да | Токен аутентификации Twilio (Auth Token) |
| `to` | string | Нет | Фильтрация по звонкам на этот номер телефона в формате E.164 \(например, +14155551234\) |
| `from` | string | Нет | Фильтрация по звонкам с этого номера телефона в формате E.164 \(например, +14155559876\) |
| `status` | string | Нет | Фильтрация по статусу звонка \(например, queued, ringing, in-progress, completed, busy, failed, no-answer, canceled\) |
| `startTimeAfter` | string | Нет | Фильтрация звонков, начавшихся начиная с этой даты \(YYYY-MM-DD\) |
| `startTimeBefore` | string | Нет | Фильтрация звонков, начавшихся до этой даты \(YYYY-MM-DD\) |
| `pageSize` | number | Нет | Количество записей для возврата \(макс. 1000, по умолчанию 50\) |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `success` | boolean | Успешно ли получены звонки |
| `calls` | array | Массив объектов звонков |
| `total` | number | Общее количество возвращённых звонков |
| `page` | number | Номер текущей страницы |
| `pageSize` | number | Количество звонков на странице |
| `error` | string | Сообщение об ошибке, если получение не удалось |

### `twilio_voice_get_recording`

Получение информации о записи звонка и транскрипции (если включено через TwiML).

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `recordingSid` | string | Да | Идентификатор записи (Recording SID) для получения \(например, RExxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx\) |
| `accountSid` | string | Да | Идентификатор аккаунта Twilio (Account SID) |
| `authToken` | string | Да | Токен аутентификации Twilio (Auth Token) |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `success` | boolean | Успешно ли получена запись |
| `recordingSid` | string | Уникальный идентификатор записи |
| `callSid` | string | Идентификатор звонка (Call SID), к которому относится эта запись |
| `duration` | number | Длительность записи в секундах |
| `status` | string | Статус записи \(completed, processing и т.д.\) |
| `channels` | number | Количество каналов \(1 для моно, 2 для двойного\) |
| `source` | string | Как была создана запись |
| `mediaUrl` | string | URL для загрузки медиафайла записи |
| `file` | file | Загруженный медиафайл записи |
| `price` | string | Стоимость записи |
| `priceUnit` | string | Валюта стоимости |
| `uri` | string | Относительный URI ресурса записи |
| `transcriptionText` | string | Транскрибированный текст из записи \(если доступно\) |
| `transcriptionStatus` | string | Статус транскрипции \(completed, in-progress, failed\) |
| `transcriptionPrice` | string | Стоимость транскрипции |
| `transcriptionPriceUnit` | string | Валюта стоимости транскрипции |
| `error` | string | Сообщение об ошибке, если получение не удалось |



## Триггеры

**Триггер** — это блок, который запускает рабочий процесс при наступлении события в этом сервисе.

### Twilio Voice Webhook

Запуск рабочего процесса при получении телефонных звонков через Twilio Voice

#### Конфигурация

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `accountSid` | string | Да | Ваш идентификатор аккаунта Twilio (Account SID) из консоли Twilio |
| `authToken` | string | Да | Ваш токен аутентификации Twilio (Auth Token) для проверки подписи вебхука |
| `twimlResponse` | string | Нет | Инструкции TwiML для немедленного возврата в Twilio. Используйте квадратные скобки вместо угловых \(например, \[Response\] вместо &lt;Response&gt;\). Это управляет тем, что происходит при входящем звонке \(например, воспроизведение сообщения, запись, сбор ввода\). Ваш рабочий процесс будет выполняться в фоновом режиме. |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `callSid` | string | Уникальный идентификатор этого звонка |
| `accountSid` | string | Идентификатор аккаунта Twilio (Account SID) |
| `from` | string | Номер телефона звонящего \(формат E.164\) |
| `to` | string | Номер телефона получателя \(ваш номер Twilio\) |
| `callStatus` | string | Статус звонка \(queued, ringing, in-progress, completed и т.д.\) |
| `direction` | string | Направление звонка: inbound или outbound |
| `apiVersion` | string | Версия API Twilio |
| `callerName` | string | Имя абонента, если доступно |
| `forwardedFrom` | string | Номер телефона, с которого переадресован этот звонок |
| `digits` | string | DTMF-цифры, введённые звонящим \(из &lt;Gather&gt;\) |
| `speechResult` | string | Результат распознавания речи \(если используется &lt;Gather&gt; с речью\) |
| `recordingUrl` | string | URL записи звонка, если доступно |
| `recordingSid` | string | Идентификатор записи (Recording SID), если доступно |
| `called` | string | Номер телефона, на который звонили \(то же, что "to"\) |
| `caller` | string | Номер телефона звонящего \(то же, что "from"\) |
| `toCity` | string | Город вызываемого номера |
| `toState` | string | Штат/регион вызываемого номера |
| `toZip` | string | Почтовый индекс вызываемого номера |
| `toCountry` | string | Страна вызываемого номера |
| `fromCity` | string | Город звонящего |
| `fromState` | string | Штат/регион звонящего |
| `fromZip` | string | Почтовый индекс звонящего |
| `fromCountry` | string | Страна звонящего |
| `calledCity` | string | Город вызываемого номера \(то же, что toCity\) |
| `calledState` | string | Штат вызываемого номера \(то же, что toState\) |
| `calledZip` | string | Почтовый индекс вызываемого номера \(то же, что toZip\) |
| `calledCountry` | string | Страна вызываемого номера \(то же, что toCountry\) |
| `callerCity` | string | Город звонящего \(то же, что fromCity\) |
| `callerState` | string | Штат звонящего \(то же, что fromState\) |
| `callerZip` | string | Почтовый индекс звонящего \(то же, что fromZip\) |
| `callerCountry` | string | Страна звонящего \(то же, что fromCountry\) |
| `callToken` | string | Токен звонка Twilio для аутентификации |
| `raw` | string | Полная полезная нагрузка вебхука от Twilio в виде строки JSON |
