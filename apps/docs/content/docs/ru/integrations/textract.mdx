[AWS Textract](https://aws.amazon.com/textract/) — это мощный AI-сервис от Amazon Web Services, предназначенный для автоматического извлечения печатного текста, рукописного текста, таблиц, форм, пар ключ-значение и других структурированных данных из отсканированных документов и изображений. Textract использует передовое оптическое распознавание символов (OCR) и анализ документов для преобразования документов в пригодные для использования данные, обеспечивая автоматизацию, аналитику, соответствие требованиям и многое другое.

С помощью AWS Textract вы можете:

- **Извлекать текст из изображений и документов**: Распознавайте печатный и рукописный текст в форматах PDF, JPEG, PNG или TIFF
- **Обнаруживать и извлекать таблицы**: Автоматически находите таблицы и выводите их структурированное содержимое
- **Разбирать формы и пары ключ-значение**: Извлекайте структурированные данные из форм, включая поля и соответствующие им значения
- **Идентифицировать подписи и элементы макета**: Обнаруживайте подписи, геометрический макет и связи между элементами документа
- **Настраивать извлечение с помощью запросов**: Извлекайте определённые поля и ответы с помощью извлечения на основе запросов (например, «Какой номер счёта?»)

В Sim интеграция с AWS Textract позволяет вашим агентам интеллектуально обрабатывать документы в рамках своих рабочих процессов. Это открывает сценарии автоматизации, такие как ввод данных из счетов, документов о приёме на работу, контрактов, квитанций и многого другого. Ваши агенты могут извлекать релевантные данные, анализировать структурированные формы и генерировать резюме или отчёты непосредственно из загруженных документов или URL. Подключив Sim к AWS Textract, вы можете сократить ручные усилия, повысить точность данных и оптимизировать бизнес-процессы с помощью надёжного понимания документов.

## Инструкции по использованию

Интегрируйте AWS Textract в рабочий процесс для извлечения текста, таблиц, форм и пар ключ-значение из документов. Одностраничный режим поддерживает JPEG, PNG и одностраничные PDF. Многостраничный режим поддерживает многостраничные PDF и TIFF.



## Действия

### `textract_parser`

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `file` | file | Нет | Документ для обработки \(JPEG, PNG или одностраничный PDF\). |

#### Выход

| Параметр | Тип | Описание |
| --------- | ---- | ----------- |
| `blocks` | array | Массив объектов Block, содержащих обнаруженный текст, таблицы, формы и другие элементы |
|   ↳ `BlockType` | string | Тип блока \(PAGE, LINE, WORD, TABLE, CELL, KEY_VALUE_SET и т.д.\) |
|   ↳ `Id` | string | Уникальный идентификатор блока |
|   ↳ `Text` | string | Текстовое содержимое \(для блоков LINE и WORD\) |
|   ↳ `TextType` | string | Тип текста \(PRINTED или HANDWRITING\) |
|   ↳ `Confidence` | number | Оценка уверенности \(0-100\) |
|   ↳ `Page` | number | Номер страницы |
|   ↳ `Geometry` | object | Информация о расположении и ограничивающей рамке |
|     ↳ `BoundingBox` | object | Вывод BoundingBox от инструмента |
|       ↳ `Height` | number | Высота как отношение к высоте документа |
|       ↳ `Left` | number | Левая позиция как отношение к ширине документа |
|       ↳ `Top` | number | Верхняя позиция как отношение к высоте документа |
|       ↳ `Width` | number | Ширина как отношение к ширине документа |
|     ↳ `Polygon` | array | Координаты полигона |
|       ↳ `X` | number | Координата X |
|       ↳ `Y` | number | Координата Y |
|   ↳ `Relationships` | array | Отношения с другими блоками |
|     ↳ `Type` | string | Тип отношения \(CHILD, VALUE, ANSWER и т.д.\) |
|     ↳ `Ids` | array | Идентификаторы связанных блоков |
|   ↳ `EntityTypes` | array | Типы сущностей для KEY_VALUE_SET \(KEY или VALUE\) |
|   ↳ `SelectionStatus` | string | Для флажков: SELECTED или NOT_SELECTED |
|   ↳ `RowIndex` | number | Индекс строки для ячеек таблицы |
|   ↳ `ColumnIndex` | number | Индекс столбца для ячеек таблицы |
|   ↳ `RowSpan` | number | Объединение строк для объединённых ячеек |
|   ↳ `ColumnSpan` | number | Объединение столбцов для объединённых ячеек |
|   ↳ `Query` | object | Информация о запросе для блоков QUERY |
|     ↳ `Text` | string | Текст запроса |
|     ↳ `Alias` | string | Псевдоним запроса |
|     ↳ `Pages` | array | Страницы для поиска |
| `documentMetadata` | object | Метаданные об анализируемом документе |
|   ↳ `pages` | number | Количество страниц в документе |
| `modelVersion` | string | Версия модели Textract, использованной для обработки |
