## Overview

[Tailscale](https://tailscale.com) is a zero-config mesh VPN built on WireGuard that makes it easy to connect devices, services, and users across any network. The Tailscale block lets you automate network management tasks like device provisioning, access control, route management, and DNS configuration directly from your Sim workflows.

## Authentication

The Tailscale block uses Ключ API authentication. To get an Ключ API:

1. Go to the [Tailscale admin console](https://login.tailscale.com/admin/settings/keys)
2. Navigate to **Settings > Keys**
3. Click **Generate Ключ API**
4. Set an expiry (1-90 days) and copy the key (starts with `tskey-api-`)

You must have an **Owner**, **Admin**, **IT admin**, or **Network admin** role to generate Ключ APIs.

## Tailnet Identifier

Every operation requires a **tailnet** parameter. This is typically your organization's domain name (e.g., `example.com`). You can also use `"-"` to refer to your по умолчанию tailnet.

## Common Use Cases

- **Device inventory**: Получение списка and monitor all devices connected to your network
- **Automated provisioning**: Create and manage auth keys to pre-authorize new devices
- **Access control**: Authorize or deauthorize devices, manage device tags for ACL policies
- **Route management**: View and enable subnet routes for devices acting as subnet routers
- **DNS management**: Configure nameservers, MagicDNS, and search paths
- **Key lifecycle**: Create, list, inspect, and revoke auth keys
- **User auditing**: Получение списка users in the tailnet and their roles
- **Policy review**: Получение the current ACL policy for inspection or backup

## Инструкции по использованию

Interact with the Tailscale API to manage devices, DNS, ACLs, auth keys, users, and routes across your tailnet.



## Действия

### `tailscale_list_devices`

Получение списка devices in the tailnet

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Да | Tailscale Ключ API |
| `tailnet` | string | Да | Tailnet name (например, example.com\) or "-" for по умолчанию |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `devices` | array | Получение списка of devices in the tailnet |
|   ↳ `id` | string | Device ID |
|   ↳ `name` | string | Device name |
|   ↳ `hostname` | string | Device hostname |
|   ↳ `user` | string | Associated user |
|   ↳ `os` | string | Operating system |
|   ↳ `clientVersion` | string | Tailscale client version |
|   ↳ `addresses` | array | Tailscale IP addresses |
|   ↳ `tags` | array | Device tags |
|   ↳ `authorized` | boolean | Указывает,  device is authorized |
|   ↳ `blocksIncomingConnections` | boolean | Указывает,  device blocks incoming connections |
|   ↳ `lastSeen` | string | Last seen timestamp |
|   ↳ `created` | string | Creation timestamp |
| `count` | number | Total number of devices |

### `tailscale_create_auth_key`

Создание нового auth key for the tailnet to pre-authorize devices

#### Вход

| Параметр | Тип | Обязательный | Описание |
| --------- | ---- | -------- | ----------- |
| `apiKey` | string | Да | Tailscale Ключ API |
| `tailnet` | string | Да | Tailnet name (например, example.com\) or "-" for по умолчанию |
| `reusable` | boolean | Нет | Указывает,  key can be used more than once |
| `ephemeral` | boolean | Нет | Whether devices authenticated with this key are ephemeral |
| `preauthorized` | boolean | Нет | Whether devices are pre-authorized \(skip manual approval\) |
| `tags` | string | Нет | Comma-separated list of tags for devices using this key (например, "tag:server,tag:prod"\) |
| `description` | string | Нет | Description for the auth key |
| `expirySeconds` | number | Нет | Key expiry time in seconds \(по умолчанию: 90 days\) |

#### Выход

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `id` | string | Auth key ID |
| `key` | string | The auth key value \(only shown once at creation\) |
| `description` | string | Key description |
| `created` | string | Creation timestamp |
| `expires` | string | Expiration timestamp |
| `revoked` | string | Revocation timestamp \(empty if not revoked\) |
| `capabilities` | object | Key capabilities |
|   ↳ `reusable` | boolean | Указывает,  key is reusable |
|   ↳ `ephemeral` | boolean | Whether devices are ephemeral |
|   ↳ `preauthorized` | boolean | Whether devices are pre-authorized |
|   ↳ `tags` | array | Tags applied to devices using this key |
