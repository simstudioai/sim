[Asana](https://asana.com/) — это ведущая платформа управления работой, разработанная для помощи командам в организации, отслеживании и управлении задачами и проектами более эффективно. С помощью Asana отдельные сотрудники и организации могут оптимизировать планирование проектов, делегировать обязанности, отслеживать прогресс и беспрепятственно сотрудничать в различных рабочих пространствах и проектах.

С помощью Asana вы можете:

- **Создавать, назначать и обновлять задачи**: разбивайте работу на actionable элементы, назначайте их участникам команды и держите всех в курсе.
- **Организовывать проекты**: группируйте связанные задачи в проекты, устанавливайте сроки и визуализируйте поток работы от начала до конца.
- **Устанавливать приоритеты и сроки**: обеспечьте своевременное выполнение важных задач и соответствие целям проекта.
- **Отслеживать прогресс и завершение**: отслеживайте задачи по мере их прохождения через различные стадии и быстро выявляйте препятствия.
- **Сотрудничать с командой**: делитесь заметками, прикрепляйте соответствующие ресурсы и обменивайтесь обновлениями непосредственно в задачах.

В Sim интеграция с Asana позволяет вашим агентам программно взаимодействовать с Asana через набор гибких инструментов, описанных ниже. Ваши агенты могут получать, создавать и обновлять задачи, что упрощает автоматизацию рабочих процессов управления проектами, синхронизацию статуса с другими инструментами или запуск действий на основе событий задач Asana. Используйте эти инструменты для повышения продуктивности вашей команды и поддержания всех проектов организованными и актуальными непосредственно в ваших проектах Sim.

## Инструкция по использованию

Интегрируйте Asana в рабочий процесс. Позволяет читать, создавать и обновлять задачи.



## Действия

### `asana_get_task`

Получение одной задачи по GID или нескольких задач с фильтрами

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `taskGid` | string | No | Глобально уникальный идентификатор (GID) задачи. Если не указан, будут получены несколько задач. |
| `workspace` | string | No | GID рабочего пространства Asana (числовая строка) для фильтрации задач (обязательно, если не используется taskGid) |
| `project` | string | No | GID проекта Asana (числовая строка) для фильтрации задач |
| `limit` | number | No | Максимальное количество возвращаемых задач (по умолчанию: 50) |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Статус успешности операции |
| `ts` | string | Временная метка ответа |
| `gid` | string | Глобально уникальный идентификатор задачи |
| `resource_type` | string | Тип ресурса (task) |
| `resource_subtype` | string | Подтип ресурса |
| `name` | string | Название задачи |
| `notes` | string | Заметки или описание задачи |
| `completed` | boolean | Завершена ли задача |
| `assignee` | object | Данные назначенного исполнителя |
|   ↳ `gid` | string | GID исполнителя |
|   ↳ `name` | string | Имя исполнителя |
| `created_by` | object | Данные создателя |
|   ↳ `gid` | string | GID создателя |
|   ↳ `name` | string | Имя создателя |
| `due_on` | string | Срок выполнения (ГГГГ-ММ-ДД) |
| `created_at` | string | Временная метка создания задачи |
| `modified_at` | string | Временная метка последнего изменения задачи |
| `tasks` | array | Массив задач (при получении нескольких) |
|   ↳ `gid` | string | GID задачи |
|   ↳ `name` | string | Название задачи |
|   ↳ `completed` | boolean | Статус завершения |

### `asana_create_task`

Создание новой задачи в Asana

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `workspace` | string | Yes | GID рабочего пространства Asana (числовая строка), в котором будет создана задача |
| `name` | string | Yes | Название задачи |
| `notes` | string | No | Заметки или описание задачи |
| `assignee` | string | No | GID пользователя, которому назначить задачу |
| `due_on` | string | No | Срок выполнения в формате ГГГГ-ММ-ДД |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Статус успешности операции |
| `ts` | string | Временная метка ответа |
| `gid` | string | Глобально уникальный идентификатор задачи |
| `name` | string | Название задачи |
| `notes` | string | Заметки или описание задачи |
| `completed` | boolean | Завершена ли задача |
| `created_at` | string | Временная метка создания задачи |
| `permalink_url` | string | URL задачи в Asana |

### `asana_update_task`

Обновление существующей задачи в Asana

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `taskGid` | string | Yes | GID задачи Asana (числовая строка) для обновления |
| `name` | string | No | Обновлённое название задачи |
| `notes` | string | No | Обновлённые заметки или описание задачи |
| `assignee` | string | No | Обновлённый GID назначенного пользователя |
| `completed` | boolean | No | Отметить задачу как завершённую или незавершённую |
| `due_on` | string | No | Обновлённый срок выполнения в формате ГГГГ-ММ-ДД |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Статус успешности операции |
| `ts` | string | Временная метка ответа |
| `gid` | string | Глобально уникальный идентификатор задачи |
| `name` | string | Название задачи |
| `notes` | string | Заметки или описание задачи |
| `completed` | boolean | Завершена ли задача |
| `modified_at` | string | Временная метка последнего изменения задачи |

### `asana_get_projects`

Получение всех проектов из рабочего пространства Asana

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `workspace` | string | Yes | GID рабочего пространства Asana (числовая строка) для получения проектов |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Статус успешности операции |
| `ts` | string | Временная метка ответа |
| `projects` | array | Массив проектов |
|   ↳ `gid` | string | GID проекта |
|   ↳ `name` | string | Название проекта |
|   ↳ `resource_type` | string | Тип ресурса (project) |

### `asana_search_tasks`

Поиск задач в рабочем пространстве Asana

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `workspace` | string | Yes | GID рабочего пространства Asana (числовая строка) для поиска задач |
| `text` | string | No | Текст для поиска в названиях задач |
| `assignee` | string | No | Фильтр задач по GID назначенного пользователя |
| `projects` | array | No | Массив GID проектов Asana (числовые строки) для фильтрации задач |
| `completed` | boolean | No | Фильтр по статусу завершения |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Статус успешности операции |
| `ts` | string | Временная метка ответа |
| `tasks` | array | Массив подходящих задач |
|   ↳ `gid` | string | GID задачи |
|   ↳ `resource_type` | string | Тип ресурса |
|   ↳ `resource_subtype` | string | Подтип ресурса |
|   ↳ `name` | string | Название задачи |
|   ↳ `notes` | string | Заметки задачи |
|   ↳ `completed` | boolean | Статус завершения |
|   ↳ `assignee` | object | Данные назначенного исполнителя |
|     ↳ `gid` | string | GID исполнителя |
|     ↳ `name` | string | Имя исполнителя |
|   ↳ `due_on` | string | Срок выполнения |
|   ↳ `created_at` | string | Временная метка создания |
|   ↳ `modified_at` | string | Временная метка изменения |
| `next_page` | object | Информация о пагинации |
|   ↳ `offset` | string | Токен смещения |
|   ↳ `path` | string | Путь API |
|   ↳ `uri` | string | Полный URI |

### `asana_add_comment`

Добавление комментария (истории) к задаче Asana

#### Input

| Parameter | Type | Required | Description |
| --------- | ---- | -------- | ----------- |
| `taskGid` | string | Yes | GID задачи Asana (числовая строка) |
| `text` | string | Yes | Текстовое содержимое комментария |

#### Output

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `success` | boolean | Статус успешности операции |
| `ts` | string | Временная метка ответа |
| `gid` | string | Глобально уникальный идентификатор комментария |
| `text` | string | Текстовое содержимое комментария |
| `created_at` | string | Временная метка создания комментария |
| `created_by` | object | Данные автора комментария |
|   ↳ `gid` | string | GID автора |
|   ↳ `name` | string | Имя автора |
