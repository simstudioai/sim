**Таблица** — это сетка типизированных колонок в вашем рабочем пространстве, как электронная таблица со схемой. Используйте её для хранения справочных данных, сбора результатов воркфлоу или структурированных записей, которые читают и записывают ваши агенты.

## Типы колонок

Каждая колонка имеет тип, который определяет, как её значения хранятся и проверяются.

| Тип | Хранит | Пример |
| --- | --- | --- |
| **Text** | Строку свободной формы | `"Acme Corp"` |
| **Number** | Числовое значение | `42` |
| **Boolean** | `true` или `false` | `true` |
| **Date** | Дату | `2026-03-16` |
| **JSON** | Объект или массив | `{ "tier": "pro" }` |

Типы проверяются по мере ввода значений, так что колонка Number принимает только числа.

## Редактирование таблицы

Откройте раздел **Tables** на боковой панели и нажмите **New table**, чтобы создать таблицу. Добавляйте колонки из заголовка колонки, вводите данные в ячейку для редактирования и вставляйте строки из электронной таблицы для массовой загрузки. Фильтруйте и сортируйте с панели инструментов, не изменяя исходные данные. Редактор имеет полную поддержку клавиатуры; см. [клавиатурные сокращения](/keyboard-shortcuts).

## Таблицы в воркфлоу

[Блок Table](/tables/using-in-workflows) читает и записывает строки изнутри воркфлоу: запрашивает строки для справочных данных, записывает результаты обратно или обрабатывает пакет. [Столбец-воркфлоу](/tables/workflow-columns) идёт на шаг дальше и запускает воркфлоу для каждой строки отдельно. Всё в таблице также доступно через [REST API](/api-reference/getting-started).
