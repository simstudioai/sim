Официальный Python SDK для Sim позволяет выполнять рабочие процессы программно из ваших Python-приложений с использованием официального Python SDK.

<Callout type="info">
  Python SDK поддерживает Python 3.8+ с поддержкой асинхронного выполнения, автоматическим ограничением частоты запросов с экспоненциальной отсрочкой и отслеживанием использования.
</Callout>

## Установка

Установите SDK с помощью pip:

```bash
pip install simstudio-sdk
```
