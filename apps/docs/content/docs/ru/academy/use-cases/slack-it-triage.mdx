## Что вы построите

**Триггер Slack** запускает воркфлоу при каждом новом сообщении. Блок **Agent** классифицирует запрос и выбирает путь: **поиск по базе знаний** составляет ответ для типовых вопросов, а всё, что он не может решить, направляется человеку или оформляется как тикет. Каждый запуск записывается в [логах](/logs-debugging).

Это объединяет [воркфлоу](/academy/workflows/intro), [агентов](/academy/agents/intro) и [базы знаний](/academy/knowledge-bases/intro) в одну систему — основы, скомпонованные вместе.
