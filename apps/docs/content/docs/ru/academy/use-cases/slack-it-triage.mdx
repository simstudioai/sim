## Что вы создадите
**Триггер Slack** запускает рабочий процесс при получении каждого нового сообщения. Блок "**Agent**" классифицирует запрос и определяет дальнейший путь: поиск в базе знаний предлагает ответ на распространенные вопросы, а все, что не может быть решено автоматически, передается человеку или оформляется как задача. Каждый запуск записывается в[журналы](/logs-debugging)журналы/журнал отладки.

Это объединяет[рабочие процессы](/academy/workflows/intro)рабочие процессы/курсы/рабочие процессы /введение, [агенты](/academy/agents/intro)агенты/академия/представители/введениеи[базы знаний](/academy/knowledge-bases/intro)базы знаний/академия/базы знаний /введениев одну систему – основные компоненты, собранные вместе.
