## Что вы создадите
Приходит **файл**, блок с **агентом** извлекает поля и помещает их в…[структурированный вывод](/academy/agents/intro)структурированный вывод/академия/представители/введениеТаким образом, форма данных предсказуема, и блок "Таблица" записывает строку. Если запустить его для набора файлов, то папка с документами превращается в таблицу, которую можно запросить.

Это включает в себя[файлы](/academy/files/intro)файлы/academy/файлы/введение, [агенты](/academy/agents/intro)представители/академия/представители/введениеи[столы](/academy/tables/intro)столы/академия/таблицы/введение.
