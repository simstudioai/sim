# Быстрый старт

Создайте агента-исследователя за 10 минут. Он принимает имя через чат-интерфейс, ищет информацию в интернете с помощью ИИ-инструментов и возвращает структурированный JSON с местоположением, профессией и образованием человека.

## Руководство

<Steps>
  <Step title="Создайте воркфлоу и добавьте агента">
    На боковой панели нажмите **+** рядом с **Workflows** и назовите новый воркфлоу «Getting Started». Блок **Start** уже находится на холсте — это точка входа, которая принимает ввод из чата.

    Перетащите блок **Agent** на холст и настройте его **Messages**:
    - **System**: «You are a people research agent. When given a person's name, use your search tools to find their location, profession, educational background, and other relevant details.»
    - **User**: вставьте `<start.input>`, чтобы агент читал то, что получает чат.

    Оставьте **Model** по умолчанию (`claude-sonnet-4-6`) или выберите любую другую.

    <div className="mx-auto w-full overflow-hidden rounded-lg">
      <Video src="getting-started/started-2.mp4" width={700} height={450} />
    </div>
  </Step>

  <Step title="Добавьте поисковые инструменты">
    В секции **Tools** блока Agent добавьте **Exa** и **Linkup**. Агент сам решает, когда их вызывать.

    <div className="mx-auto w-3/5 overflow-hidden rounded-lg">
      <Video src="getting-started/started-3.mp4" width={700} height={450} />
    </div>
  </Step>

  <Step title="Протестируйте">
    Откройте панель **Chat**, выберите `agent1.content` в качестве отображаемого вывода и отправьте: *«John is a software engineer from San Francisco who studied Computer Science at Stanford University.»*

    Агент выполняет поиск, затем возвращает найденную информацию.

    <Callout type="info">
    Если запуск не завершился успешно, откройте выпадающий список вывода и посмотрите, что вернул каждый блок — причина ошибки почти всегда видна там. Две самые частые причины: отсутствующий API-ключ у инструмента и ссылка на значение, которого нет. Подробный метод см. в разделе [трассировка ошибок в обратном порядке](/logs-debugging).
    </Callout>

    <div className="mx-auto w-full overflow-hidden rounded-lg">
      <Video src="getting-started/started-4.mp4" width={700} height={450} />
    </div>
  </Step>

  <Step title="Добавьте структурированный вывод">
    В секции **Response Format** блока Agent нажмите на волшебную палочку (✨) и введите промпт: *«create a schema named person, that contains location, profession, and education»*. Схема будет создана автоматически.

    <div className="mx-auto w-full overflow-hidden rounded-lg">
      <Video src="getting-started/started-5.mp4" width={700} height={450} />
    </div>
  </Step>

  <Step title="Протестируйте структурированный вывод">
    Вернитесь в панель **Chat**, выберите новый вариант структурированного вывода и отправьте: *«Sarah is a marketing manager from New York who has an MBA from Harvard Business School.»*

    Теперь агент возвращает JSON с полями `location`, `profession` и `education` — готовый для чтения по имени любым последующим блоком.

    <div className="mx-auto w-full overflow-hidden rounded-lg">
      <Video src="getting-started/started-6.mp4" width={700} height={450} />
    </div>
  </Step>
</Steps>
