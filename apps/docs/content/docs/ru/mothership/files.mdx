Опишите документ, презентацию, изображение или визуализацию, и Mothership создаст это — транслируя содержимое в реальном времени на панель ресурсов по мере написания. Прикрепите любой файл к вашему сообщению, и Mothership прочитает его, обработает и сохранит в ваше рабочее пространство.

## Загрузка файлов в рабочее пространство

Прикрепите любой файл прямо к сообщению Mothership — перетащите его в поле ввода, вставьте или нажмите значок вложения. Mothership читает файл как контекст и сохраняет его в ваше рабочее пространство.

Используйте это, чтобы:
- Передать Mothership документ и попросить обработать, суммировать или извлечь из него данные
- Загрузить CSV и создать из него таблицу
- Перетащить PDF и попросить Mothership превратить его в документ базы знаний
- Прикрепить дизайн-макет и попросить Mothership описать его или сгенерировать код

Загруженные файлы отображаются на панели Файлы в боковой панели и доступны всем рабочим процессам в рабочем пространстве. Mothership также может загрузить файл напрямую по URL и сохранить его: «Скачай JSON по адресу [URL] и сохрани в рабочее пространство».

## Создание документов

Mothership может записывать любые текстовые файлы — markdown, обычный текст, файлы кода, CSV, JSON или любой другой формат:

- «Напиши техническую спецификацию для новой системы аутентификации как markdown-файл»
- «Создай CSV наших тестовых аккаунтов с колонками name, email и plan tier»
- «Напиши Python-скрипт, который вызывает API нашего рабочего процесса и обрабатывает ответ»
- «Составь посмертный анализ сбоя в прошлый вторник и сохрани как markdown-файл»
- «Напиши персонализированное исходящее письмо для Acme Corp на основе их недавнего объявления о финансировании»
- «Составь еженедельную операционную сводку с количеством запусков рабочих процессов, ошибками и основными сбоями за последние 7 дней»

Файлы сохраняются в ваше рабочее пространство и доступны с панели Файлы в боковой панели.

## Редактирование существующих файлов

Откройте файл с помощью `@имяфайла` или меню **+**, затем опишите изменение:

- «Обнови раздел с ценами, чтобы отразить новые тарифы»
- «Перепиши этот Python-скрипт с использованием async/await»
- «Добавь раздел об обработке ошибок в эту спецификацию»
- «Перепиши введение этого отчёта, чтобы оно было более лаконичным»

## Презентации

Mothership может генерировать файлы `.pptx`:

- «Создай питч-дек для обзора Q3 — 8 слайдов о росте, удержании и дорожной карте»
- «Преврати этот исследовательский отчёт в презентацию на 10 слайдов»
- «Создай презентацию, описывающую процесс онбординга через наш API»
- «Создай презентацию с карточками сравнения для наших топ-3 конкурентов — по одному слайду на позиционирование, цены и как мы выигрываем»
- «Создай план аккаунта для Acme Corp — их приоритеты, наше решение и предлагаемые следующие шаги»

Файл сохраняется в ваше рабочее пространство и может быть скачан.

## Изображения

Mothership может генерировать изображения с помощью ИИ, а также использовать существующее изображение в качестве образца:

**Генерация изображений:**
- «Сгенерируй баннер для анонса новой функции — тёмный фон, чистый шрифт»
- «Создай диаграмму, показывающую поток данных через наш конвейер вебхуков»
- «Сделай социальную карточку для блог-поста с заголовком и именем автора»

**Использование образца изображения:**
- Прикрепите существующее изображение к сообщению, затем опишите, что вы хотите: «Сгенерируй новую версию этого баннера с синей цветовой схемой вместо зелёной»
- «Создай вариант этой диаграммы с горизонтальным расположением блоков [прикрепите изображение]»

## Диаграммы и визуализации

Mothership может генерировать диаграммы и визуализации данных на основе описанных или указанных вами данных:

- «Построй график количества запусков рабочих процессов из таблицы metrics в виде столбчатой диаграммы, сгруппированной по неделям»
- «Создай линейный график использования токенов за последние 30 дней на основе этих данных [вставьте данные]»
- «Сгенерируй круговую диаграмму распределения источников лидов из таблицы leads»

## Вычисления и обработка данных

Для разовых вычислений и преобразований данных опишите, что вам нужно, и Mothership выполнит это прямо в чате:

- «Разбери этот JSON и извлеки все записи, где статус 'failed'»
- «Рассчитай p95 задержки из этих значений времени: [вставьте значения]»
- «Преобразуй эти Unix-временные метки в ISO 8601»
- «Дедуплицируй этот список email без учёта регистра»

Результаты возвращаются прямо в чат. Попросите Mothership сохранить вывод как файл, если он вам нужен.

## Режимы просмотра файлов

Когда файл открывается на панели ресурсов, вы можете переключаться между тремя режимами:

| Режим | Что показывает |
|------|--------------|
| **Редактор** | Редактируемый текст |
| **Предпросмотр** | Отрендеренный вывод (markdown, HTML) |
| **Разделённый** | Редактор и предпросмотр рядом |
