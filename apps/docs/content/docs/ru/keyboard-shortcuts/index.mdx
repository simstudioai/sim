Sim имеет клавиатурные сокращения для редактора воркфлоу и для таблиц. Каждый набор работает, когда соответствующая поверхность в фокусе и вы не вводите текст в поле.

<Callout type="info">
  **Mod** — это `Cmd` на macOS и `Ctrl` на Windows и Linux.
</Callout>

## Редактор воркфлоу

### Мышь

| Действие | Управление |
|--------|---------|
| Панорамирование холста | Левое перетаскивание по пустому пространству или прокрутка/трекпад |
| Выбор нескольких блоков | Правое перетаскивание для рисования рамки выбора |
| Перетаскивание блока | Левое перетаскивание за заголовок блока |
| Добавить к выбору | `Mod` + клик по блокам |

### Действия

| Сокращение | Действие |
|----------|--------|
| `Mod` + `Enter` | Запустить воркфлоу (или отменить, если выполняется) |
| `Mod` + `Z` | Отменить |
| `Mod` + `Shift` + `Z` | Повторить |
| `Mod` + `C` / `X` / `V` | Копировать / вырезать / вставить выбранные блоки |
| `Delete` или `Backspace` | Удалить выбранные блоки или связи |
| `Shift` + `L` | Автоматическая компоновка холста |
| `Mod` + `Shift` + `F` | По размеру окна |
| `Mod` + `F` | Поиск и замена в воркфлоу |
| `Mod` + `Shift` + `Enter` | Принять изменения Copilot |
| `Mod` + `Alt` + `F` | Фокус на поиск панели инструментов |

## Таблицы

Работают, когда таблица в фокусе и ни одна ячейка не редактируется.

### Навигация

| Сокращение | Действие |
|----------|--------|
| Клавиши стрелок | Переместиться на одну ячейку |
| `Mod` + клавиши стрелок | Перейти к краю таблицы |
| `Tab` / `Shift` + `Tab` | Перейти к следующей / предыдущей ячейке |
| `Escape` | Снять выбор |

### Выбор

| Сокращение | Действие |
|----------|--------|
| `Shift` + клавиши стрелок | Расширить выбор на одну ячейку |
| `Mod` + `Shift` + клавиши стрелок | Расширить выбор до края |
| `Shift` + `Space` | Выбрать текущую строку |
| `Mod` + `Space` | Выбрать текущую колонку |

### Редактирование

| Сокращение | Действие |
|----------|--------|
| `Enter` или `F2` | Начать редактирование выбранной ячейки |
| Любой символ | Начать редактирование с этого символа |
| `Escape` | Отменить редактирование |
| `Shift` + `Enter` | Вставить новую строку ниже |
| `Space` | Развернуть детали строки |

### Буфер обмена

| Сокращение | Действие |
|----------|--------|
| `Mod` + `C` / `X` / `V` | Копировать / вырезать / вставить ячейки |
| `Delete` или `Backspace` | Очистить выбранные ячейки |

### История

| Сокращение | Действие |
|----------|--------|
| `Mod` + `Z` | Отменить |
| `Mod` + `Shift` + `Z` или `Mod` + `Y` | Повторить |

## Глобальные

| Сокращение | Действие |
|----------|--------|
| `Mod` + `K` | Открыть поиск |
| `Mod` + `Shift` + `A` | Создать нового агента |
| `Mod` + `Shift` + `P` | Создать новый воркфлоу |
| `Mod` + `L` | Перейти к логам |
| `Mod` + `D` | Очистить консоль терминала |
| `Mod` + `E` | Очистить уведомления |
